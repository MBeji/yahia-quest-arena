## معنى «النخبة» في عربية التاسعة

- **الإعراب الدقيق**: إعراب كلمة في تركيب مركّب حيث يتقارب وجهان (فاعل مرفوع مقابل نائب فاعل؛ اسم
  إنّ مقابل خبر كان)، وتمييز الحالات المتشابهة.
- **الميزان الصرفي**: وزن كلمة معتلّة أو مزيدة (الإعلال، الإبدال، الحذف) — سببٌ خصب للتمييز.
- **التحويل المقيّد**: تحويل جملة مع دخول ناسخ **مع** تغيير الزمن/العدد؛ بناء المشتقّ المطلوب (اسم
  فاعل/مفعول/مبالغة) من فعل معتلّ.
- **الأسلوب**: تمييز أسلوب من آخر متقارب (استثناء تامّ موجب مقابل منفيّ؛ تعجّب مقابل استفهام).
- **البلاغة**: التمييز بين صور بيانية متقاربة (تشبيه/استعارة؛ كناية/مجاز) وبين محسّنات (طباق/مقابلة؛
  جناس/سجع) داخل شاهد.
- **صيد الخطأ**: إعرابٌ أو وزنٌ «تلميذيّ» فيه خطأ واحد يُطلب تحديده.

## معجم الأخطاء — كلّ مُشتِّت ينفّذ خطأً واحدًا حتى نهايته

| الباب           | الأخطاء التي تُنفَّذ (الناتج الخاطئ = المُشتِّت)                                                       |
| --------------- | ------------------------------------------------------------------------------------------------------ |
| الإعراب والبناء | خلط المبنيّ بالمُعرَب؛ علامة الرفع/النصب الفرعية مُهمَلة (المثنّى، جمع المذكّر السالم، الأسماء الخمسة) |
| النواسخ         | خبر كان منصوب/خبر إنّ مرفوع (عكس الصواب)؛ اسم الناسخ وخبره متبادلان                                    |
| المنصوبات       | الحال والتمييز متبادلان؛ المفعول المطلق والمفعول لأجله ملتبسان؛ المستثنى في النفي التامّ               |
| الصرف/الميزان   | وزن الكلمة دون ردّ المحذوف؛ إهمال الإعلال/الإبدال؛ عدّ حروف الزيادة خطأً                               |
| المشتقّات       | اسم الفاعل مقابل اسم المفعول من الأجوف؛ صيغة المبالغة مقابل اسم الفاعل                                 |
| البلاغة         | التشبيه (بأداة) مقابل الاستعارة؛ الكناية مقابل التصريح؛ الطباق مقابل المقابلة                          |
| الأساليب        | الاستثناء التامّ المنفيّ يجيز البدل؛ خلط التعجّب بالاستفهام                                            |

## دقّة اللغة

جُمل الأسئلة فصيحة لا لحن فيها؛ المُشتِّتات جُملٌ سليمة التركيب لكنّها خاطئة في النقطة المدروسة —
لا كلام مُبهم. اشكل أواخر الكلمات المُعرَبة عند الحاجة لرفع اللبس. تحقّق من الشواهد (الآيات ⟵ رواية
قالون حصريًّا، cf. content-ecole-tn) واذكر المصادر في `chapter.json` `sources[]`.

## سير العمل

1. الفصل/الفصول → **تدقيق السلّم الموجود** (`content/arabic/<NN>/exercices/*.json`) — لا تكرار،
   وفوق السقف الحالي دائمًا.
2. ‏6 أسئلة فأكثر، ‏≥ 3 نماذج (`expert-exercises.md`)، تدرّج داخليّ 2→3.
3. المُشتِّتات = أخطاء الجدول مُنفَّذة؛ الشرح = القاعدة + تطبيقها + «الخطأ الشائع…» (إلزاميّ في d4).
4. الملفّات: إكمال `02-boss`/`04-defi`، أو `06-defi-concours.json` فوق الموجود؛ نماذج المناظرة في
   `11-annales-subur`.
5. حلّ مزدوج + تمريرة تفنيديّة → `npm run content:check` → `npm run content:qa:strict` → تقرير —
   **ملفّات فقط، دون build/apply/push إلّا بطلب**.
