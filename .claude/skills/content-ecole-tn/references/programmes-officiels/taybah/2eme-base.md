## Français (`fr`)

### Trimestre 1 — Modules : Module 1 « À l'école » · Module 2 « À table » · Module 3 « Flocons d'hiver flocons blancs »

- **Poème/Chanson** :
  - Module 1 : C'est la rentrée
  - Module 2 : Quand on fait des crêpes
  - Module 3 : Vive le vent
- **Communication orale** :
  - Module 1 : Conte « un doudou très jaloux » · Poster « à l'école »
  - Module 2 : Conte « la famille mange tout ? » · Poster « qu'est ce qu'il y a à manger ? »
  - Module 3 : Conte « Le grand voyage de Nino » · Poster « Les activités en hiver »
- **Lexique** :
  - Module 1 : les émotions : content(e) / triste / heureux(se) / gentille — La classe : les tables / les chaises / le tableau / la porte / la fenêtre / le cartable.. (En classe il y a...) — Le matériel de l'enfant : le crayon / la gomme / le livre / Le cahier / la trousse / la feuille.. (dans mon cartable...) — Dans la cour : la balançoire / le toboggan / la marelle / Écrire / lire / dessiner / effacer.. (avec quoi tu....)
  - Module 2 : La famille / les ustensiles / la nourriture / les meubles de la cuisine / les repas / les boissons / Pour mettre la table il faut... / J'ai envie de manger... / J'aimerais boire.... / Dans la cuisine, … / Sur la table, il y a …
  - Module 3 : En hiver / en été. J'ai froid / chaud, un esquimau, un igloo, la banquise, un arbre vert, un arbre dénudé, il est mouillé / sec. Une montagne couverte de neige, Un bonhomme de neige, une boule de neige, des flocons de neige, un igloo, la banquise. *Structure : Quel temps fait-il ? — Des skis, des bâtons, une luge, un chalet, un parapluie, un imperméable, une flaque d'eau, des gouttes (de pluie, d'eau). *Structure : Que font-ils / elles ? / Qu'est ce qu'il / elle porte ?
- **Étude de graphies** :
  - Module 1 : Les graphies [i] [l] [m] [a] [n] [e]
  - Module 2 : Les graphies [o] [d] [é] [c=k] [u] [t]
  - Module 3 : Les graphies [r] [è] [ê] [s] [ou]
- **Lecture texte** (commun) : Lire d'une manière expressive un mini texte composé de mots et des graphies étudiés et répondre à des questions simples de compréhension portant sur le lieu, moment, personnages, vocabulaire
- **Langue** :
  - Module 1 : il-elle / un-une-des / e- ent / est - sont — Mot outil : c'est - dans - il y a — Utiliser les prépositions : sous - sur - dans
  - Module 2 : Ils-elles / Le-la-les — Va + verbe en er / ne ..... pas / et- est — Mot outil : sous / devant / derrière — Utiliser les prépositions : devant / derrière / à côté de
  - Module 3 : La phrase négative : Ne….. pas …. — Emploi de la préposition : Chez + une personne / Je fais + de + nom de jeu / Je mets + un + nom de vêtement — Bon usage des pronoms "au" et "en" + saisons
- **Production écrite** (commun) : Associer la phrase à son image. Compléter un paragraphe ou une phrase avec les mots proposés. Remettre en ordre des étiquettes pour produire une phrase.

Le programme English du T2 (Units 3-5) n'existe que dans la version A.
