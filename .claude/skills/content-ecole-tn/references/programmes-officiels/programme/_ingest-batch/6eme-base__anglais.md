### Structure (fidèle au découpage de la source)

Découpage à deux niveaux : **6 Units** repérables (I → VI) → chacune **5 Lessons** + un **Progress Check** de fin d'unité + un **Project** de fin d'unité. Unit VI est **tronquée** (Lesson 1 seule). Le guide annonce **sept** projets « one at the end of each Unit » ⇒ une **7ᵉ Unit existe** dans l'ouvrage mais tombe au-delà de la troncature. Total transcrit : **26 leçons** (5×5 + 1).

Chaque fiche-leçon du guide énonce systématiquement : _Objectives_ (« Upon completion of the lesson, the learners will be able to… »), _Vocabulary_, _Grammar_ (parfois _Function_), puis les corrigés des rubriques d'activités.

#### Unit I — Entertaining Myself (p. 9–17)

- **Lesson 1 : Let's Have Fun** (p. 9–10) — Obj. : nommer des activités de loisir ; parler de ses loisirs préférés. Vocab. : fishing / museum / go camping / riding a bike / gardening. Grammaire : _because_ ; _like/enjoy + verb + -ing_ (expression du goût et du dégoût : enjoy/like vs hate/dislike + -ing). Activités : label pictures, tableau à cocher, dialogue en binôme, _Let's write_ (paragraphe loisirs), _Let's pronounce_ (odd-one-out /aɪ/), _Let's play_.
- **Lesson 2 : My Free Time Activities** (p. 11–12) — Obj. : nommer des loisirs ; écrire sur ses loisirs préférés. Vocab. : drawing / gardening / shopping. Grammaire : adverbes de fréquence (always / sometimes / usually / never — accord _+s_ à la 3ᵉ pers.) ; prépositions _on / in_ (on the weekend, on Monday morning ; in the morning/afternoon/evening — consolidation). Activités : reading + match, _Let's speak_ (bulles à compléter, linkers/ponctuation), _Let's pronounce_ (tri Same/Different).
- **Lesson 3 : A Day Out** (p. 13) — Obj. : nommer des activités de plein air ; faire et répondre à des suggestions. Vocab. : bored / hide and seek / tired / zoo / far. Grammaire : _Let's + infinitive_ / _How about + V-ing?_ / _What about + V-ing?_. Fonction : **making suggestions**. Activités : _Let's Listen_ (tapescript Adam/Linda) + comprehension + remise en ordre, role play, _Let's pronounce_ (/s/ vs /z/).
- **Lesson 4 : Let's go on a picnic** (p. 14–15) — Obj. : nommer des objets de pique-nique ; parler des activités de pique-nique. Vocab. : get ready / river / wash / arrive / rope skipping / fly a kite. Grammaire : linkers _and – but_ ; adjectifs possessifs _his – her – their_ (I–my … they–their). Activités : reading (true/false + match), Think-Pair-Share, _Let's write_ (5 phrases → paragraphe, linkers/ponctuation), _Let's play_.
- **Lesson 5 : Pass the Ball** (p. 16–17) — Obj. : nommer des loisirs ; faire offres et requêtes avec _Would you like…?_. Vocab. : lose / team / win / kick / run. Grammaire : _Would you like…?_ (inviting = Would you like + to + verb ? ; making a suggestion = Let's + verb). Activités : label pictures, reading (beach) + comprehension, role play (Speaker 1/2), _Let's pronounce_ (match des sons), _Let's play_ (sports matching).
- **Progress Check 1** (p. 17) — Reading Comprehension (tableau Name/Age/Town/Favourite activities ; true/false ; blanks) ; Language (pronoms ; would you like/how about/let's) ; Writing (remise en ordre de phrases + brainstorming → paragraphe).

#### Unit II — Keeping Fit (p. 18–25)

- **Lesson 1 : I'm Healthy, I'm Happy** (p. 18) — Obj. : classer les aliments sains/malsains ; donner des conseils pour rester en bonne santé. Vocab. : fruits / vegetables / lots / healthy / unhealthy. Grammaire : _should / shouldn't_ (giving advice). Activités : label + tick/cross healthy foods, questionnaire + dialogue de conseil, _Let's pronounce_, _Let's play_.
- **Lesson 2 : Eat well, Live well** (p. 19–20) — Obj. : décrire les éléments d'une alimentation saine ; dire aux autres de faire des choix sains avec _must / mustn't_. Vocab. : vegetables / teeth / egg / keep fit / bad for. Grammaire : _must / mustn't_. Activités : _Let's Listen_ (tapescript Anna) + comprehension, _Let's write_ (health poster en groupe de 4), _Let's pronounce_ (/ɪ/ vs /iː/), _Let's play_.
- **Lesson 3 : I'm Healthy, I'm Fit** (p. 20–21) — Obj. : décrire personnes en forme / hors forme ; distinguer habitudes saines et malsaines. Vocab. : doughnuts / fit / fat / tired / excited / sad / unhappy. Grammaire : _should/shouldn't_ ; _must/mustn't_ ; **can / can't** (expressing ability / inability). Activités : spot-the-difference + comparaison (slim/fat/thin/fit), reading (true/false + match), _Let's speak_, _Let's pronounce_, _Let's play_ (healthy food).
- **Lesson 4 : How are you Feeling?** (p. 22) — Obj. : parler de trois maux (ailments) ; donner des conseils santé. Vocab. : toothache / stomachache / sick / tired / headache. Grammaire : _have a + ailment_ ; _to be + adjective_. Activités : listen/repeat/mime, _Let's read + Let's Listen_ simultanés + comprehension, role play patient/docteur, _Let's pronounce_, _Let's play_ (grille de mots-mêlés).
- **Lesson 5 : How to Stay Fit** (p. 23–24) — Obj. : identifier des activités pour rester en forme ; catégoriser les aliments ; définir un menu sain. Vocab. : oil / sweets / egg / meat / pasta / yoghurt. Grammaire : _how many_ (+ countable) / _how much_ (+ uncountable). Activités : reading (true/false + blanks), _Let's write_ (menu sain en binôme à partir de la food pyramid), _Let's pronounce_, _Let's play_.
- **Progress Check 2** (p. 24–25) — Listen/Read : comprehension (tableau Vegetables/Fruit ; true/false) ; Language (conseils should/shouldn't ; blanks) ; Writing (matching, remise en ordre, catégorisation, phrases guidées).

#### Unit III — Having fun in the seasons (p. 26–32)

- **Lesson 1 : The Four Seasons** (p. 26) — Obj. : nommer les quatre saisons ; décrire les saisons (météo, activités). Vocab. : seasons / months / year / autumn / spring / summer / winter / the twelve months of the year. Grammaire : **adjectif dérivé** _Noun + y = adjective_ (cloud + y = cloudy ; règle du redoublement de la consonne finale C-V-C : foggy, sunny). Activités : nommer saisons + 12 mois, compléter lettres manquantes (windy/cloudy/foggy/snowy/sunny), _Let's speak_ (saison préférée + _because_), _Let's play_ (classer mois par saison + couleur).
- **Lesson 2 : Having Fun in Spring** (p. 27) — Obj. : identifier les caractéristiques du printemps ; parler des activités de printemps. Vocab. : sky / clear / outside / air / butterfly / next / catch / collect. Grammaire : **futur** _will / 'll_ ; _will not / won't_ ; future words (tomorrow, next year/spring/month/week…). Activités : reading + comprehension + gap-fill, _Let's write_ (guidé par questions), _Let's pronounce_ (/ð/ « the »), _Let's play_ (former des mots).
- **Lesson 3 : Having Fun in Summer** (p. 28–29) — Obj. : identifier les caractéristiques de l'été ; parler des activités estivales. Vocab. : shorts / swimsuit / beach / sandcastle. Grammaire : _to be on holidays_ ; _will / will not + verb_. Activités : reading + match, tri de vêtements, _Let's write_ (email), _Let's pronounce_ (/iː/ + tri S/D), _Let's play_ (comparer 2 images).
- **Lesson 4 : Having Fun in Autumn** (p. 30) — Obj. : identifier les caractéristiques de l'automne ; parler des activités d'automne. Vocab. : warm / cold / windy / go back / meet. Grammaire : **comparatif** (short) _adjective + er (+ than)_ (cold+er = colder than) ; _What's the weather like?_ / _It's + adjective_. Activités : reading (true/false), practise (hotter/colder/warmer/slower/longer), _Let's speak_ (activités + _because_), _Let's pronounce_, _Let's play_.
- **Lesson 5 : Having Fun in Winter** (p. 31–32) — Obj. : identifier les caractéristiques de l'hiver ; parler des activités hivernales. Vocab. : gloves / socks / fresh air / boots / between / sweater. Grammaire : _Verb + -ING = Activity_ (gérondif-activité). Activités : reading + comprehension + vêtements d'hiver, _Let's write_ (post « winter is the best season »), _Let's pronounce_ (tri S/D + familles de sons season/play/cold), _Let's play_ (mots en « S »).
- **Progress Check 3** (p. 32) — Reading Comprehension (tableau Season/weather ; true/false) ; Language (adjectifs météo ; present continuous / futur ; saisons) ; Writing (mots, phrases futur/comparatif, ID card de la saison préférée).

#### Unit IV — Caring (p. 33–41)

- **Lesson 1 : Let's help each other** (p. 33–34) — Obj. : lister des exemples d'entraide ; identifier des façons d'aider les autres. Vocab. : care / share / take care / wash the dishes / dirty / tidy / help / each other. Grammaire : adjectifs possessifs (I–my … they–their). Activités : commenter images, _Let's Listen_ (« I love my family ») + remise en ordre, practise (your/my/our/their/her/his), _Let's speak_ (questionnaire + fréquence), _Let's pronounce_ (/h/ audible ou non), _Let's play_ (sons initiaux).
- **Lesson 2 : Sharing is caring** (p. 35–36) — Obj. : comparer lieux et personnes au superlatif ; écrire sur les activités de son quartier. Vocab. : city / come together / neighbour / to clean up / alone / poor / old / young / cross. Grammaire : **superlatif** _the + short adjective + est_ (the clean+est = the cleanest). Activités : reading (email sender/receiver) + comprehension, practise (largest/hottest/coldest/nicest/tallest/smartest/shortest), _Let's write_ (email), _Let's play : Boardgame_ (dé, lire une phrase / décrire une image).
- **Lesson 3 : We are stronger together** (p. 37) — Obj. : décrire cinq métiers ; parler des façons d'aider les autres. Vocab. : village / job / agree / work / say / sick / learn / dice / builder / carpenter / farmer / gamer / participate / competition. Grammaire : **dérivation** _Verb + er = Noun_ (farm+er = farmer). Note du guide : « not all jobs are formed as verb + er ». Source du texte : adapté de _Disagreement Among Occupations_, FreeKidsBooks. Activités : reading (tableau Who?/What he-she does), practise (builder/teacher/baker/driver/players/gamer), _Let's speak_, _Let's pronounce_, _Let's play_ (métiers ↔ images).
- **Lesson 4 : Caring for my school** (p. 38–39) — Obj. : dire comment prendre soin de son école ; écrire une charte de classe (classroom charter). Vocab. : clean / collect / tidy-up / mop / blackboard / finish / walk in / floor. Grammaire : **futur (affirmatif/négatif)** will / won't. Activités : compléter bulles, _Let's Listen_ (Headmaster announcement) + comprehension (tableau Name/Activity), practise (will/will not), _Let's write_ (5 règles de classe), _Let's pronounce_ (/ʃ/ et /ʧ/ tongue twisters), _Let's play_.
- **Lesson 5 : Caring for my pet** (p. 40–41) — Obj. : nommer six animaux de compagnie ; parler de comment prendre soin des animaux. Vocab. : take care of / vet / sick / twice / give / hit. Grammaire : **impératif** ; **prohibition** _Don't + verb_ (ordres/consignes ; interdiction : chose non autorisée, règle existante, danger/risque). Activités : nommer 6 pets (audio Year 4), reading + comprehension, practise (Don't play / Be / Brush / Read / Eat), _Let's write_ (note dos/don'ts), _Let's pronounce_ (/eɪ/ « Day » vs /aɪ/ « Why »), _Let's play_ (mots-mêlés).
- **Progress Check 4** (p. 41) — Comprehension (choix, true/false) ; Language (futur affirmatif/négatif ; blanks) ; Writing (mots, lettres manquantes, paragraphe de présentation d'un animal — N.B. pronoms he/she + possessifs his/her).

### Compétences & progression (telles qu'énoncées par le guide)

- **Quatre compétences** structurent chaque leçon via les rubriques récurrentes : **écoute/lecture** (_Let's Listen_ / _Let's read_ → comprehension), **grammaire en contexte** (_Let's complete the rule_ → _Let's practise_, en 2 temps : observation/conceptualisation puis consolidation en contexte communicatif), **oral** (_Let's speak_ : role play / simulation), **écrit** (_Let's write_ : du niveau phrase vers de courts textes — posts, emails, notes ; approche par _genres_ : genre / audience / purpose). S'y ajoutent **prononciation** (_Let's pronounce_) et **jeu** (_Let's play_, wrap-up).
- **Progression grammaticale (fil directeur transcrit)** : goûts (like/enjoy + -ing) → fréquence (always/sometimes/never, +s 3ᵉ pers.) → suggestions (Let's / How about / What about) → invitations (Would you like…?) → conseil/obligation (should/shouldn't, must/mustn't) → capacité (can/can't) → quantité (how many/how much) → dérivation adjectivale (Noun+y) → **futur** (will/won't) → **comparatif** (adj+er than) → **superlatif** (the …est) → dérivation nominale (Verb+er) → impératif & prohibition (Don't) → droits (have/has the right to) → **simple past** (verbes réguliers/irréguliers, -ed endings) → prépositions + simple past (Unit VI). Progression cohérente présent → futur → passé, du mot vers le texte.
- **Life skills** explicitement ciblées par rubrique (Introduction) : acceptance, respect for differences, communication (effective speaking, active listening, expressing ideas), creativity, critical thinking, decision-making, empathy, perspective-taking, collaboration, focus & self-control, problem-solving, turn taking, self-confidence, coping with emotions, stress management.
- **Évaluation** : un **Progress Check** clôt chaque unité (Reading Comprehension / Language / Writing) ; un **Project** clôt chaque unité (posters, food pyramids, présentations PowerPoint, vidéos, histoires, journaux/diaries) évalué via une _evaluation grid_ apprenant décrite p. 8.

### Notions / contenus clés (agrégés)

- **Grammaire** : présent simple + adverbes de fréquence ; -ing (goûts, activités) ; modaux should/shouldn't, must/mustn't, can/can't, Would you like…? ; quantifieurs how many/how much ; dérivation Noun+y (adjectif) et Verb+er (nom) ; comparatif court (-er than) & superlatif court (the …est) ; futur will/won't ; impératif & prohibition (Don't) ; have/has the right to ; **simple past** (réguliers + irréguliers, formes -ed) ; prépositions de lieu ; linkers and/but/because ; adjectifs & pronoms possessifs.
- **Lexique thématique** (par unité) : loisirs & plein air ; santé/alimentation/forme/maux ; saisons/mois/météo/vêtements ; entraide/métiers/école/animaux ; fêtes & célébrations (Children's/Mother's/Teacher's Day, New Year, end-of-year party) ; commerces & achats.

### Types d'exercices / activités (relevés dans la source)

- Étiquetage d'images (label the pictures) ; tableaux à cocher/croiser (tick/cross).
- Compréhension écrite/orale : gist + scanning ; yes/no & Wh-questions ; true/false (justifié) ; complétion de phrases avec mots du texte ; transfert d'infos vers un tableau ; matching (parties de phrases, images↔mots) ; questions de référence ; remise en ordre de phrases.
- Grammaire : compléter la règle (conceptualisation) puis _practise_ (gap-fill / choix / transformation).
- Oral : **role play** / simulation, questionnaires, bulles à compléter, **Think–Pair–Share**.
- Écrit : phrase → paragraphe ; genres = email, post, note, health poster, classroom charter, menu, ID card d'une saison.
- Prononciation : écouter/répéter, classer par son, minimal pairs, tongue twisters, odd-one-out.
- Jeux : _Let's play_ (mots-mêlés, roues de mots, associations, **board games** au dé), _Projects_ de fin d'unité, _Progress Checks_ d'évaluation.

### Incertitudes de transcription

Le nombre total de leçons de l'ouvrage complet (probablement 7×5 = 35 + 7 Progress Checks) n'est **pas** vérifiable ici — 26 leçons seulement sont transcrites.
- **Intitulé de la Unit VII inconnu** (déduit comme existante par la mention « seven projects », jamais nommée dans le texte capté).
- **Titre exact des projets par unité non listé** individuellement (seule la nature générale est donnée p. 8 : posters, food pyramids, PowerPoint, vidéos, stories, diaries). L'appariement projet↔unité n'est pas explicite dans la partie captée.

- Le guide mentionne un **audio « Year 4 »** réutilisé (Unit IV Lesson 5) : dépendance à des supports externes non inclus.
