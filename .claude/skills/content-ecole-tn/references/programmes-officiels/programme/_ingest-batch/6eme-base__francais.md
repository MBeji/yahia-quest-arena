### Architecture générale du programme (Guide `521612`, fidèle au SOMMAIRE et au ch. 4)

Le projet fédérateur de l'année est **« Notre projet de sixième » = le journal de la classe** (au minimum 3 numéros), autour duquel gravitent tous les apprentissages (pédagogie de projet + approche par compétences + situations-problèmes).

Organisation modulaire (ch. 4, p.14-16) :

- **1 module d'évaluation des prérequis** (2 à 3 semaines) + 1-2 journées de présentation (projet de l'année, manuel, cahier).
- **8 modules d'apprentissage** (2 semaines / 8 séances chacun, 8 h/semaine).
- **4 modules d'intégration** (« journées-paliers », 2 jours).
- **4 modules d'évaluation-remédiation** (1 semaine / 4 séances).
- Ces modules sont regroupés en **4 unités d'apprentissage** ; plan-type d'une unité = **2 modules d'apprentissage + 1 module d'intégration + 1 module d'évaluation-remédiation**.
- **Phase 1** = modules 1-4 (2 projets d'écriture, soit 1 par unité). **Phase 2** = modules 5-8 (4 projets d'écriture, soit 1 par module). Journée-palier supplémentaire au terme des modules 5 et 7.

Découpage annuel en 4 unités (p.15-16) :

- **Unité 1** : M1 + M2 → intégration → évaluation-remédiation.
- **Unité 2** : M3 + M4 → intégration → évaluation-remédiation.
- **Unité 3** : M5 (+ journée-palier suppl.) + M6 → intégration → évaluation-remédiation.
- **Unité 4** : M7 (+ journée-palier suppl.) + M8 → intégration → évaluation-remédiation.

Emploi du temps (2 modèles, p.20-22) : la journée de 8 activités type comprend Mise en train (poème/chant), Expression orale, Lecture-compréhension, Lecture-fonctionnement, Grammaire, Conjugaison, Orthographe, Projet d'écriture, Lecture documentaire (J7), Lecture-action, Page vocabulaire (J8), Bibliothèque de classe, Lecture suivie, Dictée/Autodictée. Seules les séances de **projet d'écriture** diffèrent entre phase 1 et phase 2.

### Structure (fidèle au découpage en 8 modules d'apprentissage)

Chaque module = 1 thème du programme officiel + son sous-thème (« slogan ») + un projet d'écriture + 5 textes de lecture (dont 1 documentaire J7 + 1 lecture-action) + 2 poèmes/chants + 1 page vocabulaire (J8) + les notions de langue (grammaire / conjugaison / orthographe). Correspondances thème↔sous-thème↔module d'après le « Tableau de répartition » (p.13) et le « Tableau de bord » (5-2, p.18-19).

- **Module 1 — Thème « Travail » · sous-thème « Travailler pour s'épanouir »** (p.9-10, manuel) :
  - **Projet d'écriture n°1** (unité 1) : « Je raconte un événement en rapport avec le thème du travail ».
  - **Objectifs de communication (oral)** : Informer/s'informer ; Décrire/raconter un événement ; Situer des lieux ; Justifier un choix.
  - **Poèmes** : _Au travail_ (Auguste Brizeux) ; _Le boulanger_ (Raymond Richard).
  - **Textes** : _Apprentie comédienne_ (d'après Anne Mirman) ; _Une parfumeuse en herbe_ ; _Témoignages_ ; _Quel métier choisir ?_ (documentaire) ; _Un tableau découpé_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : reconnaître et utiliser les déterminants, les noms et les pronoms personnels.
  - **Conjugaison** : reconnaître les trois temps (présent, passé composé, futur) ; utiliser des verbes à l'infinitif et des verbes conjugués.
  - **Orthographe** : utiliser l'infinitif après les prépositions à / de / par / pour / sans.

- **Module 2 — Thème « Médias et nouvelles technologies » · sous-thème « Communiquer avec les autres »** (p.18, guide) :
  - Projet : partie de l'unité 1 (projet n°1 finalisé après M2 ; J1 de M2 = rappel/mise au point du projet).
  - **Objectifs de communication (oral)** : Informer/s'informer ; Exprimer un point de vue ; Porter un jugement ; Décrire une scène.
  - **Poèmes** : _Les machines_ ; _Moi j'irai dans la lune_ (guide) / _Conversation_ (manuel).
  - **Textes** : _Le vieux robot_ ; _Une télé pas comme les autres_ ; _Le monde du journalisme_ ; _Faire « son journal »_ (documentaire) ; _Faire la maquette de la « Une » d'un journal pour enfants_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : reconnaître et utiliser les déterminants possessifs et les déterminants démonstratifs.
  - **Conjugaison** : conjuguer les verbes en -er, les verbes usuels en -ir et les verbes usuels du 3ème groupe à l'impératif.
  - **Orthographe** : homophones **a / à**.

- **Module 3 — Thème « Paix et tolérance » · sous-thème « Accepter les autres »** (p.18, guide) :
  - **Projet d'écriture n°2** (unité 2, partagé M3+M4) : « Je raconte un événement en rapport avec le thème de la solidarité et je fais parler des personnages dans mon récit ».
  - **Objectifs de communication (oral)** : Décrire ; Raconter ; Informer/s'informer ; Exprimer un point de vue.
  - **Poèmes** : _Marie et moi_ ; _Donne autour de toi_.
  - **Textes** : _Le petit lapin (tout) blanc_ ; _Le nouveau costume_ ; _L'Indien qui ne savait pas courir_ ; _Les ressemblances et les différences entre les (êtres) humains_ (documentaire) ; _Le bouquet de bienvenue_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : reconnaître et utiliser l'adjectif épithète et l'adjectif attribut.
  - **Conjugaison** : conjuguer les verbes **être** et **avoir** au futur et au passé composé.
  - **Orthographe** : homophones **son / sont**.

- **Module 4 — Thème « Solidarité et citoyenneté » · sous-thème « S'entraider pour mieux réussir »** (p.18, guide) :
  - Projet : finalisation du projet n°2 (unité 2) ; J1 de M4 = rappel/mise au point.
  - **Objectifs de communication (oral)** : Décrire des scènes ; Porter un jugement ; Prendre position ; Justifier un point de vue.
  - **Poèmes** : _L'homme et le chien_ ; _Si vous n'avez rien à me dire_.
  - **Textes** : _L'écureuil et l'escargot_ ; _Le courage d'un jeune apprenti_ ; _Un bon compagnon_ ; _« L'école, un droit pour tous »_ (documentaire) ; _Le jeu du Petit Poucet_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : utiliser la forme (phrase) négative **ne… plus / ne… jamais**.
  - **Conjugaison** : conjuguer les verbes usuels du type **finir** au passé composé et au futur.
  - **Orthographe** : homophones **et / est**.

- **Module 5 — Thème « Environnement » · sous-thème « Sauver la nature »** (p.19, guide) :
  - **Projet d'écriture n°3** (phase 2, module) : « Je raconte un événement en rapport avec le thème de l'environnement et j'intègre un passage descriptif dans mon récit ».
  - **Objectifs de communication (oral)** : Donner des conseils ; Informer/s'informer ; Raconter ; Décrire une personne.
  - **Poèmes** : _Qu'elle est belle la terre !_ ; _L'arbre volant_.
  - **Textes** : _Sauvez Keiko_ ; _La ménagère de la mer_ ; _Les parapluies_ ; _Comment éviter le gaspillage de l'eau_ (documentaire) ; _Fabriquer une famille de cygnes_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : utiliser le complément essentiel et le complément non essentiel.
  - **Conjugaison** : conjuguer les verbes du type **prendre** et **mettre** au passé composé et au futur.
  - **Orthographe / accord** : accorder le verbe avec son sujet.

- **Module 6 — Thème « Santé et bien-être » · sous-thème « Être en forme et mieux se porter »** (p.19, guide) :
  - **Projet d'écriture n°4** (phase 2) : « Je raconte un événement en rapport avec le thème de la santé et j'intègre des répliques et un passage descriptif dans mon récit ».
  - **Objectifs de communication (oral)** : Décrire une situation/une scène ; Donner un avis/le justifier ; Informer/s'informer.
  - **Poèmes** : _Un matin_ ; _Le vélo (à) mille-pattes_.
  - **Textes** : _La jeune acrobate_ ; _Des vignettes… pour un ballon_ ; _La lettre de Jean_ ; _Comment lutter contre le stress_ (documentaire) ; _Faire des exercices d'étirement_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : utiliser le complément de lieu.
  - **Conjugaison** : conjuguer les verbes **aller** et **faire** au passé composé et au futur.
  - **Accord** : accorder en genre et en nombre les adjectifs du type heureux→heureuse, mûr→mûre.

- **Module 7 — Thème « Loisirs » · sous-thème « Profiter de son temps libre »** (p.19, guide) :
  - **Projet d'écriture n°5** (phase 2) : « J'écris une lettre à un ami ou à un parent pour lui parler de mes loisirs ».
  - **Objectifs de communication (oral)** : Informer/s'informer ; Décrire un lieu, un personnage ; Exprimer un avis/une préférence ; Décrire un mode de vie.
  - **Poème** : _Dans le grenier de ma grand-mère_.
  - **Textes** : _La lettre de Paul_ ; _La lettre de Daniel_ ; _La lettre de Monsieur Crêpe_ ; _Peinture, sculpture et théâtre de marionnettes_ (documentaire) ; _Jouer au nombre pensé_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : utiliser le complément de temps.
  - **Conjugaison** : conjuguer les verbes **dire, lire, écrire** au présent, au passé composé, au futur et à l'impératif.
  - **Accord** : accorder en genre et en nombre les adjectifs du type neuf/neuve, gentil/gentille, beau/belle, bon/bonne.

- **Module 8 — Thème « Culture et découverte du monde » · sous-thème « Découvrir d'autres modes de vie »** (p.19, guide) :
  - **Projet d'écriture n°6** (phase 2) : « Je raconte un événement en rapport avec le thème de la culture et de la découverte du monde et j'intègre un passage descriptif dans mon récit ».
  - **Objectifs de communication (oral)** : Décrire-raconter ; Exprimer un sentiment / une préférence / un refus ; Émettre des hypothèses.
  - **Poèmes** : _Le vent a fait le tour du monde_ ; _Pour faire le portrait d'un oiseau_ ; _Une graine voyageait_.
  - **Textes** : _Le grand voyage_ ; _Une sortie de classe au Canada_ ; _L'île au trésor_ ; _Quelques fêtes d'autres pays_ (documentaire) ; _Fabriquer la tour Eiffel_ (lecture-action) ; _Page vocabulaire_.
  - **Grammaire** : utiliser le complément de manière.
  - **Conjugaison** : conjuguer les verbes **vouloir** et **pouvoir** au présent.
  - **Accord** : accorder le participe passé employé avec **être**.

> Remarque du guide (p.19) : les **verbes pronominaux usuels** du programme officiel sont introduits à l'oral et à l'écrit et étudiés **tout au long de l'année**, non pas en une seule séance.

### Module d'évaluation des prérequis (prérequis de 5ème à réviser — manuel `121614`, p.5)

Annoncés comme « connaissances nécessaires pour réussir la 6ème » :

- **Grammaire** : la phrase et ses deux groupes GNS / GV ; l'accord sujet-verbe ; nom propre / nom commun ; phrases déclarative, interrogative, exclamative, affirmative, négative (ne… pas).
- **Conjugaison** : verbes en -er (type dessiner) au présent, futur, passé composé ; être et avoir au présent ; verbes du type finir au présent ; prendre, mettre, aller, faire, lire, écrire, dire au présent.
- **Orthographe** : pluriel des noms en -al, -eau, -eu ; féminin des noms en -ée / -ue / -ie ; noms en -é, -et, -er, -ez ; g = gu / g = j ; orthographe de mots-outils et expressions invariables.

### Compétences & progression (Guide `521612`, ch. 2, 7, 8, 9)

**Trois compétences terminales disciplinaires** (français) + compétences transversales.

- **Compétence terminale de l'ORAL** (savoir-communiquer oralement : écouter, comprendre, parler) : « Intégrer les acquis (savoirs, savoir-faire, savoir-être) dans des énoncés oraux cohérents et intelligibles pour : manifester sa compréhension d'un message oral ; rendre compte d'un événement de la vie quotidienne ; communiquer avec autrui dans une situation significative. » Composantes 6ème : **Écouter** (écouter l'autre) · **Comprendre** (manifester sa compréhension d'un message oral) · **Parler** (établir un contact social ; intégrer les règles de prononciation du français ; intégrer les règles de fonctionnement de la langue).
- **Compétence terminale de la LECTURE** (savoir-lire) : « Intégrer les mécanismes de base de la lecture (décodage et encodage) pour lire des textes variés et rendre compte de sa compréhension. » Composantes : **Lire** (intégrer les mécanismes de base d'une lecture vocale intelligible ; mettre en œuvre des stratégies de lecture pour construire le sens ; apprécier une œuvre complète).
- **Compétence terminale de l'ÉCRIT** (savoir-écrire) : « Intégrer les acquis (savoirs, savoir-faire, savoir-être) pour produire des énoncés écrits variés et cohérents. » Composantes : **Écrire** (intégrer les acquis liés aux règles de fonctionnement de la langue ; mettre en œuvre des stratégies d'écriture : planifier – mettre en texte – réviser).

**Compétences transversales** (ch. 2, tableau p.10) — 4 ordres :

- _Ordre de la communication_ : communiquer de façon appropriée.
- _Ordre méthodologique_ : mettre en œuvre des méthodes de travail efficaces ; exploiter les TIC.
- _Ordre individuel et social_ : travailler en coopération ; réaliser un projet.
- _Ordre intellectuel_ : exploiter l'information ; résoudre des problèmes ; exercer sa pensée critique.

### Types d'exercices / activités (relevés dans les sources)

- **Oral** : mise en train (poème/chant : articuler, réciter de façon expressive, interpréter en respectant la mélodie) ; phonétique corrective (oppositions [y/u], [y/i], [p/b], [f/v], voyelles orales/nasales, liaisons, intonation déclarative/interrogative/exclamative/impérative) ; expression orale via situations-problèmes et **carte d'exploration de pistes** (J1) ; jeux de rôles, dramatisation ; 4 types de situations : exploration, apprentissage systématique/structuré, intégration, évaluation.
- **Lecture** : lecture oralisée ; lecture-compréhension (anticipation/hypothèses, approche globale, approche analytique, vocabulaire, synthèse-dépassement) ; lecture-fonctionnement (structure et types d'écrit : récit, BD, interview, enquête, dialogue, affiche, lettre, dictionnaire) ; lecture documentaire (J7) ; lecture-action (texte injonctif/prescriptif → fabriquer un objet/jeu) ; page vocabulaire J8 (« La boîte à mots », « Jeux de mots ») ; **bibliothèque de classe** (≥ 1 livre/mois) et **lecture suivie** (4 contes en 2 épisodes chacun — voir vocabulaire officiel).
- **Écrit** : **projet d'écriture** (6 projets → journal de classe) en étapes planification / mise en texte / révision / correction / réécriture-finalisation ; produire un récit, insérer un passage descriptif, insérer des répliques/un dialogue, écrire une lettre ; dictée / autodictée. Fiches d'expression écrite = catégorie unique « entraînement/remédiation ».
- **Cahier d'activités** (`121615`) : rubriques « Je m'entraîne », « Je produis », « Je fais le point » ; exercices de classement, complètement de texte/dialogue, reliage (flèches), transformation (pluriel, remplacement de sujet), repérage de la structure du récit (préambule / début / suite / fin). Découpage interne du cahier en **Journées** (« Module 1 Journée 1/2… »).
- **Logos élève (manuel, p.8)** : « Je récite / Je lis / Je lis pour m'informer et me documenter / Je lis pour agir / J'observe / Je retiens / J'ouvre la boîte à mots / Je joue avec les mots » (manuel) ; « Je m'entraîne / Je produis / Je fais le point » (cahier).

### Incertitudes de transcription

- **Toutes les sources sont tronquées** (70000 chars chacune, sauf le P.V. `521613` complet mais purement bordereau). Le Guide `521612` s'arrête **en pleine phrase** (« Les modalités de réalisation du… ») **avant** les chapitres 11-14 (intégration, évaluation, évaluation des prérequis détaillée, et surtout **les unités d'apprentissage détaillées p.79-183**) : ces parcours séance-par-séance ne sont **pas** dans le corpus lu.
- **Layout multi-colonnes cassé** : les tableaux du guide (compétences transversales p.10, tableau de bord oral/écrit p.17-19, emplois du temps p.20-22) et le « TABLEAU DE BORD » du manuel (p.6-7) sortent en texte **interleavé** (cellules empilées hors ordre). Les associations module↔notion↔texte ont été **reconstituées par recoupement** entre guide (5-2) et manuel ; fiables sur le fond, mais l'ordre exact de certaines lignes n'est pas garanti au caractère près.
- **Poèmes/chants du module 2** : le manuel imprime « Les machines / Conversation », le guide « Les machines / Moi j'irai dans la lune » — léger écart entre supports (signalé, non tranché).
- **Fichier-classe `521616`** : seule la partie **Grammaire – fiches de base (jusqu'à ~B8)** est lisible ; les répertoires Conjugaison / Orthographe / Expression écrite et toutes les fiches d'intégration/remédiation sont **au-delà de la troncature**.
- Quelques coquilles OCR-libres du texte source d'origine (ex. « pensé critique » pour « pensée critique », « son arabes », symboles phonétiques approximatifs `[ ȶ / e ]`, `[ ¡j / œj ]`) sont **reproduites/normalisées** sans invention.
- La **pagination** indiquée renvoie aux numéros imprimés lisibles dans la couche-texte ; l'imprimeur précise que cette pagination « joue un rôle secondaire ».
