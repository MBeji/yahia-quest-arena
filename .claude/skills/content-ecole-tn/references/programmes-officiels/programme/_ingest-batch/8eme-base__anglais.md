### Architecture d'ensemble (fidèle à la source)

- Le programme est organisé en **5 modules**, chacun de **5 leçons** (soit 25 leçons). Chaque module s'ouvre sur un **Module Map** (topics · skills and strategies · grammar & functions · vocabulary · projects) et se clôt sur un **Mock Test** + une **Checklist** d'auto-évaluation (« Check your learning — A checklist of the abilities developed in Module N »).
- **Structure interne de chaque leçon** (Teacher's Book, Part 2) : **LET'S GET READY (LGR)** → **LET'S DISCOVER (LD)** → **LET'S COMMUNICATE (LC)**, chaque partie subdivisée en _steps_. La **leçon 1** de chaque module est une « Review and introductory lesson » (réactive les acquis + met en place l'histoire du module).
- **Fil narratif** (Teacher's Book, « Sequence in presenting the materials ») : les **modules 1–3** racontent l'histoire d'un adolescent tunisien invité à Londres par une famille anglaise (les Brown) ; le **module 4** relate le séjour d'une famille anglaise passant ses vacances de Pâques à Édimbourg (Écosse) ; le **module 5** a pour thème « Relationships » (parents, amis, environnement, animaux).
- Approche : communicative, centrée apprenant, par les **4 skills** (reading, writing, listening, speaking) + **strategies** (learning how to learn) + **project work** (une production par module). Évaluation : self-evaluation + formative.

### Structure détaillée (les 25 leçons)

**Module 1 — les Brown / présentations / voyage à Londres** (SB p.7–30)

- **Lesson 1** (p.7–10) : _Meet the Browns_ — Review and introductory lesson. Objectifs (TG) : lire et comprendre l'information explicite ; écrire quelques lignes pour présenter des personnes sur une photo. Grammar & functions : Yes/no questions ; Wh-questions ; inviter (_Would you like…?_) ; présenter (_This is…_). Vocab : baker, become, peaceful, quiet, go back, sell, leave.
- **Lesson 2** (p.11–13) : _A letter from an English friend_. Objectifs : lire une lettre pour le sens global (reading for gist) ; reconnaître expéditeur/destinataire ; identifier les traits d'une lettre informelle. Skills : reading for gist ; recognise sender and receiver. Grammar : révision des temps + découverte du **simple future**. Vocab : make the beds, set the table, bake, still, begin, both, cheerful, expect, free, hope, regret, diary, plan, promise.
- **Lesson 3** (p.14–18) : _Can she go to London?_ Objectifs : identifier les locuteurs ; identifier le topic ; reconnaître les discourse markers (adverbes séquentiels) ; écrire un court texte argumentatif. Grammar & functions : exprimer l'aptitude au futur (_will be able to_) ; sequential adverbs ; parler d'événements futurs. Spelling : dictation. Pronunciation : stressed syllable. Vocab : miss, call someone, cup of tea, wait, discuss, of course, stay, don't be long!, all day long, improve.
- **Lesson 4** (p.19–22) : _Preparing for the trip_. Objectifs : identifier les locuteurs au téléphone ; travailler en groupe ; produire une conversation téléphonique sur modèle ; inférer l'humeur du locuteur ; stratégie pour compléter un texte lacunaire. Functions : révision _telling the time_ ; exprimer la surprise (_How + adjective_ / _What + noun_ / _What a surprise!_). Vocab : telephone box, post office, mobile phone, stamp, postman, reset a watch, travel, give a call, let someone know, book a flight.
- **Lesson 5** (p.23–29) : _London wonders_. Objectifs : associer textes et visuels ; présenter l'information sous une autre forme ; écrire des entrées de journal (diary) à partir de notes ; produire une lettre informelle (process writing strategy). Functions : décrire des lieux ; dire les dates. Vocab : build/built, tall column, tower, river, bridge, pets, go on a walk, traffic, palace, royal, kill, jail, bored, dome, sight, century.
- **Mock Test + Checklist** (p.30).

**Module 2 — English secondary schools / vie scolaire** (SB p.34–58)

- **Lesson 1** (p.35–39) : _English secondary schools (1)_ — Review and introductory lesson. Objectifs : inférer le sens de mots inconnus par le contexte ; écrire quelques lignes sur ses projets futurs ; identifier les stratégies de collecte d'information. Grammar : le futur avec _going to_ ; parler de ses projets. Vocab : take the bus, take pictures, show someone round a place, dining hall, library, assembly hall, main entrance, staff room, hit the ball, take off, fall.
- **Lesson 2** (p.40–43) : _English secondary schools (2)_. Objectifs : compléter un texte lacunaire / une conversation / un rapport en transférant l'information d'une source à l'autre (timetable ↔ report ↔ conversation) ; écrire des rapports sur son école. Functions : tell the time ; talk about age ; _How long…?_ (durée d'une activité). Vocab : state schools, private schools, nursery schools, to be over, attend, meeting, event, break, start, compulsory, hymn, uniform.
- **Lesson 3** (p.44–47) : _English secondary schools (3)_. Objectifs : remplir son emploi du temps hebdomadaire ; compléter la forme paraphrasée d'un texte. Pronunciation : syllabe accentuée des mots en _-tion_ et en _-al_. Vocab : choose, optional, extra-curricular activities, organise, drama, perform a play, orchestra, hockey, competition, high jump, long jump, have a reputation, fast food, snacks, packed lunch, surf the net.
- **Lesson 4** (p.48–51) : _Do you like school?_ Objectifs : lire/comprendre des opinions sur l'école ; travailler en groupe pour partager des points de vue ; écrire un e-mail exprimant son attitude vis-à-vis de l'école. Skills : scan a text for details ; group-work strategies. Functions : exprimer goûts et dégoûts (like, love, enjoy, hate, favourite, prefer) ; exprimer des vues personnelles (_find something + adjective_). Vocab : fun, feel, cool, tired, tiring, boring, plenty of, post, alright, whole, all kinds of.
- **Lesson 5** (p.52–58) : _To be pushy or not to be pushy, that's the question_. Objectifs : inférer le sens de mots nouveaux par le contexte ; écouter et identifier les attitudes des locuteurs ; écrire un paragraphe sur le rôle des parents dans l'éducation. Grammar : **should** (obligation morale / faire ce qui est juste). Vocab : work hard, pushy, show interest in something, learning problems, satisfy, care about, decide, decision, do well.
- **Mock Test + Checklist** (self-evaluation + group-work self-check).

**Module 3 — Entertainment** (SB p.60–89)

- **Lesson 1** (p.61–64) : _Preparing for the party_ — Review and introductory lesson (thème « Entertainment »). Objectifs : écouter et identifier les locuteurs ; écouter sélectivement pour les détails ; écrire une carte d'invitation ; écrire une liste de courses ; écrire puis jouer une conversation téléphonique. Functions : demander l'opinion de qqn ; exprimer son opinion ; arranger un rendez-vous (_Can we meet…? / Can I see you…? / Can you come?_) ; inviter (_would you like to + verb_) ; demander le sens d'un mot/expression. Vocab : special, give a party, evening, guest, join, forget, bring.
- **Lesson 2** (p.65–70) : _The party is on_. Objectifs : lire et prédire la suite ; écouter une chanson et l'apprécier. Skills : prédiction ; réponse non-verbale (mime) ; écrire la fin d'une histoire. Functions : exprimer le doute (_perhaps / maybe_) ; donner des instructions (révision) ; prépositions (révision). Vocab : refreshments, set a table, sit on the sofa, chat, against the wall, fill, empty, turn down the music, loud, shy, concentrate, take up classes, choreography, put on a show.
- **Lesson 3** (p.71–76) : _Having dinner with the Smiths_. Objectifs : lire et prédire la suite ; montrer la compréhension par réponse non-verbale (mime) ; écrire la fin d'une histoire. Vocab : plates, knives, forks, spoons, napkins, dressing, table cloth, clear the table, lay the table, a bit, to rest, to start out, bear, frightened, nervous, fall, climb on top of, breathe, sniff, dead, go away, munch, stretch out, dip, upside down, choice.
- **Lesson 4** (p.77–81) : _Sporting activities_. Objectifs : écouter pour le sens global et les détails ; donner une présentation sur les bienfaits des activités sportives. Grammar & functions : _verb + ing_ = sujet/objet ; _comparative + and + comparative_ (changement graduel) ; exprimer accord et désaccord. Vocab : sporting activities, regularly, enough, drive s.o. home/to school, fizzy drink, a mile, practise sport, enjoyment, a sense of, sound mind/body, to matter, forget.
- **Lesson 5** (p.82–89) : _What do you do in your spare time?_ Objectifs : lire et présenter l'information sous une autre forme (table) ; relier l'information aux visuels ; écrire un paragraphe sur ses activités de loisir. Grammar : **past progressive** ; _spend time + v-ing_ ; _spend time + with someone_. Vocab : get on someone's nerves, good fun, good laugh, lots of, nearby, bench, lawn, tall, hang around, lick, bask, enough, freezing cold.
- **Mock Test + Self-evaluation**.

**Module 4 — Séjour au Royaume-Uni / Édimbourg (vacances de Pâques)** (SB p.91–114). Project du module : produire une brochure sur une station touristique tunisienne.

- **Lesson 1** (p.92–95) : _Planning Easter holidays_ — Review and introductory lesson. Objectifs (TG) : écouter pour le sens global / les détails ; identifier des lieux sur une carte ; reconnaître les énoncés d'accord. Grammar : questions avec _how_ (how long, how far, how often, how much, how many). Pronunciation : mots liés au Royaume-Uni. Vocab : cash a cheque, far, close, it's OK with me, that'll be just fine, country, capital city, south, north, east, west.
- **Lesson 2** (p.96–100) : _Transport_. Skills : skim a text ; survoler / scanner ; utiliser un tableau pour résoudre un problème. Grammar : compound nouns (noun + noun) ; prononciation du marqueur du pluriel « s ». Vocab : take ages, traffic jam, rush hour, underground, queue, get annoyed, push in, journey, get around, pollution, freedom, benefits, means of transport.
- **Lesson 3** (p.101–104) : _Getting ready for the trip_. Objectifs : écouter une conversation et prendre des notes ; prendre part à une conversation téléphonique (réceptionniste d'hôtel ↔ client réservant un hébergement). Grammar : pronoms/adverbes relatifs (who, which / where, when) ; compound adjectives (numeral + hyphen + singular noun). Pronunciation : syllabes accentuées. Vocab : luxurious, popular, moderate prices, hostels, fairly, inns, bed and breakfast, receptionist, parking, seat, check in/out, single/double room, to book. Project : collecter des photos d'hébergements + décrire chaque hôtel et ses prix.
- **Lesson 4** (p.105–108) : _Accommodation_. Objectifs : stratégie pour compléter un texte lacunaire (deviner puis vérifier à l'écoute) ; identifier le décor (setting) ; utiliser un tableau pour résoudre un problème ; écrire un rapport. Grammar : prépositions de lieu (in, on, at) ; prépositions de temps (in, on, at). Vocab : currency, penny, pence, pound, coin, note, ticket, office, fare, jump the queue, bank employee, loan, to lend, to borrow, spend, first class, second class, single/return ticket. Project : écrire sur les services bancaires de la station.
- **Lesson 5** (p.109–114) : _Shopping in Edinburgh_. Objectifs : écouter et compléter la conversation ; effectuer une transaction dans un magasin de chaussures. Functions : demander prix, couleur, taille, matière, description. Grammar : compound nouns (v-ing + noun). Vocab : size, wrap, woollen, pleated, genuine, Scottish tartan, to feel nice, try on, fitting room. Project : décrire un centre commercial de la station.
- **Mock Test + Checklist** (p.114).

**Module 5 — Relationships** (SB p.116–141). Project du module : compiler des photos de pollution et rédiger des légendes (captions).

- **Lesson 1** (p.117–119) : _Relationships_ — Review and introductory lesson. Objectifs : lire pour le sens global ; répondre à un questionnaire ; donner un exposé de 3 minutes sur l'importance de bonnes relations. Vocab : get on well, fight, keep someone company, lonely, turn to, to comfort, guilty, cruel, share, strike up new relationships, nasty, relaxed, last, rely on.
- **Lesson 2** (p.120–124) : _Friends_. Objectifs : écouter pour les détails ; écrire une description de soi et de son ami(e) ; en groupe, identifier la meilleure qualité d'un ami. Grammar & functions : exprimer l'addition et l'opposition. Vocab : handsome, pretty, good-looking, curly hair, round face, naughty, lazy, jealous, selfish, understanding, confident, moody, easy-going.
- **Lesson 3** (p.125–128) : _Family relationships_. Objectifs : lire pour les détails ; identifier les mots exprimant des sentiments ; donner un conseil ; écrire une lettre informelle. Grammar : reflexive pronouns ; function : giving advice. Vocab : punish, lonely, proud, scared, happy, besides, upset, fault.
- **Lesson 4** (p.129–133) : _Save our planet_. Objectifs : lire et transférer l'information de sources diverses dans un tableau ; en groupe, choisir une action pour protéger l'environnement ; écrire une affiche (poster). Grammar : possessive pronouns ; questions avec _whose_. Vocab : save, planet, protect, plant, make an effort, leave the lights on, leave the water running, cut down, litter, pollute, destroy.
- **Lesson 5** (p.134–138) : _Pets_. Objectifs : deviner le sujet d'une histoire à partir du titre et des visuels ; identifier personnages et relations ; représenter l'intrigue dans un diagramme ; identifier le climax et le dénouement ; réagir à l'information ; écrire sur les animaux. Grammar : **could** (aptitude au passé / requête polie / suggestion). Vocab : go for a walk, take a walk, pet, afraid, die, be in trouble, the woods, carry, hurry, smart.
- **Mock Test + Checklist** (p.141).

### Compétences & progression (énoncées par le guide)

- **Learning principles** (Teacher's Book, Part 1) : l'apprentissage est un processus de découverte, multi-dimensionnel, responsabilité de l'apprenant, orienté produit-et-processus ; l'erreur fait partie de l'apprentissage ; motivation en climat sans tension.
- **Rôles** : l'apprenant est au cœur du processus ; le professeur met en scène l'apprentissage, entraîne l'autonomie et un répertoire de stratégies.
- **Évaluation** : self-evaluation (checklist « I can… » + mock test à la fin de chaque module) ; formative evaluation par le professeur.
- **Progression grammaticale annuelle** (consolidée dans le « Grammar Focus », SB p.142–152) : Yes/No & Wh-questions → questions avec _how_ (far/long/often/much/many) → _Would you like + nom / to + verbe_ (inviter) → demander description/prix/taille/matière/couleur → **could** → **simple future (will)** → **past progressive** → **be going to** → _will be able to_ (aptitude future) → exprimer la surprise → **should** → demander/exprimer une opinion → inviter/accepter/refuser → décrire (look/seem) → _V-ing_ sujet/objet → donner des instructions (impératif) → incertitude (maybe/perhaps) → _spend + time + …_ → comparatif + comparatif (changement graduel) → adverbes séquentiels → prépositions in/on/at (temps & lieu) → adjectifs composés (numéral-trait d'union-nom sing.) → noms composés (nom+nom ; v-ing+nom) → pronoms & adverbes relatifs (who/which/where/when) → pronoms sujet/objet/réfléchis + adjectifs/pronoms possessifs.

### Notions / contenus clés

- **Fonctions communicatives** : présenter qqn ; inviter / accepter / refuser ; demander & exprimer une opinion ; exprimer goûts/dégoûts ; exprimer la surprise, le doute, l'accord/désaccord, l'obligation morale (should), le conseil ; décrire personnes & lieux ; dire l'heure, l'âge, les dates ; effectuer une transaction / réserver un hébergement ; demander prix/taille/couleur/matière.
- **Temps & structures** : présent simple/progressif, passé simple (verbes irréguliers listés SB p.158–159), **past progressive**, **futur (will / going to / will be able to)**, could, should, verbe-ing, comparatifs graduels, relatifs, possessifs, réfléchis, noms & adjectifs composés, prépositions in/on/at.
- **Lexique thématique** : famille & présentations, courrier/téléphone/poste, tourisme & monuments de Londres, vie scolaire (secondaire britannique), fêtes/repas/loisirs/sport, transports, hébergement/banque/achats (Édimbourg), relations (amis/famille), environnement, animaux domestiques.
- **Phonétique / prononciation** : syllabe accentuée ; suffixes -tion / -al ; marqueur du pluriel « s » ; mots liés au Royaume-Uni ; table des symboles phonétiques (voyelles, diphtongues, consonnes) SB p.160.

### Types d'exercices / activités (relevés dans la source)

- **Reading** : for gist, for details, scanning, skimming, matching text↔visuel, prédiction, compléter un texte lacunaire (gapped text), compléter une paraphrase, transférer l'information vers un tableau, questionnaire.
- **Listening** : identifier locuteurs / topic / attitudes / humeur, écouter pour détails, prendre des notes, chanson.
- **Writing** : présenter des personnes (photo), lettre informelle (process writing), e-mail, rapport (report), carte d'invitation, liste de courses, notes de journal (diary), texte argumentatif court, affiche (poster), fin d'histoire, description de soi/d'un ami, paragraphe (bienfaits du sport, rôle des parents), légendes (captions).
- **Speaking / interaction** : conversation téléphonique (jeu de rôle), transaction commerciale, exposé de 3 minutes, travail en paires et en groupes, mime (réponse non-verbale), dessin.
- **Stratégies** : « learning how to learn », collecte & traitement de l'information, remplir un texte lacunaire, inférer le sens par le contexte, identifier les traits d'un genre (lettre informelle, narratif).
- **Project work** : une production par module (brochure, poster/photos, fichier sur son école, etc.) — évaluée régulièrement ; **Mock Test** + **Checklist « I can… »** en fin de module.

### Incertitudes de transcription

- **Teacher's Book (541805) TRONQUÉ** à 70000/70738 chars : coupe à l'intérieur du Module 5 Lesson 5 (« Pets », Session 1). Les toutes dernières procédures + éventuels appendices ne sont pas capturés (≈ 700 chars manquants — quasi complet). Tous les **objectifs par leçon** (Modules 1→5) sont en revanche présents.
- **Activity Book (141807) TRONQUÉ** à 70000/119632 chars (≈ moitié). C'est un cahier d'exercices qui reflète leçon par leçon le Student's Book ; la couverture structurelle reste complète via SB + Teacher's Book, mais les activités des derniers modules ne sont pas toutes lisibles ici.
- **Glossaire « List of words » (SB p.153–157)** : bilingue anglais↔arabe ; la colonne arabe est illisible dans la couche texte (mojibake / caractères brouillés). Seules les têtes de mot anglaises sont exploitables. Idem, les pages « ICONS USED IN THIS BOOK » (SB p.4, AB p.4) sont des images sans texte capturé.
- **Tables « Module Map » aplaties** : le PDF est multi-colonnes ; l'extraction entrelace parfois les cellules (skills / grammar / vocabulary d'une leçon peuvent déborder sur la ligne voisine). Les **titres de leçons et les plages de pages** sont fiables ; une attribution fine skill↔leçon peut comporter de légers décalages.
- **Bornes de page des modules** : les pages de début de module (p.34 M2, p.60 M3, p.91/116 M4/M5) sont déduites des marqueurs « Page N » ; les plages de pages des leçons sont explicites dans les maps.
- **Légère incohérence dans le Teacher's Book** : les objectifs listés pour le Module 4 Lesson 1 (« Planning Easter holidays ») mentionnent « write an invitation card », ce qui semble un copier-coller du module 3 ; transcrit tel quel — le Module Map (SB) est la source structurelle la plus fiable pour cette leçon.
