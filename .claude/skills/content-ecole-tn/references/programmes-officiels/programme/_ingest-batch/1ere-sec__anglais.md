### Cadre pédagogique (préface + en-tête du Book Map)

- **Deux compétences** structurent tout le manuel (mentions explicites de l'en-tête du Book Map) :
  - **Competency 1 : constructing meaning** (construire du sens).
  - **Competency 2 : communicating** (communiquer).
- **Quatre skills** développés « in a naturally integrated way » : **reading, listening, speaking, writing**.
- Convention du Book Map : « Words and grammar structures **in pink are consolidation items** » (items de consolidation / réactivation d'acquis antérieurs).
- Postulats affichés (préface) : rôle central de l'apprenant et **autonomie** ; la classe comme contexte valide de communication ; l'apprentissage comme « give and take » (hypothèses, négociation, vérification) ; langue = moyen de communication **et** objet d'étude (study skills & strategies) ; **authenticité** de la langue et des activités + importance du contexte.
- **Structure interne récurrente d'une leçon** (confirmée sur les leçons 1–13 lues in extenso) :
  - Ouverture par **« Getting Started »** (met en route la leçon et les compétences visées).
  - Puis **« Getting the Message » / « Getting the Gist »** (compréhension).
  - Puis phases **« Sharing » / « Communicating » / « Practise your Language Skills » / « Check your Vocabulary Skills » / « Being Creative »** (réinvestissement, production).
  - Clôture systématique par **« CHECK WHAT YOU HAVE LEARNT TODAY »** (grille : Ideas / Functions / Grammar / Vocabulary / Other).
  - Rubriques transverses ponctuelles : **« IT'S YOUR TURN TO ASK THE QUESTIONS »** (l'élève rédige le test oral) et **« PLANNING AHEAD »** (préparation d'une leçon future, ex. recherche sur Ibn Khaldoun avant la leçon 10).
- « Some of the lessons may require more than one teaching session » — la durée par leçon est laissée au professeur.

### Compétences & progression — « Competency Components » (stratégies visées)

Composantes des deux compétences, telles qu'énumérées leçon par leçon dans le Book Map :

- **Construire du sens (Competency 1)** : scan for specific information ; distinguish / identify the main idea ; answer reference & inference questions ; infer implicit / missing information ; predict content from core vocabulary ; guess the meaning of unfamiliar words from context / through affixation ; recognize the relationship between form and meaning ; recognize reduced / short forms of words ; differentiate relevant / irrelevant (important / unimportant) information ; categorize words and ideas ; evaluate information or ideas ; transfer information to a table / graph ; identify the communicative function of a text ; recognize relations between parts of a text ; match dictionary choices / definitions with context.
- **Communiquer (Competency 2)** : express opinions / tentative statements of judgement ; enquire about others' opinions ; express agreement / disagreement ; ask for & report information ; produce descriptions, reports, summaries, letters from notes ; reinvest new words & structures in one's speech / writing ; work in pairs / groups ; self-evaluation, self-check & peer correction ; reinvest real-world knowledge & experience.

### Types d'exercices / activités (relevés dans le corps, leçons 1–13)

- **Compréhension écrite/orale** : lecture pour le sens (gist) ; vrai/faux à justifier par des détails du texte ; questions de repérage (nom, lieu, occupation…) ; remplir un tableau à partir du texte ; remettre des phrases dans l'ordre ; répondre à des questions d'inférence.
- **Texte à trous / manipulation** : compléter un récit avec les bons temps/verbes irréguliers (ex. L2, L3) ; choisir le mot entendu parmi 3 alternatives (L5) ; compléter avec pronoms/prépositions (L5) ; compléter des mots amputés (bulles, « each dot stands for a letter » — L3, L4).
- **Lexique** : mots croisés (crossword — L2) ; appariement mot ↔ définition (colonnes A/B — L7, L8) ; appariement énoncés ↔ fonctions (L9) ; catégorisation.
- **Interaction orale** : pair work (interview d'un camarade) ; group work (deviner une personne célèbre) ; sharing / débat d'opinion ; questionnaires (auto-évaluation L6, argent de poche L7, style d'amitié L11, types d'intelligence L8).
- **Production écrite** : rédiger un paragraphe similaire sur soi ; écrire une description (personne « douée ») avec brouillon + correction par les pairs ; poème acrostiche (STUDENT → PUPILS) ; « Report Card » d'un récit ; rédiger le « Day 7 » d'un journal.
- **Projet / recherche** : project work (collecter des informations sur une personnalité, via Internet/dictionnaires) ; « Planning Ahead » (recherche préparatoire, ex. Ibn Khaldoun).
- **Métacognition** : grille « Check what you have learnt today » (Ideas/Functions/Grammar/Vocabulary/Other) en fin de leçon ; l'élève rédige lui-même le test oral (« It's your turn to ask the questions »).
