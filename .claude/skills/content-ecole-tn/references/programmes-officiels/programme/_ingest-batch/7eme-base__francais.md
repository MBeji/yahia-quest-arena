### Compétences & progression (telles qu'énoncées dans les sources)

- Objectif transversal du manuel (préface) : « communiquer » — écouter les autres et prendre la parole, réagir et traduire ses pensées, lire des textes variés, écrire et mettre en mots un message, évaluer ses acquis.
- Compétences affichées par module (« J'apprends… ») : Module 1 → lire une image, étudier des situations d'écrit, évaluer ses apprentissages ; Oral M1 → prendre la parole en classe, mieux communiquer avec ses camarades, se présenter, présenter quelqu'un.
- Progression conjugaison (déductible du parascolaire) : présent de l'indicatif → passé composé → verbes pronominaux → futur simple → impératif.
- Démarche d'autoévaluation régulière (grilles oui/non, fiche de route annuelle avec autoévaluations datées) présente dès le MCCP et à chaque activité.

### Notions / contenus clés (relevés dans les sources)

- **Conjugaison** : présent de l'indicatif (terminaisons par groupe) ; passé composé (auxiliaires être/avoir, participe passé, accord avec être) ; verbes pronominaux (présent, passé composé, impératif, forme négative) ; futur simple (rai/ras/ra/rons/rez/ront) ; impératif (3 personnes, sans sujet exprimé) ; valeurs du présent (action en cours, présent de narration, vérité générale) et du passé composé (action achevée).
- **Syntaxe** : quatre types de phrases + formes affirmative/négative ; phrase verbale/non verbale ; phrase minimale (GNS + GV) / phrase étendue ; compléments circonstanciels (temps, lieu, manière) ; expansions du GN (adjectif épithète, complément du nom, proposition relative).
- **Orthographe / homophones** : ce/se · ces/ses · c'est/s'est ; à/as/a ; et/est/es ; on/ont.
- **Vocabulaire** : sens d'un mot par sa formation ou le contexte ; champ lexical de la famille ; verbes d'action ; verbes de perception ; mots de reprise (reprises nominales et pronominales) ; connaître/savoir.
- **Lecture / types de textes** : conte, poème, B.D., récit, dialogue ; récit à la 1re personne ; les trois parties du récit ; le narrateur.
- **Oral** : se présenter / présenter quelqu'un ; comprendre une consigne ; participer à une discussion ; dire l'heure ; écoute d'une chanson.

### Types d'exercices / activités (relevés dans la source)

- Lecture silencieuse, lecture à haute voix (pauses, ponctuation, intonation, e muet), lecture-compréhension avec questionnement.
- Grilles d'autoévaluation (oui/non ; bien / plutôt bien / à améliorer / à apprendre) ; fiche personnelle de progrès.
- Jeux de rôles et prises de parole (porte-parole, dialogues de présentation).
- Exercices de conjugaison (« conjugue les verbes entre parenthèses au temps qu'il faut », transposition de temps/personne).
- Exercices de grammaire/syntaxe (souligner GNS/GV, réduire/enrichir une phrase, transformer type/forme, relier par flèches, classer dans un tableau).
- Exercices d'orthographe (compléter par l'homophone qui convient).
- Exercices de vocabulaire (relier mot ↔ définition, relever synonymes, enrichir une liste).
- Production écrite guidée (situations d'écrit, consignes, plan, rédaction, relecture/correction) ; remise en ordre de récit ; imaginer une suite / une fin.
- (Parascolaire `521702`) exercices à trous, numérotation dans l'ordre, « barre ce qui est faux », QCM de titre, le tout **avec corrigés**.
