### Compétences & progression

- **Quatre skills** intégrées — Listening, Speaking, Reading, Writing — avec la règle explicite du Teacher's Book : « **at least two skills are integrated in every teaching/learning session** ». S'y ajoutent systématiquement **pronunciation** et **spelling** (dictée + variantes), travaillées via le magnétophone (« imperative teaching aid »).
- **Subskills catalogués** (agrégés des grilles Teacher's Book) : discriminate/among sounds (minimal pairs) ; decode sound-symbol correspondence ; distinguish word boundaries ; recognize word-order & syntactic patterns ; understand explicitly stated information ; identify relationships/number/gender of interlocutors ; scan for specific information ; skim for gist ; infer meaning of words from context ; identify communicative functions of utterances ; identify reference relations ; transfer information onto a table / a house plan ; read a non-linear text ; write from dictation ; write legibly and accurately ; use basic punctuation (capitals, periods, exclamation) ; construct a coherent paragraph from notes ; sustain an exchange on a familiar topic ; produce oral descriptions from prompts ; fill in forms ; reinvest acquired language in speech & writing ; **demonstrate understanding through non-verbal response (TPR)** ; retain/repeat chunks of language.
- **Progression thématique (« Chinese boxes »)** : M1 soi/famille → M2 l'ami & l'accueil du correspondant → M3 la maison & l'environnement immédiat (ferme, marché) → M4 santé/sécurité/nature → M5 école & civisme (aider les autres, garder l'école propre). Du _self_ vers le _outside world / citizen_.
- **Progression grammaticale (fil directeur transcrit)** : _to be_ + pronoms sujets + démonstratifs → _have/has got_ + short answers + how many/how old + numbers → présent simple (3 formes) + adverbes de fréquence + prépositions de temps + linkers → like/love/enjoy → adjectifs & pronoms possessifs + génitif _'s_ + pronoms compléments → **present progressive** → modal _can_ (requête/permission/capacité) + _would like_ + _what about_ + _let's_ → **must/mustn't + do not + impératif + be careful** → **simple past** (_be_, réguliers puis irréguliers) → _much/many_ & _how much/how many_ (countable/uncountable) → articles a/an/the → _there is/there are_ + prépositions de lieu → pluriels réguliers/irréguliers → dérivation (**Noun+y=adjectif**, **adjective+ly=adverbe**, **verb+er=nom**) → comparaison (_Noun + be + like + noun_). Progression présent → passé, du mot vers le paragraphe.
- **Évaluation & autonomie** : chaque module se clôt par `Let's review` (révision/tests-types), `Let's remember` (récapitulatif grammatical/fonctionnel), `Self-evaluation` (checklist « Now I can… » à cocher — instaure une culture de l'auto-évaluation). **Project Work** collaboratif en 4 étapes (Modules 1, 3, 4) « meant to foster socio-psychological skills, research strategies and learner autonomy » (Official Programmes, p. 5).

### Types d'exercices / activités (relevés dans la source)

- **Compréhension écrite/orale** : read/listen + answer ; true/false (justifié) ; tick the right box/alternative ; complétion avec mots du texte ; transfert d'infos vers un tableau ou un plan de maison ; matching (énoncés↔fonctions, images↔mots, couleurs↔objets) ; questions de référence (« what does 'it/them/here/there' refer to? »).
- **Grammaire/lexique en contexte** : gap-filling ; circle the correct alternative/spelling/word with a different sound ; put the verbs in the right tense/form ; reorder words → sentences ; complete tables (formes du verbe, colonnes son).
- **Oral** : enact the conversation / role play / pair work ; ask & answer (interview de camarades) ; « tell me about… » ; simulations.
- **Écrit** : de la phrase au paragraphe ; genres = letter/postcard, paragraph (famille, routines, anniversaire, enseignant préféré), report, house description, list of tips/leaflet, dialogue completion.
- **Prononciation** : listen & repeat ; same/different ; odd man out ; put the words in the right column ; number/circle the stressed syllable ; match words with phonetic transcriptions ; dictée (« Listen and write / spell out »).
- **TPR & ludique** : listen and mime / point ; time for a song ; **time for a game** (word snake, word snail, crossword, word search, dice, secret word, puzzles, spidergram) ; time for a joke ; story time (lecture-plaisir) ; let's draw / read & colour.
- **Projet & évaluation** : Project work en 4 étapes (Modules 1/3/4) ; `Let's review` ; `Self-evaluation` (checklist « Now I can… »).

### Bornes de scope (INCLUS / EXCLU)

- **INCLUS** : anglais, 7ème année de base (7ᵗʰ Year Basic Education), manuel national CNP _Let's Learn English_ opérationnalisant le programme officiel « English Programme for Basic Education » (mars 2005). Couverture complète : les **5 Modules × 4 Sections (= 20 sections)** + les 5 Reviews (`Let's review` / `Let's remember` / `Self-evaluation`), avec pour chaque section : intitulé exact, page (Table of Contents SB), skills, functions, grammar, lexis, pronunciation, project step le cas échéant. Sont aussi transcrits : le cadre pédagogique (Teacher's Book Part I), l'inventaire phonétique, la table des verbes irréguliers, les glossaires officiels par module, et le miroir Activity Book.
- **EXCLU** : aucune autre matière/niveau. Ne sont **pas** reproduits in extenso (mais présents dans la source et disponibles pour l'ingestion de contenu) : le corps détaillé de chaque activité (énoncés complets, images), les **answer keys** du Teacher's Book, et les **tapescripts** (Part III) — la fiche en donne la nature/structure, pas le verbatim exercice par exercice. Les supports audio (Course tape) sont référencés mais non fournis comme fichiers.

### Incertitudes

- **unitCount** : compté à la granularité **Section** = 20 (5 modules × 4). Selon la maille retenue on pourrait dire « 5 modules » ou « 20 sections (+ 5 reviews) » ; les deux nombres sont explicités ci-dessus.
- **Symboles phonétiques** : l'extraction `pdf-text` a rendu les glyphes IPA de façon dégradée (ex. `∧`→/ʌ/, `⊃`→/ɔ/, `∂`→/ə/, `∫`→/ʃ/, `t∫`→/tʃ/, `dz`/`a`→/dʒ//ð/, `∂U`/`,u`→/əʊ/). Ils sont **normalisés vers l'IPA standard** ci-dessus ; quelques cases des grilles « put in the right column » ont été éclatées en lettres/symboles isolés — les phonèmes cités sont ceux lisibles sans ambiguïté. Deux sections présentent une **légère divergence de grille phonétique entre le Student's Book et le Teacher's Book** (ex. M3 S1 : /ɔi a: əʊ dʒ z/ côté SB vs /i aʊ a: dʒ z/ côté TB) — les deux variantes sont notées.
- **Coquilles de la source (glossaires/manuel)** : l'extraction contient des fautes de frappe d'origine — _penfreinds → penfriends, tronsers → trousers, environnement → environment, desser → dessert, pttery → pottery, evrybody → everybody, litle/scool/techer/dauter_ (mots-pièges des exercices d'orthographe), _foll ow/flow_ (au lieu de « follow »). Le **Vocabulaire officiel** ci-dessus rétablit l'orthographe anglaise standard (mots non ambigus) ; la structure et les intitulés de sections, eux, sont transcrits tels quels.
- **Pages** : lues sur la Table of Contents du Student's Book (numéros de page du manuel élève). Le Module 5 ne comporte, dans la TOC, ni `Let's review` ni `Self-evaluation` séparés — seulement `Let's remember` (p. 165), suivi de la table des verbes irréguliers (p. 168). Fiables ; à revérifier au besoin sur le PDF paginé.
- **Nom du personnage / graphie** : la source alterne « Aly » (constant) et de rares variantes typographiques (Soltane) ; retenu tel qu'écrit majoritairement.
- **Grammaire « projets »** : seuls les Modules 1, 3 et 4 portent une colonne/rubrique Project explicite ; l'absence de projet formel en Modules 2 et 5 est déduite de l'absence de la colonne « Project » dans leurs grilles SB et de l'absence d'étape « Project work » dans les procédures TB correspondantes.
