### Structure (fidèle, exhaustive)

**Architecture globale.** L'ouvrage est bâti sur **6 modules thématiques**, chacun composé de **5 leçons** (soit **30 leçons** au total). Le Teacher's Book décrit chaque module comme « an introductory lesson and four other lessons where the focus is on one macro-skill (Reading, Listening, Speaking or Writing) ». Student's Book, Activity Book et Teacher's Book partagent la **même** table de modules (colonnes : Page / Lesson / Skills / Functions / Grammar / Lexis / Stress & pronunciation / Project work-Writing), placée en tête de chaque module, complétée d'un bandeau **SubSkills** en bas.

**Grille interne d'une leçon.**

- _Student's Book_ — 3 parties : **GET READY** (mise en situation par images, brainstorming, « Step 1/2… », pair/group work) → **LEARN MORE ABOUT IT** (passage de lecture/écoute, poème ou conversation + questions de compréhension) → **BRUSH UP YOUR GRAMMAR** et/ou **YOUR NEW GRAMMAR POINT** (rubrique « Read and think », encadrés « REMEMBER! »). En fin de volume : section **GRAMMAR REVIEW** (récapitulatifs encadrés leçon par leçon, module 1 → module 6, p. 145-160).
- _Activity Book_ — par leçon : **COMPREHENSION CHECK** (Activity 1, 2, 3…) → **LANGUAGE WORK** (Activity 1, 2…) → **PRONUNCIATION** ou **PRONUNCIATION PRACTICE** → selon la leçon **SPELLING** et/ou **SPEAKING** → **WRITING** (One/Two/Three — 3 leçons d'écriture par module, cf. « the learners have been given the opportunity to write in three lessons per module ») → **PROJECT WORK** (Step 1/2/3 répartis sur le module) → **THE LIGHTER SIDE** (proverb, saying, riddle, joke of the day, brain teaser, « Did You Know? »). Par module : **GRANDMA'S CORNER** (un récit) + **SELF-EVALUATION** (notée /20) + **WORD LIST** (glossaire alphabétique).
- _Teacher's Book_ — table des matières : Introduction · The material designed to teach the 9th grade · Methodology and principles in the design of the materials · Breakdown of skills and strategies over the modules · Module maps, lesson descriptions and teaching hints · **Tapescript** (annoncé p. 9 mais **absent de l'extraction** — voir Incertitudes). Pour chaque leçon, le guide donne : Objective · Main skills · Subskills · Vocabulary · Procedure · Structure(s)/Function(s).

### Compétences & progression

**Macro-compétences (skills).** Reading, Listening, Speaking, Writing — une macro-compétence dominante par leçon. Chaque module comporte 3 leçons d'écriture (**Writing One/Two/Three**) et un **Project Work** en 3 étapes ; le Listening est introduit dans la 2ᵉ leçon de chaque module, le Speaking dans la 4ᵉ.

**Sous-compétences (SubSkills) — listées par module dans les tables officielles :**

- **M1** : infer relationships between interlocutors ; predicting possible outcomes ; producing a text from various sources of information ; information transfer ; scanning for specific information ; reinvesting previously acquired language ; read for the gist ; transfer information onto a table ; produce written descriptions from prompts ; listing and writing notes ; distinguishing between fact and opinion ; drawing on background knowledge to understand text ; listen to infer emotional state ; write to recycle structures covered in class ; free writing leading to fluency.
- **M2** : recognising basic syntactic patterns and devices ; reinvesting acquired language in one's writing ; skimming and scanning for information ; sharing information to perform an oral task and solve a problem ; reading for the gist ; scanning for specific information ; making lists ; identifying the communicative functions of utterances ; using cohesive devices ; reordering words to make sentences ; working in groups and sharing information to perform an oral task ; guided writing.
- **M3** : skim a text to get the gist ; predicting possible outcomes ; use free writing to develop fluency ; scanning for specific information ; reinvesting previously acquired language ; read for the gist ; scan a text for specific information ; work in pairs or small groups to make out the meaning of a text ; drawing on background knowledge to understand a text ; listen to infer emotional state ; identify/select main idea, supporting details, facts ; write to recycle structures covered in class.
- **M4** : distinguish facts from opinion ; predicting possible outcomes ; use free writing to develop fluency ; infer relationships between interlocutors ; work in pairs and share information to perform an oral task ; infer the emotional state of a speaker (e.g., from intonation, speech rate) ; infer the communicative functions of utterances (situations, participants, goals) ; process writing ; draw on background knowledge ; listen to infer the meaning of new words/expressions ; information transfer ; predict the gist and type of text from illustrations ; infer the possible meanings of a word/expression ; reinvest language covered in class in project work.
- **M5** : drawing on prior knowledge ; identifying activities from pictures ; reading for the gist ; making suggestions ; identifying stress at word level ; showing interest ; showing indifference ; matching words to make expressions ; matching parts of sentences ; interpreting pictures ; scanning a text for specific information ; information transfer ; completing an interview ; identifying different sounds ; identifying vocabulary related to food ; matching words with definitions ; listening for the gist ; reinvesting language in one's writing ; describing people ; reading for specific information ; predicting possible outcomes ; writing a biography.

**Fonctions langagières (functions) — progression sur les 6 modules :** asking questions · describing (1, 2) · comparing (1/2/3, equality & superlative) · contrasting · expressing possession · expressing intention (going to / future) · expressing a completed past action · obligation · sequencing events · expressing possibility & probability · defining people & objects · giving advice / obligation / prohibition · noun formation (2, 3) · making polite requests · requesting others to do something · warning · thanking · first conditional · expressing (un)certainty · exclamations & expressing satisfaction · making suggestions · expressing regret / apologising · showing interest & indifference · expressing ability/inability · offering help · describing past actions · expressing opinion · expressing hope.

**Principes méthodologiques (Teacher's Book, « Guiding principles ») :** (1) « The learner is an intelligent human being » ; (2) « The learner is at the core of the learning process » (« active user of English ») ; (3) « The learner's participation and interaction are important in learning » (travail coopératif/collaboratif, en groupes) ; (4) « Contextuality in learning » (la langue = moyen de communication, non un ensemble de structures décontextualisées). ### Types d'exercices/activités

**Compréhension :** true/false + correction ; choix multiple (tick / circle / underline the right alternative) ; questions ouvertes (wh-questions) ; remise en ordre de phrases (a, b, c…) ; matching (mots↔définitions, parties de phrases, colonnes A/B/C, pictures↔labels) ; find equivalents / synonyms / antonyms in the text ; information transfer (remplir tableaux, spidergrams, charts) ; reference questions (« what does X refer to? »).
**Langue :** fill in the blanks / gap-filling (avec boîte de mots) ; mettre le verbe au temps/à la forme voulus ; circle/choose the right form ; transformations (intention↔planned action, my → mine, active → "able to") ; make sentences from prompts/hints ; compléter la règle ; mots mélangés → phrase cohérente.
**Prononciation / orthographe :** classer des mots par son (headings) ; tick the right sound ; identifier/accentuer la syllabe tonique ; ajouter des mots à un groupe de sons ; repérer les lettres muettes ; transcription phonétique ; choisir le bon pluriel.
**Production :** guided / free writing (paragraphes, lettre, note d'excuse, article de magazine, biographie, blog) ; speaking / pair-work / group-work ; enact / act out a conversation ; **Project Work** en 3 étapes par module (poster de vêtements de générations, leaflet, liste de problèmes scolaires + solutions, blog internet, wall display sur un artiste, action civique).
**Détente :** « The Lighter Side » (proverbs, sayings, riddles, jokes of the day, brain teasers, « Did You Know? »).
**Évaluation :** **SELF-EVALUATION** en fin de chaque module, notée **/20**, avec la consigne « If your (total) score is less than 10/20, ask your teacher for help ».

### Incertitudes

- **Tapescript manquant.** La table des matières du Teacher's Book annonce une partie « Tapescript » (matériel audio, p. 9), mais l'extraction de `541905P00.md` s'arrête à la fin des descriptions de leçons (Module 6 Lesson 5, p. 44).
