### Types d'exercices/activites

- **Étude de texte / lecture expliquée** : découverte (titre, illustrations, hypothèses de lecture) → compréhension globale → analyse → appréciation → synthèse écrite ; questions ouvertes de compréhension ; QCM de compréhension globale (« j'écris les bonnes réponses », module de lecture 1).
- **Oral** : situations de communication (se présenter / présenter, demander & fournir des informations, conseiller, discuter, exprimer & échanger des points de vue) ; jeux de rôles à deux ou à trois ; devinette en cinq questions ; exposés préparés en groupes.
- **Langue (manipulation en contexte)** : transformation, substitution, complétion, réécriture ; identification / relevé ; conjugaison ; appariement (rattacher question ↔ réponse) ; complétion de phrases par connecteurs / temps ; chaque leçon débouche sur une petite production écrite (« J'écris »).
- **Vocabulaire** : remplacer une expression par son synonyme / contraire ; former des dérivés (noms, adjectifs, verbes) ; nuancer un champ lexical ; recours au dictionnaire ; emploi en phrases.
- **Écrit / production** : rédiger un court récit ; insérer des passages descriptifs ; faire un portrait ; insérer un dialogue (ponctuation, verbes introducteurs) ; produire un paragraphe explicatif / argumenté ; réécriture (imiter, modifier début/fin, changer de point de vue, imaginer une suite ou une autre fin).
- **Débat** : questions de discussion en fin de module de lecture (ex. « Les fantômes sont-ils réels ou imaginaires ? »).
- **Évaluation (document)** : devoir de contrôle (étude de texte + langue + expression écrite), devoir de synthèse ; évaluation diagnostique d'entrée (QCM, appariement, exercices à trous). Le manuel annonce (avant-propos p.3) des **grilles d'évaluation** en fin de chaque module.

### Bornes de scope (INCLUS/EXCLU)

- ✅ **INCLUS (8ème base, transcrit)** :
  - Temps : imparfait + passé simple (temps du récit) ; réactivation présent / passé composé / futur ; **introduction** du conditionnel présent et du subjonctif présent (sensibilisation, M5).
  - Grammaire : présentatifs ; substituts / mots de reprise ; pronoms personnels (sujet, COD) ; complément du nom ; genre & nombre du nom ; déterminants ; **expansions du GN par groupe adjectival ET proposition relative** ; verbes d'action / d'état + attribut du sujet ; détermination ; expression du lieu / du temps ; forme affirmative / négative ; phrase interrogative ; temps du récit / temps du discours ; verbes introducteurs ; cause / conséquence / but.
  - Orthographe : accord du participe passé (être + avoir/COD antéposé) ; genre & nombre du nom ; homophones _ou / où_ (+ _ont/on/est/et_ dans les devoirs) ; suffixes orthographiques.
  - Discours : raconter · raconter en décrivant (personne, animal, lieu, objet) · raconter en faisant parler (dialogue inséré **au discours direct**) · informer · expliquer / justifier / argumenter (sensibilisation) · portrait physique et moral.
  - Lecture : textes narratifs, explicatifs, poétiques, extraits de théâtre ; lecture de l'image (BD incluse) ; **2 œuvres / textes longs intégraux** (un conte merveilleux — « Le toucher d'or » ; un récit fantastique — « La Filleule de la Mort ») ; formes du récit (roman / conte / nouvelle).
- ⚠️ **Précisions de scope (document)** : « En 8ème année de base, **l'étude du passé simple vise surtout à faciliter la compréhension des textes** » (p.23) → privilégier les verbes fréquents, surtout aux 1ère et 3ème personnes ; on fait réécrire au passé composé / présent de narration → le passé simple n'est **pas** un temps de production systématique. Conditionnel & subjonctif = **emplois introduits en contexte**, pas conjugaison exhaustive.
- ⛔ **EXCLU (relève de la 9ème — borne haute)** : propositions subordonnées étudiées comme système (complétives / circonstancielles classées) ; voix active / voix passive comme transformation systématique ; discours indirect transposé + concordance des temps (en 8ème : insertion de dialogue au discours **direct**, opposition récit/discours, verbes introducteurs — pas la transposition) ; figures de style théorisées ; système modes/temps exhaustif.
- ⛔ **EXCLU (déjà acquis en 7ème — borne basse)** : présent / passé composé / futur / impératif comme première découverte (réactivés) ; expansion du GN par simple adjectif épithète seul (la 8ème **ajoute** le groupe adjectival complexe et la relative) ; types & formes de phrases élémentaires ; déterminants de base.
