### Structure (fidele, exhaustive)

**Architecture de la collection** (Introduction SB/AB p.3 ; Teacher's Book Preface p.3) — le programme s'implemente via **quatre supports** :

- **Student's Book** (SB) — les documents-supports + les regles + les consignes (l'eleve **n'ecrit pas** dessus).
- **Activity Book** (AB) — le jeu d'activites lecon par lecon (l'eleve y ecrit toutes les reponses) + un **Mock Test** par module + tous les **tapescripts** audio en fin d'ouvrage.
- **Teacher's Book** (TB) — Partie 1 : principes de conception ; Partie 2 : schema d'implementation lecon par lecon (objectifs + aims + procedure).
- **A cassette or CD** pour les documents d'ecoute (aural input).

**Ossature** : **5 modules × 5 lecons = 25 lecons.** Chaque module s'ouvre par un **Module Map** (topics / skills & strategies / grammar & functions / vocabulary [+ colonne « Projects » aux modules 4 et 5]), se termine par **« The lighter side » + « Jokes »** (grammaire/phonetique par l'humour), une **checklist « Check your learning » (« I can… »)** d'auto-evaluation (SB) et un **Mock Test** (AB). La **1re lecon de chaque module** = « Review and introductory lesson » (active les acquis + pose le decor de l'histoire du module).

**Structure interne d'une lecon** (3 parties, chacune en « Steps ») : **LET'S GET READY** (LGR — revision/mise en route, prise de conscience de l'objectif) → **LET'S DISCOVER** (LD — decouverte du nouveau contenu via lecture/ecoute) → **LET'S COMMUNICATE** (LC — reinvestissement en production orale/ecrite). (TB abreviations : AB, SB, LGR, LD, LC, Act=Activity, Lg. Study=Language Study, Sup. Act=Supplementary Activity, →→=go to, sd=student, bb=Black Board.)

**Fil narratif** (TB Preface p.7) : « The first 3 modules tell the story of a Tunisian teenager [Imene] who was invited to London by an English family [the Browns]. The fourth module relates the story of an English family who have decided to spend their Easter Holidays in Edinburgh, Scotland. "Relationships" was the theme we selected for the materials of the 5th module. This has included relationships with parents, friends, the environment and pets. »

**Pages liminaires** : couverture (p.1) ; copyright (p.2) ; Introduction « To the students… » (p.3) ; **ICONS USED IN THIS BOOK** (p.4 — pictogrammes, non extraits car images).

#### MODULE 1 — Meet the Browns / London (Module Map ~p.6 ; lecons p.7–30)

- **L1 — Meet the Browns** (Review and introductory lesson) · **pp.7–10**. Objectifs (TB) : lire et comprendre l'information explicite ; ecrire quelques lignes pour presenter des personnes sur une photo. Skills : Reading (understand explicitly stated information) ; Writing (introduce people in a picture). Grammar & functions : Yes/No questions ; Wh-questions ; Inviting (`Would you like…?`) ; Introducing (`This is…`). Supports : texte « Meet The Browns » ; tableaux Wh-/Yes-No. Activites : ecrire questions, matching, mime des activites, presentation orale.
- **L2 — A letter from an English friend** · **pp.11–13**. Objectifs (TB) : lire une lettre pour le gist ; reconnaitre expediteur/destinataire ; identifier les traits d'une lettre informelle. Grammar & functions : Review tenses ; decouvrir le **simple future** (`will`). Support : lettre informelle « Dear Imene… Your Friend, Chris » (avec adresse/date). Activites : gap-fill, layout de la lettre, tri des temps, mini-dialogues au futur.
- **L3 — Can she go to London ?** · **pp.14–18**. Objectifs (TB) : identifier les locuteurs ; identifier le topic d'une conversation ; reconnaitre les discourse markers (sequential adverbs) ; ecrire un court texte argumentatif. Grammar & functions : Expressing ability in the future (`will be able to`) ; **Sequential adverbs** (First, Second… finally) + cardinaux/ordinaux ; Talking about future events. Spelling : Dictation. Pronunciation : Stressed syllable. Support : conversation famille Najjar. Activites : listening/reading comprehension, dictee, ecriture argumentative.
- **L4 — Preparing for the trip** · **pp.19–22**. Objectifs (TB) : identifier les locuteurs au telephone ; travailler en groupe ; produire une conversation telephonique sur modele ; inferer l'humeur du locuteur ; strategie de complement d'un texte lacunaire. Grammar & functions : Review Telling the time ; **Expressing surprise** (`How + adjective` / `What + noun` / `What a surprise!`). Support : « A Telephone Call » (Imene/Chris). Activites : gap-fill guide, expression de la surprise, ordonner des actions, enact/ecrire un dialogue.
- **L5 — London wonders** · **pp.23–29**. Objectifs (TB) : apparier textes et visuels ; presenter l'information sous une autre forme ; ecrire des entrees de journal (diary) a partir de notes ; produire une lettre informelle (process writing) a partir de sources diverses. Grammar & functions : Describing places ; **Saying dates**. Supports : notes de journal d'Imene ; descriptions des sights (Trafalgar Sq., Tower Bridge, Westminster, Buckingham…). Activites : matching, dictee de dates, tableau, dessin, redaction de lettre. **Project Work** : tenir un journal personnel (diary) regulier.
- **The lighter side / Jokes** (p.30–31) · **Checklist « Check your learning » — Module 1** (p.32) · **Mock Test** (AB).

### Competences & progression

**Quatre competences integrees** (reading, writing, listening, speaking) + **prononciation** + **strategies d'apprentissage** (learning how to learn) + **projet** (project-based learning) + **valeurs/vie sociale** (cooperation, tolerance, respect des avis). L'apprentissage est pose comme **decouverte** (« Let's Discover More »), **responsabilite de l'apprenant**, **process-and-product-oriented** ; l'erreur est un **etape naturelle** de la decouverte (TB Partie 1, a–b).

**Progression des structures sur l'annee** (ordre des modules) : (1) revision des temps + **simple future (will)** ; (2) **be going to** + **should** + `How long?` ; (3) **past progressive** + `V-ing=subject/object` + comparatif graduel + `spend + time` ; (4) **questions with how** + prepositions in/on/at + relatifs (who/which/where/when) + compound nouns/adjectives ; (5) **reflexive/possessive pronouns** + `whose` + **could**. Fonctions communicatives etalees en parallele (inviter, exprimer surprise/opinion/accord/doute, demander prix/taille, decrire, conseiller, arranger un rdv…).

**Auto-evaluation** : chaque module se clot par une **checklist « I can… »** (SB) + un **Mock Test** (AB, corrige en classe). Les Mock Tests couvrent 4 rubriques : Reading Comprehension / Writing / Speaking (Role play) / Listening Comprehension. Evaluation **formative** : le maitre exploite les donnees de l'auto-evaluation pour la remediation (sessions dediees dans le TB). **Grille d'evaluation de projet** hebdomadaire (Content / Design / Presentation).

### Notions/contenus cles

**Grammaire (enseignee EN CONTEXTE, « Grammar Focus » pp.142–152)** : Yes/No & Wh-questions ; **Questions with How** (far/long/often/much/many) ; **simple future** `will/'ll (+won't)` ; **be going to** (intention) ; **will be able to** (capacite future) ; **past progressive** `was/were + v-ing` ; **could** (ability in the past / polite request / suggestion) ; **should/shouldn't** (obligation morale) ; **V-ing** comme sujet/objet ; **comparative + and + comparative** (changement graduel, adj courts `-er`+`-er` / longs `more and more`) ; **prepositions of time & place** (in/on/at) ; **relative pronouns & adverbs** (who/which/where/when) ; **compound nouns** (N+N ; V-ing+N) et **compound adjectives** (numeral+hyphen+singular noun) ; **subject/object/reflexive pronouns**, **possessive adjectives/pronouns**, **`whose`** ; **sequential adverbs** (First/Second/…/finally) + cardinaux/ordinaux ; telling the time ; saying dates ; describing places/people (`look/seem + adj`) ; giving instructions (imperatif +/−).

**Fonctions communicatives** : introducing/inviting/accepting-refusing ; expressing surprise (`How + adj` / `What + noun` / `What a surprise!`) ; asking/expressing opinion ; expressing agreement/disagreement ; expressing doubt (maybe/perhaps) ; expressing likes/dislikes ; expressing addition/opposition ; giving advice ; arranging to meet ; asking the meaning of a word ; asking for price/size/colour/material/description ; expressing addition/opposition ; `have to`/`must`.

**Prononciation** : syllabe accentuee ; mots en `_tion` / `_al` ; marqueur pluriel `-s` (`/s/ /z/ /iz/`) ; mots lies au UK ; symboles phonetiques (table p.160).

**Notation standard** : chiffres latins (dates, prix en £/pence, heures, ages, `1 mile = 1.609 km`), LTR, unites SI. Aucun chiffre arabo-indien attendu.
