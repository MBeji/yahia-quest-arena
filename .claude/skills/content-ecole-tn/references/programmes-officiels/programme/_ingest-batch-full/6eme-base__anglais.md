Le seul document du lot est le **guide de l'enseignant (Teacher's Guide)**, qui EST le programme officiel opérationnel : il porte, leçon par leçon, les _learning objectives_, le _vocabulary_, la _grammar / function_, les corrigés (« Answer Keys ») et les scripts audio (_tapescripts_). Aucun manuel élève (Student's Book) séparé n'est fourni dans le lot. La section « Answer Keys » déroule l'intégralité de la progression **Unit par Unit, Lesson par Lesson** — c'est elle qui donne la structure fidèle ci-dessous.

**Cadre pédagogique déclaré (Introduction, p. 3–8).** L'apprenant est un _agent_ (utilisateur actif de l'anglais dans les tâches communicatives proposées), non un récepteur passif d'input. Approches recommandées : constructivisme, socio-constructivisme, cognitivisme (source : « congnitivism », sic), action-oriented approach, Discovery Learning, Total Physical Response (TPR), approches humanistes. Chaque leçon ouvre sur des _learning objectives_, puis les _lexical items_ + _structure(s)_ et/ou _language function(s)_ à introduire/consolider, puis une suite d'étapes menant à une **tâche de production** (orale ou écrite) en fin de leçon. Le guide fournit les corrigés et les scripts audio. L'Introduction décrit une à une les rubriques d'une leçon (voir « Vocabulaire officiel » plus bas) et énonce (p. 8) : « The Student's Book comprises **seven** projects to be conducted at the end of each Unit ».

### Structure (fidele, exhaustive)

Découpage à deux niveaux : **7 Units** (I → VII), chacune **5 Lessons** + un **Progress Check** de fin d'unité + un **Project** de fin d'unité. **Total : 7 Units · 35 Lessons · 7 Progress Checks · 7 Projects.** L'ouvrage complet est capté ici (Units I → VII, sans troncature).

Chaque fiche-leçon énonce systématiquement : _Objectives_ (« Upon completion of the lesson, the learners will be able to… »), _Vocabulary_, _Grammar_ (parfois étiqueté _Function_), puis les corrigés des rubriques d'activités. Rubriques récurrentes d'une leçon : `Let's get started` · `Let's read` / `Let's Listen` · `Let's complete the rule` · `Let's practise` · `Let's speak` · `Let's write` · `Let's pronounce` · `Let's play` (+ `Think / Pair / Share`, `Let's check`).

#### Unit I — Entertaining Myself (p. 9–18)

- **Lesson 1 : Let's Have Fun** (p. 9–11) — Obj. : nommer des activités de loisir ; parler de ses loisirs préférés. Vocab. : fishing / museum / go camping / riding a bike / gardening. Grammaire : _because_ ; _like + verb + -ing_ ; _enjoy + verb + -ing_ (goûts : enjoy/like ; dégoûts : hate/dislike + -ing). Activités : label pictures, tableau tick/cross, dialogue en binôme (favourite hobby), _Let's write_ (paragraphe sur ses loisirs), _Let's pronounce_ (cross the odd word out), _Let's play_.
- **Lesson 2 : My Free Time Activities** (p. 11–13) — Obj. : nommer des loisirs ; écrire sur ses loisirs préférés. Vocab. : drawing / gardening / shopping. Grammaire : **adverbes de fréquence** (always / sometimes / usually / never — accord _+s_ à la 3ᵉ pers.) ; **prépositions** _on / in_ (on the weekend / on holidays / on Monday morning ; in the morning / afternoon / evening — annoté « Consolidation »). Activités : reading (tableau Name / Town / Favourite free time activities) + match adverbes, _Let's speak_ (bulles à compléter, linkers/ponctuation), _Let's pronounce_ (tri Same/Different).
- **Lesson 3 : A Day Out** (p. 13–14) — Obj. : nommer des activités de plein air ; faire et répondre à des suggestions. Vocab. : bored / hide and seek / tired / zoo / far. Grammaire : _Let's + infinitive_ / _How about + V-ing?_ / _What about + V-ing?_. Fonction : **making suggestions**. Activités : _Let's Listen_ (tapescript Adam/Linda) + comprehension + remise en ordre, role play, _Let's pronounce_ (/s/ vs /z/ : cinema/let's vs zoo/plays).
- **Lesson 4 : Let's go on a picnic** (p. 14–16) — Obj. : nommer des objets de pique-nique ; parler des activités de pique-nique. Vocab. : get ready / river / wash / arrive / rope skipping / fly a kite. Grammaire : linkers _and – but_ ; adjectifs possessifs _his – her – their_ (paradigme I–my … they–their). Activités : reading (true/false + match Dad/Mum/Annie), Think-Pair-Share, _Let's practise_ (his/her/their), _Let's write_ (5 phrases → paragraphe), _Let's pronounce_, _Let's play_.
- **Lesson 5 : Pass the Ball** (p. 16–17) — Obj. : nommer des loisirs ; faire offres et requêtes avec _Would you like…?_. Vocab. : lose / team / win / kick / run. Grammaire : _Would you like…?_ (inviting = Would you like + to + verb ? ; making a suggestion = Let's + verb). Activités : label pictures, reading (à la plage) + comprehension, role play (Speaker 1/2), _Let's pronounce_ (match des sons ; play/draw/park), _Let's play_ (associer sports : basketball/handball/football/tennis/volleyball).
- **Progress Check 1** (p. 17–18) — Reading Comprehension (tableau Name/Age/Town/Favourite activities ; true/false ; blanks) ; Language (pronoms she/her/he/their/they ; would you like/how about/let's) ; Writing (remise en ordre de phrases + brainstorming → paragraphe).

#### Unit II — Keeping Fit (p. 18–25)

- **Lesson 1 : I'm Healthy, I'm Happy** (p. 18–19) — Obj. : classer les aliments en sains/malsains ; donner des conseils pour rester en bonne santé. Vocab. : fruits / vegetables / lots / healthy / unhealthy. Grammaire : _should / shouldn't_ (giving advice). Activités : label + tick/cross healthy foods, questionnaire + dialogue de conseil, _Let's pronounce_, _Let's play_.
- **Lesson 2 : Eat well, Live well** (p. 19–20) — Obj. : décrire les éléments d'une alimentation saine ; dire aux autres de faire des choix sains avec _must / mustn't_. Vocab. : vegetables / teeth / egg / keep fit / bad for. Grammaire : _must / mustn't_. Activités : _Let's Listen_ (tapescript Anna/mum) + comprehension, _Let's practise_ (must/mustn't), _Let's write_ (health poster, groupe de 4), _Let's pronounce_ (/ɪ/ vs /iː/ : fit/hit vs cheese/meat), _Let's play_.
- **Lesson 3 : I'm Healthy, I'm Fit** (p. 20–22) — Obj. : décrire personnes en forme / hors forme ; distinguer habitudes saines et malsaines. Vocab. : doughnuts / fit / fat / tired / excited / sad / unhappy. Grammaire (en-tête) : _should/shouldn't_ ; _must/mustn't_ — la _rule_ de la leçon porte en fait sur **can / can't** (expressing ability / inability). Activités : spot-the-difference + comparaison (slim/fat/thin/fit), reading (true/false + match), _Let's speak_, _Let's pronounce_, _Let's play_ (healthy food).
- **Lesson 4 : How are you Feeling?** (p. 22–23) — Obj. : parler de trois maux (ailments) ; donner des conseils santé. Vocab. : toothache / stomachache / sick / tired / headache. Grammaire : _have a + ailment_ ; _to be + adjective_. Activités : listen/repeat/mime, _Let's read + Let's Listen_ simultanés (Simon/doctor) + comprehension, role play patient/docteur, _Let's pronounce_, _Let's play_ (grille de mots-mêlés).
- **Lesson 5 : How to Stay Fit** (p. 23–25) — Obj. : identifier des activités pour rester en forme ; catégoriser les aliments ; définir un menu sain. Vocab. : oil / sweets / egg / meat / pasta / yoghurt. Grammaire : _how many_ (+ countable) / _how much_ (+ uncountable). Activités : reading (true/false + blanks), _Let's write_ (menu sain en binôme à partir de la food pyramid), _Let's pronounce_ (tri Fruit/Vegetables), _Let's play_.
- **Progress Check 2** (p. 25) — Listen/Read : comprehension (matching 1→c,2→a,3→b,4→d ; blanks breakfast/by/lunch/After/seek/his) ; Language (correction/remise en ordre de phrases) ; Writing (catégorisation Vegetables/Unhealthy/Should/Mustn't/Stomachache/Because ; phrases guidées).

#### Unit III — Having fun in the seasons (p. 26–33)

- **Lesson 1 : The Four Seasons** (p. 26–27) — Obj. : nommer les quatre saisons ; décrire les saisons (météo, activités communes). Vocab. : seasons / months / year / autumn / spring / summer / winter / the twelve months of the year. Grammaire : **adjectif dérivé** _Noun + y = adjective_ (cloud + y = cloudy ; règle du redoublement de la consonne finale en C-V-C : foggy, sunny). Activités : nommer saisons + 12 mois, compléter lettres manquantes (windy/cloudy/foggy/snowy/sunny), _Let's speak_ (saison préférée + _because_), _Let's pronounce_, _Let's play_ (classer les mois par saison + couleur : Winter=bleu, Spring=vert, Summer=marron, Autumn=orange).
- **Lesson 2 : Having Fun in Spring** (p. 27–28) — Obj. : identifier les caractéristiques du printemps ; parler des activités de printemps. Vocab. : sky / clear / outside / air / butterfly / next / catch / collect (source : « next » listé 2× — artefact). Grammaire : **futur** _will / 'll_ ; _will not / won't_ ; future words (tomorrow, next year/spring/month/week…). Activités : reading + comprehension + gap-fill (green/air/butterflies), _Let's write_ (guidé par questions), _Let's pronounce_ (/ð/ « the » : weather/with/them/this/mother), _Let's play_ (former des mots).
- **Lesson 3 : Having Fun in Summer** (p. 28–29) — Obj. : identifier les caractéristiques de l'été ; parler des activités estivales. Vocab. : shorts / swimsuit / beach / sandcastle. Grammaire : _to be on holidays_ ; _will / will not + verb_. Activités : reading + match, tri de vêtements (dresses/shorts/skirts/T-shirts/swimsuits), _Let's write_ (email), _Let's pronounce_ (/iː/ + tri S/D), _Let's play_ (comparer 2 images A/B).
- **Lesson 4 : Having Fun in Autumn** (p. 30) — Obj. : identifier les caractéristiques de l'automne ; parler des activités d'automne. Vocab. : warm / cold / windy / go back / meet. Grammaire : **comparatif** (short) _adjective + er (+ than)_ (cold+er = colder than) ; _What's the weather like?_ / _It's + adjective_. Activités : reading (true/false), practise (hotter/colder/warmer/slower/longer), _Let's speak_ (activités d'automne + _because_), _Let's pronounce_, _Let's play_.
- **Lesson 5 : Having Fun in Winter** (p. 31–32) — Obj. : identifier les caractéristiques de l'hiver ; parler des activités hivernales. Vocab. : gloves / socks / fresh air / boots / between / sweater. Grammaire : _Verb + -ING = Activity_. Activités : reading + comprehension + vêtements d'hiver (sweaters/socks/jackets/gloves/boots/hats/coats), _Let's write_ (post « winter is the best season »), _Let's pronounce_ (tri S/D + familles de sons season/play/cold), _Let's play_ (mots en « S » : swing/skate/swim/sofa/skiing).
- **Progress Check 3** (p. 32–33) — Reading Comprehension (tableau Season/weather : Winter–Spring–Summer–Autumn ; true/false) ; Language (adjectifs météo ; present continuous / futur / present ; saisons) ; Writing (mots ; phrases futur/comparatif ; « ID card » de la saison préférée).

#### Unit IV — Caring (p. 33–41)

- **Lesson 1 : Let's help each other** (p. 33–35) — Obj. : lister des exemples d'entraide ; identifier des façons d'aider les autres. Vocab. : care / share / take care / wash the dishes / dirty / tidy / help / each other. Grammaire : **adjectifs possessifs** (I–My, You–Your, He–His, She–Her, It–Its, We–Our, They–Their). Activités : commenter des images (tâches ménagères), _Let's Listen_ (« I love my family ») + remise en ordre, practise (your/my/our/their/her/his), _Let's speak_ (questionnaire + fréquence), _Let's pronounce_ (/h/ audible ou muet : house/home/happy vs hour/where), _Let's play_ (sons initiaux c/a/r/e).
- **Lesson 2 : Sharing is caring** (p. 35–37) — Obj. : comparer lieux et personnes au superlatif ; écrire sur les activités de son quartier. Vocab. : city / come together / neighbour / to clean up / alone / poor / old / young / cross. Grammaire : **superlatif** _the + short adjective + est_ (the clean+est = the cleanest). Activités : reading (email ; sender/receiver expliqués via verb+er) + comprehension, practise (the largest/hottest/coldest/nicest/tallest/smartest/shortest), _Let's write_ (email), _Let's play : Boardgame_ (dé → lire une phrase / décrire une image).
- **Lesson 3 : We are stronger together** (p. 37) — Obj. : décrire cinq métiers ; parler des façons d'aider les autres. Vocab. : village / job / agree / work / say / sick / learn / dice / builder / carpenter / farmer / gamer / participate / competition. Grammaire : **dérivation** _Verb + er = Noun_ (farm+er = farmer). N.B. du guide : « not all jobs are formed as verb + er ». Texte : adapté de _Disagreement Among Occupations_, FreeKidsBooks (freekidsbooks.org). Activités : reading (tableau Who? / What he-she does : farmer/carpenter/builder/teacher), practise (builder/teacher/baker/driver/players/gamer), _Let's speak_, _Let's pronounce_, _Let's play_ (métiers ↔ images).
- **Lesson 4 : Caring for my school** (p. 38–40) — Obj. : dire comment prendre soin de son école ; écrire une charte de classe (classroom charter). Vocab. : clean / collect / tidy-up / mop / blackboard / finish / walk in / floor. Grammaire (en-tête) : **futur (affirmatif/négatif)** will / won't — la _rule_ introduit aussi l'**impératif** et la **prohibition** (Don't + verb). Activités : compléter des bulles (desks/walls/floor/plants/classroom/basket), _Let's Listen_ (Headmaster announcement) + comprehension (tableau Name/Activity : Charlie/Shirly/Shane/Chad/Sarah/Tom), practise (will / will not), _Let's write_ (5 règles de classe), _Let's pronounce_ (/ʃ/ et /ʧ/ tongue twisters), _Let's play_.
- **Lesson 5 : Caring for my pet** (p. 40–41) — Obj. : nommer six animaux de compagnie ; parler de comment prendre soin des animaux. Vocab. : take care of / vet / sick / twice / give / hit. Grammaire : **impératif** (Close the door) ; **prohibition** _Don't + verb_ (usages : ordres/consignes ; interdiction : chose non autorisée / rappel d'une règle / danger-risque). Activités : nommer 6 pets (dog/rabbit/fish/tortoise/cat/bird — audio « Year 4 » réutilisé), reading + comprehension, practise (Don't play / Be / Brush / Read / Eat), _Let's write_ (note dos/don'ts), _Let's pronounce_ (/eɪ/ « Day » vs /aɪ/ « Why »), _Let's play_ (mots-mêlés).
- **Progress Check 4** (p. 41) — Comprehension questions (choix ; true/false) ; Language (verbes au futur affirmatif/négatif ; blanks grandparents'/neighbours/together/rainy/come) ; Writing (mots ; lettres manquantes ; paragraphe de présentation d'un animal — N.B. pronoms he/she + possessifs his/her avec les animaux).

#### Unit V — Celebrations (p. 42–51)

- **Lesson 1 : Children's Day** (p. 42–44) — Obj. : parler de la Journée de l'enfant ; identifier cinq droits de l'enfant. Vocab. : right / education / life / clothing / health / celebrate. Grammaire : _have the right to + something_ (I/You/We/They **have** ; He/She/It **has**). **Ancrage Tunisie** : « In Tunisia, we celebrate it on January, 11 » (occasion de travailler les dates). Activités : reconstituer « Children's Day » (lettres), _Let's speak_ (étiqueter les droits), practise (droits de l'enfant + bulle), _Let's pronounce_ (croiser le mot différent : life/right/sick/child…), _Let's play_ (word wheel : care/life/home/name).
- **Lesson 2 : Mother's Day** (p. 44–45) — Obj. : parler de la fête des mères ; décrire les activités ; **narrer des événements passés**. Vocab. : perfume / give / handbag / cute / necklace / surprised. Grammaire : **simple past (forme affirmative)**. Activités : _Let's Listen_ (tapescript Sam/Andy) + Think-Pair-Share, tableau infinitif→prétérit (buy/bought, write/wrote, prepare/prepared, do/did, have/had, be/was–were, love/loved), practise, _Let's write_, _Let's pronounce_ (/aʊ/ « now/house/cow » vs /ɔɪ/ « joy/boy/toy »), _Let's play_.
- **Lesson 3 : Happy New Year** (p. 45–47) — Obj. : lister les activités d'une fête du Nouvel An ; écrire sur cette fête. Vocab. : hug / snacks / crackers / blow up / put on / midnight / wait for. Grammaire : **simple past** (tableaux infinitif→prétérit : blow up/blew up, drink/drank, eat/ate, wait/waited, put/put, wish/wished, make/made ; say/said, ask/asked, laugh/laughed, talk/talked, count/counted, hug/hugged, want/wanted). Texte adapté d'une vidéo YouTube (« SHORT STORIES »). Activités : reading + comprehension + gap-fill, practise (dialogue put/drank/blew up/ate/came/had/danced), _Let's write_ (mail), _Let's pronounce_, _Let's play_.
- **Lesson 4 : Teacher's Day** (p. 47–49) — Obj. : nommer les objets nécessaires à une fête ; parler des préparatifs. Vocab. : kids / decorate / forget / don't worry / surprise. Grammaire : **simple past**. Activités : _Let's Listen_ (tapescript Headmaster/pupils : Jane/Tom/Alice/Sarah/Olivia/Justin) + comprehension (matching A/B/C), practise (bought/bake/came/play/wrote), _Let's speak_ (groupes de 3 + ground rules), _Let's pronounce_ (**terminaisons -ed /t/, /d/, /ɪd/** : baked/asked/talked/danced ; hugged/played/prepared/lived ; waited/counted/wanted/decorated), _Let's play_.
- **Lesson 5 : End of school year party** (p. 49–51) — Obj. : décrire une fête ; parler d'une fête scolaire. Vocab. : toothache / clown / to miss / gift / wish / doughnut / show / magic. Grammaire : **simple past**. Activités : reading + comprehension, practise (came/bought/thanked/had/played/went/ate), role play, _Let's pronounce_ (match des sons soulignés : school/meet/up/talk/seat → food/door/cut), _Let's play_ (message caché : « Special thanks to all teachers »).
- **Progress Check 5** (p. 50–51) — Reading Comprehension (blog posts ; tableau Name/Country/Children's Day date — Pedro/Tunisia/11 janvier, Samy/Mexico/30 avril, Karina/Poland/1 juin) ; Language (gap-fill enjoyed/together/with/happy ; buy/year/cards/prepare ; simple past Surprised/Clown/Crackers/Gifts) ; Writing (remise en ordre, matching, message à sa mère).

#### Unit VI — Going Shopping (p. 51–59)

- **Lesson 1 : Shops in my Town** (p. 51–52) — Obj. : nommer les différents commerces de la ville ; identifier sept produits (goods) ; localiser les commerces sur une carte. Vocab. : cheap / offer / instrument / jewellery / shopping / ring / necklace / off / sale / gold / silver. Grammaire : **prépositions – simple past**. Activités : _Let's get started_ (class activity, 4 questions sur l'image), _Let's check_ (types de commerces : shoe/music/clothes/coffee/jewellery/toy), practise (simple past + prépositions de lieu : behind / on / next to / between / in front of), Think-Pair-Share, _Let's pronounce_ (/ɪ/ vs /aɪ/), _Let's play_.
- **Lesson 2 : At the Market** (p. 52–54) — Obj. : identifier des mots liés aux commerces d'un marché ; demander et répondre sur le prix des produits ; jouer une conversation dans un magasin. Vocab. : Would like / croissant / meat / pound / customer / sell / buy. Grammaire : _How much is it? — It is… / It's…_ ; **simple past** (last week / last month / five weeks ago). Activités : _Let's Listen_ (tapescript bakery) + comprehension (tableau Bakery / Butcher's / Grocery shop), practise (How much is/are… ; Where's…), _Let's speak_ (role play shopkeeper/customer), _Let's pronounce_ (**-ed /t/ /d/ /ɪd/**), _Let's play_ (themed memory game).
- **Lesson 3 : At the Toy Shop** (p. 54–55) — Obj. : nommer six jouets ; exprimer des préférences. Vocab. : prize / race / present / win / choose / fan / join. Grammaire : **simple past**. Activités : label toys (ball / jump rope / doll / dominoes / kite / teddy bear), reading (Matt/Ned) + comprehension, practise, _Let's write_ (email à Leo — layout : sender/receiver/reason/parts), _Let's pronounce_, _Let's play_.
- **Lesson 4 : At the Clothes Shop** (p. 55–57) — Obj. : nommer des vêtements ; jouer des conversations d'achat. Vocab. : find / tie / present / blazer / vest / sweater / try on clothes / fitting room. Grammaire : **simple past** ; **numbers** (100 = a hundred … 900 = nine hundred ; 468 = four hundred and sixty-eight). Activités : label 12 vêtements (hat/blazer/sweater/skirt/shoes/T-shirt/socks/tie/shorts/shirt/dress/trousers), _Let's Listen_ (tapescript magasin, vêtements d'hiver pour le Canada) + comprehension (tableau Clothes/Price : socks 10 / coat 90 / sweaters 150 / trousers 60 dinars), practise (nombres en toutes lettres), note prépositions _in / on the corner_, _Let's play_ (clothes shop / toy shop / grocery).
- **Lesson 5 : At the Mall** (p. 57–58) — Obj. : identifier différents commerces d'un centre commercial ; décrire une journée de shopping. Vocab. : try on / fit / suit / mall / perfume. Grammaire : **simple past**. Activités : matcher images ↔ noms de commerces, reading (birthday party) + comprehension, practise (live/enjoys/went/visited/bought/chose/found), Think-Pair-Share, _Let's write_ (email), _Let's pronounce_ (/ʤ/ vs /ʒ/ ; /ʃ/ vs /ʧ/), _Let's play_ (Shopping board game au dé — nommer un commerce + 3 articles).
- **Progress Check 6** (p. 58–59) — Reading Comprehension (Phoenix mall ; true/false — texte adapté de studymumbai.com) ; Language (audio + nombres en toutes lettres 199/150/75/250/120/45 ; simple past went/wanted/gave/was/took/paid ; vocab Tie/Try on/Choose/Suit/Ring/Necklace) ; Writing (formation de questions ; matching).
- **Project Idea** (p. 59) — **Simulation** : les apprenants apportent des produits réels ou miniatures de leur choix (clothes / food / toys / jewellery…), installent leurs commerces préférés puis utilisent les objets pour jouer des dialogues.

#### Unit VII — Around the World (p. 59–70)

- **Lesson 1 : Let's travel to…** (p. 59–61) — Obj. : identifier les quatre points cardinaux ; exprimer des préférences ; écrire un paragraphe descriptif sur une ville. Vocab. : world / country / travel / east / west / south / north. Grammaire : **making suggestions**. Activités : travail sur carte (continents/pays : Africa/Asia/America/Oceania/Europe — Tunisia in Africa), _Let's speak_ (numéroter les bulles-pays par préférence de visite), Think-Pair-Share, practise (Let's / What about / How about ; south/north/east/west), _Let's write_ (paragraphe sur une ville + peer review par checklist), _Let's pronounce_ (**k muet** : know / knee / knock / knife, vs k prononcé : kangaroo / kite / kilo / kitchen / key), _Let's play_ (pays par lettre initiale A/F/C/L/T/J).
- **Lesson 2 : Greetings around the world** (p. 61–63) — Obj. : identifier quatre façons de saluer ; décrire des personnes et leurs façons de saluer. Vocab. : greet / cheek / kiss / each other / shake hands / use / think / bow (verb) / touch. Fonction : **articles the / a / an**. Activités : reading (greetings) + comprehension (true/false + phrases : touch noses / handshake / bow), practise (the/a/an), _Let's speak_ (travail de groupe), _Let's pronounce_ (/s/ vs /z/ — terminaisons du pluriel : books/cats vs cars/bags/greetings), _Let's play_ (word search : shake/kiss/bow/greet/cheek). Texte adapté d'afar.com. N.B. compétence interculturelle. **Project idea** : présenter les différentes salutations en Tunisie (jeunes vs aînés, nombre de bises selon les régions…).
- **Lesson 3 : I prefer to travel by…** (p. 63–64) — Obj. : identifier différents moyens de transport ; exprimer des préférences ; écrire sur un voyage en famille. ### Competences & progression

- **Quatre compétences** structurent chaque leçon via des rubriques récurrentes : **écoute/lecture** (_Let's Listen_ / _Let's read_ → comprehension : gist puis scanning), **grammaire en contexte** (_Let's complete the rule_ → _Let's practise_, en 2 temps : observation/conceptualisation d'un exemple tiré du texte, puis consolidation en contexte communicatif), **oral** (_Let's speak_ : role play / simulation), **écrit** (_Let's write_ : du niveau phrase vers de courts textes — posts, emails, notes — approche par **genres** : genre / audience / purpose). S'y ajoutent **prononciation** (_Let's pronounce_) et **jeu** (_Let's play_, souvent en wrap-up).
- **Progression grammaticale (fil directeur transcrit, Unit I → VII)** : goûts (like/enjoy + -ing) → fréquence (always/sometimes/never, +s 3ᵉ pers.) → suggestions (Let's / How about / What about) → invitations (Would you like…?) → conseil/obligation (should/shouldn't, must/mustn't) → capacité (can/can't) → quantité (how many / how much) → dérivation adjectivale (Noun+y) → **futur** (will / won't) → **comparatif** (adj+er than) → **superlatif** (the …est) → dérivation nominale (Verb+er) → impératif & prohibition (Don't) → droits (have/has the right to) → **simple past** (verbes réguliers/irréguliers, terminaisons -ed) → prépositions de lieu + simple past + nombres (Unit VI) → suggestions/comparatif/superlatif + **articles a/an/the** (Unit VII). Progression cohérente **présent → futur → passé**, et **du mot vers le texte**.
- **Life skills** explicitement ciblées par rubrique (Introduction) : acceptance, respect for differences, communication (effective speaking, active listening, expressing ideas, responding appropriately, describing feelings), creativity, critical thinking, decision-making, empathy, perspective-taking, making connections, collaboration, focus & self-control, problem-solving, turn taking, (self-)confidence, adaptability, coping with emotions, stress management, building healthy relationships, self-management.
- **Compétence interculturelle** travaillée en Unit VII (greetings around the world, customs and habits) : respect et appréciation des autres cultures, ouverture, avec ancrage **Tunisie** récurrent (Children's Day 11 janvier ; voyage Djerba/Sousse ; El Jem « biggest amphitheatre in Africa » ; habits traditionnels tunisiens).
- **Évaluation** : un **Progress Check** clôt chaque unité (rubriques Reading Comprehension / Listen-Read / Language / Writing) ; un **Project** clôt chaque unité, évalué via une _evaluation grid_ apprenant décrite p. 8 (auto- et co-évaluation entre pairs : clarté, voix, contact visuel, prononciation, phrases correctes, apport nouveau…).

### Notions/contenus cles

- **Grammaire** : présent simple + adverbes de fréquence ; -ing (goûts, activités) ; modaux should/shouldn't, must/mustn't, can/can't, Would you like…? ; quantifieurs how many / how much ; dérivation Noun+y (adjectif) et Verb+er (nom) ; comparatif court (-er than) & superlatif court (the …est) ; futur will/won't ; impératif & prohibition (Don't) ; have/has the right to ; **simple past** (réguliers + irréguliers, formes -ed) ; prépositions de lieu (on/in, behind, next to, between, in front of) ; **articles a/an/the** ; nombres (dizaines→centaines, en toutes lettres) ; linkers and/but/because ; adjectifs & pronoms possessifs.
- **Lexique thématique (par unité)** : (I) loisirs & plein air ; (II) santé / alimentation / forme / maux ; (III) saisons / mois / météo / vêtements ; (IV) entraide / métiers / école / animaux ; (V) fêtes & célébrations (Children's / Mother's / Teacher's Day, New Year, end-of-year party) ; (VI) commerces, produits, vêtements, jouets, nombres/prix ; (VII) pays & continents, points cardinaux, salutations, moyens de transport, monuments, coutumes.

Techniques : minimal pairs, tri Same/Different, tongue twisters, odd-one-out, classification par son, symboles phonétiques, drilling choral/individuel.
