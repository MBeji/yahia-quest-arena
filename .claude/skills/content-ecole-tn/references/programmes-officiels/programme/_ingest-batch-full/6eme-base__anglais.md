**Cadre pédagogique déclaré (Introduction, p. 3–8).** L'apprenant est un _agent_ (utilisateur actif de l'anglais dans des tâches communicatives), non un récepteur passif. Approches recommandées : constructivisme, socio-constructivisme, cognitivisme, action-oriented approach, Discovery Learning, Total Physical Response (TPR), approches humanistes. Chaque leçon ouvre sur des _learning objectives_, puis les _lexical items_ + _structure(s)_ et/ou _language function(s)_, puis une suite d'étapes menant à une **tâche de production** (orale ou écrite). Le guide fournit corrigés et scripts audio.

### Structure (fidele, exhaustive)

Découpage à deux niveaux : **7 Units** (I → VII), chacune **5 Lessons** + un **Progress Check** de fin d'unité + un **Project** de fin d'unité (le guide annonce p. 8 « The Student's Book comprises **seven** projects to be conducted at the end of each Unit »). **Total : 7 Units · 35 Lessons · 7 Progress Checks · 7 Projects.** L'ouvrage complet est capté ici (Units I → VII, sans troncature).

Chaque fiche-leçon énonce systématiquement : _Objectives_ (« Upon completion of the lesson, the learners will be able to… »), _Vocabulary_, _Grammar_ (parfois _Function_), puis les corrigés des rubriques d'activités. Les rubriques récurrentes d'une leçon : `Let's get started` · `Let's read` / `Let's Listen` · `Let's complete the rule` · `Let's practise` · `Let's speak` · `Let's write` · `Let's pronounce` · `Let's play` (+ `Think / Pair / Share`, `Let's check`).

#### Unit I — Entertaining Myself (p. 9–18)

- **Lesson 1 : Let's Have Fun** (p. 9–11) — Obj. : nommer des activités de loisir ; parler de ses loisirs préférés. Vocab. : fishing / museum / go camping / riding a bike / gardening. Grammaire : _because_ ; _like + verb + -ing_ ; _enjoy + verb + -ing_ (goûts : enjoy/like vs dégoûts : hate/dislike + -ing). Activités : label pictures, tableau tick/cross, dialogue en binôme, _Let's write_ (paragraphe loisirs), _Let's pronounce_ (odd-one-out), _Let's play_.
- **Lesson 2 : My Free Time Activities** (p. 11–13) — Obj. : nommer des loisirs ; écrire sur ses loisirs préférés. Vocab. : drawing / gardening / shopping. Grammaire : **adverbes de fréquence** (always / sometimes / usually / never — accord _+s_ à la 3ᵉ pers.) ; **prépositions** _on / in_ (on the weekend, on holidays, on Monday morning ; in the morning / afternoon / evening — consolidation). Activités : reading (tableau Name/Town/Favourite activities) + match, _Let's speak_ (bulles à compléter, linkers/ponctuation), _Let's pronounce_ (tri Same/Different).
- **Lesson 3 : A Day Out** (p. 13–14) — Obj. : nommer des activités de plein air ; faire et répondre à des suggestions. Vocab. : bored / hide and seek / tired / zoo / far. Grammaire : _Let's + infinitive_ / _How about + V-ing?_ / _What about + V-ing?_. Fonction : **making suggestions**. Activités : _Let's Listen_ (tapescript Adam/Linda) + comprehension + remise en ordre, role play, _Let's pronounce_ (/s/ vs /z/).
- **Lesson 4 : Let's go on a picnic** (p. 14–16) — Obj. : nommer des objets de pique-nique ; parler des activités de pique-nique. Vocab. : get ready / river / wash / arrive / rope skipping / fly a kite. Grammaire : linkers _and – but_ ; adjectifs possessifs _his – her – their_ (I–my … they–their). Activités : reading (true/false + match), Think-Pair-Share, _Let's write_ (5 phrases → paragraphe), _Let's pronounce_, _Let's play_.
- **Lesson 5 : Pass the Ball** (p. 16–17) — Obj. : nommer des loisirs ; faire offres et requêtes avec _Would you like…?_. Vocab. : lose / team / win / kick / run. Grammaire : _Would you like…?_ (inviting = Would you like + to + verb ? ; making a suggestion = Let's + verb). Activités : label pictures, reading (beach) + comprehension, role play (Speaker 1/2), _Let's pronounce_ (match des sons ; play/draw/park), _Let's play_ (sports matching).
- **Progress Check 1** (p. 17–18) — Reading Comprehension (tableau Name/Age/Town/Favourite activities ; true/false ; blanks) ; Language (pronoms she/her/he/their/they ; would you like/how about/let's) ; Writing (remise en ordre de phrases + brainstorming → paragraphe).

#### Unit II — Keeping Fit (p. 18–25)

- **Lesson 1 : I'm Healthy, I'm Happy** (p. 18–19) — Obj. : classer les aliments sains/malsains ; donner des conseils pour rester en bonne santé. Vocab. : fruits / vegetables / lots / healthy / unhealthy. Grammaire : _should / shouldn't_ (giving advice). Activités : label + tick/cross healthy foods, questionnaire + dialogue de conseil, _Let's pronounce_, _Let's play_.
- **Lesson 2 : Eat well, Live well** (p. 19–20) — Obj. : décrire les éléments d'une alimentation saine ; dire aux autres de faire des choix sains avec _must / mustn't_. Vocab. : vegetables / teeth / egg / keep fit / bad for. Grammaire : _must / mustn't_. Activités : _Let's Listen_ (tapescript Anna/mum) + comprehension, _Let's write_ (health poster, groupe de 4), _Let's pronounce_ (/ɪ/ vs /iː/), _Let's play_.
- **Lesson 3 : I'm Healthy, I'm Fit** (p. 20–22) — Obj. : décrire personnes en forme / hors forme ; distinguer habitudes saines et malsaines. Vocab. : doughnuts / fit / fat / tired / excited / sad / unhappy. Grammaire : _should/shouldn't_ ; _must/mustn't_ ; **can / can't** (expressing ability / inability). Activités : spot-the-difference + comparaison (slim/fat/thin/fit), reading (true/false + match), _Let's speak_, _Let's pronounce_, _Let's play_ (healthy food).
- **Lesson 4 : How are you Feeling?** (p. 22–23) — Obj. : parler de trois maux (ailments) ; donner des conseils santé. Vocab. : toothache / stomachache / sick / tired / headache. Grammaire : _have a + ailment_ ; _to be + adjective_. Activités : listen/repeat/mime, _Let's read + Let's Listen_ simultanés (Simon/doctor) + comprehension, role play patient/docteur, _Let's pronounce_, _Let's play_ (grille de mots-mêlés).
- **Lesson 5 : How to Stay Fit** (p. 23–25) — Obj. : identifier des activités pour rester en forme ; catégoriser les aliments ; définir un menu sain. Vocab. : oil / sweets / egg / meat / pasta / yoghurt. Grammaire : _how many_ (+ countable) / _how much_ (+ uncountable). Activités : reading (true/false + blanks), _Let's write_ (menu sain en binôme à partir de la food pyramid), _Let's pronounce_ (fruit/vegetables sorting), _Let's play_.
- **Progress Check 2** (p. 25) — Listen/Read : comprehension (tableau ; blanks breakfast/lunch/after… ; matching) ; Language (matching, remise en ordre de phrases) ; Writing (catégorisation Vegetables/Unhealthy/Should… , phrases guidées).

#### Unit III — Having fun in the seasons (p. 26–33)

- **Lesson 1 : The Four Seasons** (p. 26–27) — Obj. : nommer les quatre saisons ; décrire les saisons (météo, activités communes). Vocab. : seasons / months / year / autumn / spring / summer / winter / the twelve months of the year. Grammaire : **adjectif dérivé** _Noun + y = adjective_ (cloud + y = cloudy ; règle du redoublement de la consonne finale en C-V-C : foggy, sunny). Activités : nommer saisons + 12 mois, compléter lettres manquantes (windy/cloudy/foggy/snowy/sunny), _Let's speak_ (saison préférée + _because_), _Let's pronounce_, _Let's play_ (classer les mois par saison + couleur).
- **Lesson 2 : Having Fun in Spring** (p. 27–28) — Obj. : identifier les caractéristiques du printemps ; parler des activités de printemps. Vocab. : sky / clear / outside / air / butterfly / next / catch / collect. Grammaire : **futur** _will / 'll_ ; _will not / won't_ ; future words (tomorrow, next year/spring/month/week…). Activités : reading + comprehension + gap-fill, _Let's write_ (guidé par questions), _Let's pronounce_ (/ð/ « the » : weather/with/them/this/mother), _Let's play_ (former des mots).
- **Lesson 3 : Having Fun in Summer** (p. 28–29) — Obj. : identifier les caractéristiques de l'été ; parler des activités estivales. Vocab. : shorts / swimsuit / beach / sandcastle. Grammaire : _to be on holidays_ ; _will / will not + verb_. Activités : reading + match, tri de vêtements (dresses/shorts/skirts/T-shirts/swimsuits), _Let's write_ (email), _Let's pronounce_ (/iː/ + tri S/D), _Let's play_ (comparer 2 images A/B).
- **Lesson 4 : Having Fun in Autumn** (p. 30) — Obj. : identifier les caractéristiques de l'automne ; parler des activités d'automne. Vocab. : warm / cold / windy / go back / meet. Grammaire : **comparatif** (short) _adjective + er (+ than)_ (cold+er = colder than) ; _What's the weather like?_ / _It's + adjective_. Activités : reading (true/false), practise (hotter/colder/warmer/slower/longer), _Let's speak_ (activités + _because_), _Let's pronounce_, _Let's play_.
- **Lesson 5 : Having Fun in Winter** (p. 31–32) — Obj. : identifier les caractéristiques de l'hiver ; parler des activités hivernales. Vocab. : gloves / socks / fresh air / boots / between / sweater. Grammaire : _Verb + -ING = Activity_. Activités : reading + comprehension + vêtements d'hiver (sweaters/socks/jackets/gloves/boots/hats/coats), _Let's write_ (post « winter is the best season »), _Let's pronounce_ (tri S/D + familles de sons season/play/cold), _Let's play_ (mots en « S » : swing/skate/swim/sofa/skiing).
- **Progress Check 3** (p. 32–33) — Reading Comprehension (tableau Season/weather Winter–Spring–Summer–Autumn ; true/false) ; Language (adjectifs météo ; present continuous / futur / present ; saisons) ; Writing (mots ; phrases futur/comparatif ; ID card de la saison préférée).

#### Unit IV — Caring (p. 33–41)

- **Lesson 1 : Let's help each other** (p. 33–35) — Obj. : lister des exemples d'entraide ; identifier des façons d'aider les autres. Vocab. : care / share / take care / wash the dishes / dirty / tidy / help / each other. Grammaire : **adjectifs possessifs** (I–My, You–Your, He–His, She–Her, It–Its, We–Our, They–Their). Activités : commenter des images, _Let's Listen_ (« I love my family ») + remise en ordre, practise (your/my/our/their/her/his), _Let's speak_ (questionnaire + fréquence), _Let's pronounce_ (/h/ audible ou muet : house/home vs hour/where), _Let's play_ (sons initiaux c/a/r/e).
- **Lesson 2 : Sharing is caring** (p. 35–37) — Obj. : comparer lieux et personnes au superlatif ; écrire sur les activités de son quartier. Vocab. : city / come together / neighbour / to clean up / alone / poor / old / young / cross. Grammaire : **superlatif** _the + short adjective + est_ (the clean+est = the cleanest). Activités : reading (email, sender/receiver via verb+er) + comprehension, practise (the largest/hottest/coldest/nicest/tallest/smartest/shortest), _Let's write_ (email), _Let's play : Boardgame_ (dé → lire une phrase / décrire une image).
- **Lesson 3 : We are stronger together** (p. 37) — Obj. : décrire cinq métiers ; parler des façons d'aider les autres. Vocab. : village / job / agree / work / say / sick / learn / dice / builder / carpenter / farmer / gamer / participate / competition. Grammaire : **dérivation** _Verb + er = Noun_ (farm+er = farmer). Note du guide : « not all jobs are formed as verb + er ». Texte : adapté de _Disagreement Among Occupations_, FreeKidsBooks (freekidsbooks.org). Activités : reading (tableau Who? / What he-she does : farmer/carpenter/builder/teacher), practise (builder/teacher/baker/driver/players/gamer), _Let's speak_, _Let's pronounce_, _Let's play_ (métiers ↔ images).
- **Lesson 4 : Caring for my school** (p. 38–40) — Obj. : dire comment prendre soin de son école ; écrire une charte de classe (classroom charter). Vocab. : clean / collect / tidy-up / mop / blackboard / finish / walk in / floor. Grammaire : **futur (affirmatif/négatif)** will / won't. Activités : compléter des bulles (desks/walls/floor/plants/classroom/basket), _Let's Listen_ (Headmaster announcement) + comprehension (tableau Name/Activity : Charlie/Shirly/Shane/Chad/Sarah/Tom), practise (will / will not), _Let's write_ (5 règles de classe), _Let's pronounce_ (/ʃ/ et /ʧ/ tongue twisters), _Let's play_.
- **Lesson 5 : Caring for my pet** (p. 40–41) — Obj. : nommer six animaux de compagnie ; parler de comment prendre soin des animaux. Vocab. : take care of / vet / sick / twice / give / hit. Grammaire : **impératif** (Close the door) ; **prohibition** _Don't + verb_ (ordres/consignes ; interdiction : chose non autorisée / règle existante / danger/risque). Activités : nommer 6 pets (dog/rabbit/fish/tortoise/cat/bird — audio Year 4 réutilisé), reading + comprehension, practise (Don't play / Be / Brush / Read / Eat), _Let's write_ (note dos/don'ts), _Let's pronounce_ (/eɪ/ « Day » vs /aɪ/ « Why »), _Let's play_ (mots-mêlés).
- **Progress Check 4** (p. 41) — Comprehension questions (choix ; true/false) ; Language (verbes au futur affirmatif/négatif ; blanks grandparents'/neighbours/together/rainy/come) ; Writing (mots ; lettres manquantes ; paragraphe de présentation d'un animal — N.B. pronoms he/she + possessifs his/her avec les animaux).

#### Unit V — Celebrations (p. 42–51)

- **Lesson 1 : Children's Day** (p. 42–44) — Obj. : parler de la Journée de l'enfant ; identifier cinq droits de l'enfant. Vocab. : right / education / life / clothing / health / celebrate. Grammaire : _have the right to + something_ (I/You/We/They **have** ; He/She/It **has**). **Ancrage Tunisie** : « In Tunisia, we celebrate it on January, 11 » (occasion de travailler les dates). Activités : reconstituer « Children's Day », _Let's speak_ (étiqueter les droits), practise (droits de l'enfant + bulle), _Let's pronounce_ (croiser le mot différent : life/right/sick/child…), _Let's play_ (word wheel : care/life/home/name).
- **Lesson 2 : Mother's Day** (p. 44–45) — Obj. : parler de la fête des mères ; décrire les activités ; **narrer des événements passés**. Vocab. : perfume / give / handbag / cute / necklace / surprised. Grammaire : **simple past (forme affirmative)**. Activités : _Let's Listen_ (tapescript Sam/Andy) + Think-Pair-Share, tableau infinitif→prétérit (buy/bought, write/wrote, prepare/prepared, do/did, have/had, be/was-were, love/loved), practise, _Let's write_, _Let's pronounce_ (/aʊ/ « now/house/cow » vs /ɔɪ/ « joy/boy/toy »), _Let's play_.
- **Lesson 3 : Happy New Year** (p. 45–47) — Obj. : lister les activités d'une fête du Nouvel An ; écrire sur cette fête. Vocab. : hug / snacks / crackers / blow up / put on / midnight / wait for. Grammaire : **simple past** (tableau infinitif→prétérit : blow up/blew up, drink/drank, eat/ate, wait/waited, put/put, wish/wished, make/made ; say/said, ask/asked, laugh/laughed, talk/talked, count/counted, hug/hugged, want/wanted). Texte adapté d'une vidéo YouTube (SHORT STORIES). Activités : reading + comprehension + gap-fill, practise (dialogue put/drank/blew up/ate/came/had/danced), _Let's write_ (mail), _Let's pronounce_, _Let's play_.
- **Lesson 4 : Teacher's Day** (p. 47–49) — Obj. : nommer les objets nécessaires à une fête ; parler des préparatifs. Vocab. : kids / decorate / forget / don't worry / surprise. Grammaire : **simple past**. Activités : _Let's Listen_ (tapescript Headmaster/pupils : Jane/Tom/Alice/Sarah/Olivia/Justin) + comprehension (matching A/B/C), practise (bought/bake/came/play/wrote), _Let's speak_ (groupes de 3 + ground rules), _Let's pronounce_ (**terminaisons -ed /t/, /d/, /ɪd/** : baked/asked/talked/danced ; hugged/played/prepared/lived ; waited/counted/wanted/decorated), _Let's play_.
- **Lesson 5 : End of school year party** (p. 49–51) — Obj. : décrire une fête ; parler d'une fête scolaire. Vocab. : toothache / clown / to miss / gift / wish / doughnut / show / magic. Grammaire : **simple past**. Activités : reading + comprehension, practise (came/bought/thanked/had/played/went/ate), role play, _Let's pronounce_ (match des sons soulignés : school/meet/up/talk/seat → food/door/cut), _Let's play_ (message caché : « Special thanks to all teachers »).
- **Progress Check 5** (p. 50–51) — Reading Comprehension (blog posts ; tableau Name/Country/Children's Day date : Pedro–Tunisia–Jan 11, Samy–Mexico–Apr 30, Karina–Poland–Jun 1) ; Language (gap-fill enjoyed/together/with/happy ; buy/year/cards/prepare ; simple past) ; Writing (remise en ordre, matching, message à sa mère).

#### Unit VI — Going Shopping (p. 51–59)

- **Lesson 1 : Shops in my Town** (p. 51–52) — Obj. : nommer les différents commerces de la ville ; identifier sept produits (goods) ; localiser les commerces sur une carte. Vocab. : cheap / offer / instrument / jewellery / shopping / ring / necklace / off / sale / gold / silver. Grammaire : **prépositions – simple past**. Activités : _Let's get started_ (class activity, 4 questions sur l'image), _Let's check_ (types de commerces : shoe/music/clothes/coffee/jewellery/toy), practise (simple past + prépositions de lieu : behind / on / next to / between / in front of), Think-Pair-Share, _Let's pronounce_ (/ɪ/ vs /aɪ/), _Let's play_.
- **Lesson 2 : At the Market** (p. 52–54) — Obj. : identifier des mots liés aux commerces d'un marché ; demander et répondre sur le prix des produits ; jouer une conversation dans un magasin. Vocab. : Would like / croissant / meat / pound / customer / sell / buy. Grammaire : _How much is it? — It is… / It's…_ ; **simple past** (last week, last month, five weeks ago). Activités : _Let's Listen_ (tapescript bakery) + comprehension (tableau Bakery/Butcher's/Grocery shop), practise (How much is/are… ; Where's…), _Let's speak_ (role play shopkeeper/customer), _Let's pronounce_ (**-ed /t/ /d/ /ɪd/**), _Let's play_ (themed memory game).
- **Lesson 3 : At the Toy Shop** (p. 54–55) — Obj. : nommer six jouets ; exprimer des préférences. Vocab. : prize / race / present / win / choose / fan / join. Grammaire : **simple past**. Activités : label toys (ball / jump rope / doll / dominoes / kite / teddy bear), reading (Matt/Ned) + comprehension, practise, _Let's write_ (email à Leo — layout : sender/receiver/reason/parts), _Let's pronounce_, _Let's play_.
- **Lesson 4 : At the Clothes Shop** (p. 55–57) — Obj. : nommer des vêtements ; jouer des conversations d'achat. Vocab. : find / tie / present / blazer / vest / sweater / try on clothes / fitting room. Grammaire : **simple past** ; **numbers** (100 = a hundred … 900 = nine hundred ; 468 = four hundred and sixty-eight). Activités : label 12 vêtements (hat/blazer/sweater/skirt/shoes/T-shirt/socks/tie/shorts/shirt/dress/trousers), _Let's Listen_ (tapescript magasin de vêtements d'hiver) + comprehension (tableau Clothes/Price), practise (nombres en toutes lettres), note prépositions _in / on the corner_, _Let's play_ (clothes shop / toy shop / grocery).
- **Lesson 5 : At the Mall** (p. 57–58) — Obj. : identifier différents commerces d'un centre commercial ; décrire une journée de shopping. Vocab. : try on / fit / suit / mall / perfume. Grammaire : **simple past**. Activités : matcher images ↔ noms de commerces, reading (birthday party) + comprehension, practise (live/enjoys/went/visited/bought/chose/found), Think-Pair-Share, _Let's write_ (email), _Let's pronounce_ (/ʤ/ vs /ʒ/ ; /ʃ/ vs /ʧ/), _Let's play_ (Shopping board game au dé — nommer un commerce + 3 articles).
- **Progress Check 6** (p. 58–59) — Reading Comprehension (Phoenix mall ; true/false — texte adapté de studymumbai.com) ; Language (audio + nombres en toutes lettres 199/150/75/250/120/45 ; simple past went/wanted/gave/was/took/paid ; vocab tie/try on/choose/suit/ring/necklace) ; Writing (formation de questions ; matching).
- **Project Idea** (p. 59) — **Simulation** : les apprenants apportent des produits réels ou miniatures de leur choix (clothes / food / toys / jewellery…), installent leurs commerces préférés puis utilisent les objets pour jouer des dialogues.

#### Unit VII — Around the World (p. 59–70)

- **Lesson 1 : Let's travel to…** (p. 59–61) — Obj. : identifier les quatre points cardinaux ; exprimer des préférences ; écrire un paragraphe descriptif sur une ville. Vocab. : world / country / travel / east / west / south / north. Grammaire : **making suggestions**. Activités : travail sur carte (continents/pays : Africa/Asia/America/Oceania/Europe — Tunisia in Africa), _Let's speak_ (numéroter les bulles-pays par préférence), Think-Pair-Share, practise (Let's / What about / How about ; south/north/east/west), _Let's write_ (paragraphe sur une ville + peer review avec checklist), _Let's pronounce_ (**k muet** : know/knee/knock/knife/knife…), _Let's play_ (pays par lettre A/F/C/L/T/J).
- **Lesson 2 : Greetings around the world** (p. 61–63) — Obj. : identifier quatre façons de saluer ; décrire des personnes et leurs façons de saluer. Vocab. : greet / cheek / kiss / each other / shake hands / use / think / bow (verb) / touch. Fonction : **articles the / a / an**. Activités : reading (greetings) + comprehension (true/false + phrases : touch noses / handshake / bow), practise (the/a/an), _Let's speak_ (travail de groupe), _Let's pronounce_ (/s/ vs /z/ — terminaisons du pluriel books/cats vs cars/bags), _Let's play_ (word search : shake/kiss/bow/greet/cheek). Texte adapté d'afar.com. N.B. compétence interculturelle. **Project idea** : présenter les différentes salutations en Tunisie (jeunes vs aînés, nombre de bises selon les régions…).
- **Lesson 3 : I prefer to travel by…** (p. 63–64) — Obj. : identifier différents moyens de transport ; exprimer des préférences ; écrire sur un voyage en famille. Vocab. : spend / surprise / fast / slow / prefer. Fonction : **comparatif** ; _Prefer + to + verb_ ; _Prefer + verb + -ING_ ; _travel + by + moyen_ / _travel + to + lieu_. Activités : label moyens de transport (plane/bus/bike/ship/train/car), _Let's Listen_ (tapescript voyage à Djerba/Sousse) + comprehension, practise (to / walking / by / the), _Let's write_, _Let's pronounce_ (/ɜː/ first/third/shirt/skirt/girl/dirty vs /ɪ/ did/fit/sit/ship), _Let's play_ (transport).
- **Lesson 4 : Monuments in the world** (p. 65–67) — Obj. : identifier cinq monuments du monde ; nommer et décrire des monuments célèbres. Vocab. : monument / amphitheatre / famous / build (built) / old ≠ new / start ≠ finish. Fonction : **comparatif** ; **superlatif** (the + adjective + est). Activités : matcher monuments ↔ drapeaux (the pyramids / the Eiffel Tower / El Jem amphitheatre / the Great Wall of China / Big Ben), **Jigsaw reading** + tableau (Monument/Country/Town/Date of building : pyramids–Egypt–Giza ; El Jem–Tunisia–Mahdia–238 ; Eiffel Tower–France–Paris–1889 ; Big Ben–England–London–1859 ; Great Wall–China), practise (shortest/smallest/longest/highest), _Let's speak_ (présenter un monument), _Let's pronounce_ (/i/ vs /aɪ/), _Let's play_ (recherche Internet sur des monuments — devoir).

### Competences & progression

- **Quatre compétences** structurent chaque leçon via des rubriques récurrentes : **écoute/lecture** (_Let's Listen_ / _Let's read_ → comprehension : gist puis scanning), **grammaire en contexte** (_Let's complete the rule_ → _Let's practise_, en 2 temps : observation/conceptualisation d'un exemple tiré du texte, puis consolidation en contexte communicatif), **oral** (_Let's speak_ : role play / simulation), **écrit** (_Let's write_ : du niveau phrase vers de courts textes — posts, emails, notes — approche par **genres** : genre / audience / purpose). S'y ajoutent **prononciation** (_Let's pronounce_) et **jeu** (_Let's play_, souvent en wrap-up).
- **Progression grammaticale (fil directeur transcrit, Unit I → VII)** : goûts (like/enjoy + -ing) → fréquence (always/sometimes/never, +s 3ᵉ pers.) → suggestions (Let's / How about / What about) → invitations (Would you like…?) → conseil/obligation (should/shouldn't, must/mustn't) → capacité (can/can't) → quantité (how many / how much) → dérivation adjectivale (Noun+y) → **futur** (will / won't) → **comparatif** (adj+er than) → **superlatif** (the …est) → dérivation nominale (Verb+er) → impératif & prohibition (Don't) → droits (have/has the right to) → **simple past** (verbes réguliers/irréguliers, terminaisons -ed) → prépositions de lieu + simple past + nombres (Unit VI) → suggestions/comparatif/superlatif + **articles a/an/the** (Unit VII). Progression cohérente **présent → futur → passé**, et **du mot vers le texte**.
- **Life skills** explicitement ciblées par rubrique (Introduction) : acceptance, respect for differences, communication (effective speaking, active listening, expressing ideas, responding appropriately, describing feelings), creativity, critical thinking, decision-making, empathy, perspective-taking, making connections, collaboration, focus & self-control, problem-solving, turn taking, (self-)confidence, adaptability, coping with emotions, stress management, building healthy relationships, self-management.
- **Compétence interculturelle** travaillée en Unit VII (greetings around the world, customs and habits) : respect et appréciation des autres cultures, ouverture, avec ancrage **Tunisie** récurrent (Children's Day 11 janvier ; voyage Djerba/Sousse ; El Jem « biggest amphitheatre in Africa » ; habits traditionnels tunisiens).
- **Évaluation** : un **Progress Check** clôt chaque unité (rubriques Reading Comprehension / Listen-Read / Language / Writing) ; un **Project** clôt chaque unité, évalué via une _evaluation grid_ apprenant décrite p. 8 (auto- et co-évaluation entre pairs).
