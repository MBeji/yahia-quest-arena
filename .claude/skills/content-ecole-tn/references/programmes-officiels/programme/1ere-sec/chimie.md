**Chapitre II — STRUCTURE DE LA MATIÈRE (à l'échelle microscopique)** (ouverture p.18 ; plan p.19) — organisé en **3 leçons** :

- _Photo d'ouverture / question_ : dune de sable — « Peut-on imaginer que la matière est divisible indéfiniment ? »
- _Prérequis_ : puissances entières de dix ; phénomène d'électrisation ; notion de charge électrique ; les deux espèces d'électricité.
- **Leçon A — Discontinuité de la matière** (cours p.20–22 ; essentiel p.23 ; exercices p.24–25)
  - I. Divisibilité de la matière (gravier/sable ; pulvériseur de parfum ; éther volatil). II. La divisibilité est-elle limitée ? → **molécules**, matière **discontinue**. III. Les molécules sont-elles toutes identiques ? (dioxygène vs dioxyde de carbone). IV. Ordres de grandeur **dimension / masse d'une molécule** (**Angström**, 1 Å = 10⁻¹⁰ m ; masse ≈ 10⁻²⁶ kg). V. Propriétés physiques eau liquide / glace / vapeur → structures **condensée-ordonnée** (solide) / **condensée-désordonnée** (liquide) / **non condensée-désordonnée** (gaz).
- **Leçon B — Atomes et ions simples** (cours p.26–29 ; essentiel p.30 ; exercices p.31–32 ; document p.33)
  - I. La matière est-elle toujours moléculaire ? → **atome**, structure atomique (or, métaux, gaz rares). II. Ordres de grandeur dimension/masse d'un atome (≈ Å ; ≈ 10⁻²⁶ kg ; tableau H, C, O, S, Fe, Au). III. De quoi est formé un atome ? → **électron** (charge −1,6·10⁻¹⁹ C ; masse 9·10⁻³¹ kg), **charge élémentaire e**, **noyau**, **structure lacunaire**, **nuage électronique**, neutralité électrique. IV. **Symboliser un atome** (symboles chimiques ; modèles boules colorées). V. De l'atome à l'**ion simple** : **cation** / **anion** ; tableau atome↔ion (H⁺, O²⁻, Cl⁻, Cu⁺/Cu²⁺, Na⁺, Fe²⁺/Fe³⁺, Al³⁺, S²⁻).
  - _Document_ : « **Voir les atomes : rêve ou réalité ?** » (microscope optique/électronique, microscope à champ ionique, effet tunnel ; Louis de Broglie).
- **Leçon C — Molécules et ions polyatomiques** (cours p.34–37 ; exercice résolu p.38 ; exercices p.39–40 ; exercice documentaire p.41 ; document p.42 ; T.P. p.43)
  - I. De quoi est formée une molécule ? → assemblage d'atomes, **corps simple / corps composé**. II. **Modéliser** : **modèle compact** / **modèle éclaté**. III. Ce qui différencie les molécules : nature des atomes, **atomicité**, **géométrie**. IV. **Représenter une molécule** : la **formule** (indices). V. **Ion polyatomique** (nitrate, ammonium, carbonate, sulfate, hydronium…). VI. Corps pur à structure ionique → **formule statistique** (NaCl, NH₄Cl, …).
  - _Activités_ : **exercice résolu** (étiquette d'eau minérale : mélange, ions, charge, symboles/formules) ; vrai/faux (10) ; Q.C.M. (3) ; **exercice documentaire « Dalton »** (théorie atomique de 1807) ; **Document « Ions et santé »** (oligo-éléments : chrome, cobalt, cuivre, fluor, fer, iode, calcium ; carences/excès).
  - **T.P. « Utilisation des modèles moléculaires »** (BUT / MATÉRIEL / MANIPULATION ; molécules eau, dihydrogène, dioxygène, dioxyde de carbone, ammoniac ; géométrie L/P/S = linéaire/plane/spatiale ; tableau à compléter).

### Compétences & progression

Aucun **guide enseignant / référentiel de compétences formel** n'est présent dans le lot (source = manuel élève seul). La progression est explicite dans l'**enchaînement des parties et les prérequis** de chaque chapitre :
mélange → corps pur → discontinuité (molécule) → atome / ion simple → molécule / ion polyatomique → **quantité de matière / mole / masse molaire / volume molaire** → solutions (dissolution → concentration/solubilité) → réaction chimique (notion → qualitatif → quantitatif : conservation, équation, stœchiométrie) → chimie organique (hydrocarbures).
Les **objectifs opérationnels** sont lisibles dans les intitulés de section, formulés en questions (« Comment symboliser un atome ? », « Comment relier la masse et la quantité de matière ? », « De quoi dépend la solubilité ? », « Les réactifs disparaissent-ils complètement à la fin d'une réaction chimique ? ») et dans les rubriques **BUT** des T.P. Chaque T.P. suit la trame BUT / PRINCIPE / MATÉRIEL / MANIPULATION. La gradation des exercices (« Vérifier ses acquis » → « situations simples » → « synthèse ») matérialise les niveaux taxonomiques. Les objectifs pédagogiques normatifs du programme officiel restent **à confronter au guide enseignant** (non fourni dans ce lot).
