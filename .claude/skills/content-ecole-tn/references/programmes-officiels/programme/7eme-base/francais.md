## 1. Cadre & compétences (les objectifs généraux)

> ⚠️ **Le document ne formule AUCUNE compétence terminale ni objectif général.** Il n'y a pas de
> page liminaire « compétences / objectifs / méthode » : après la couverture (p.1) et une page blanche
> (p.2), le cahier **entre directement dans les exercices** (p.3 : texte de compréhension « Le conte de
> Robinson Crusoé »). Le seul cadre revendiqué est la mention de couverture **« Conforme aux programmes
> officiels 2006-2007 »**. Ce qui suit n'est donc **pas** une transcription de compétences officielles
> mais la **reconstitution du périmètre effectivement travaillé** par le cahier (à confirmer sur le
> véritable guide CNP — voir §6).

Périmètre des activités (déduit de l'organisation interne du cahier — 7ème = **1re année du collège**,
3ème degré de l'enseignement de base) :

- **Lecture / compréhension de l'écrit** : lire des textes variés (récits, contes, articles de
  journal, fiches de lecture…) ; repérer personnages, narrateur, type de texte ; relever
  synonymes/substituts ; donner un titre ; émettre des hypothèses.
- **Conjugaison** : maîtriser **présent de l'indicatif, passé composé, futur simple, impératif** ;
  accords du participe passé (être / avoir) ; verbes pronominaux et verbes de mouvement.
- **Syntaxe / grammaire** : types et formes de phrases ; phrase verbale / non verbale / minimale ;
  déterminants (articles, possessifs, démonstratifs, numéraux) ; expansion du groupe nominal
  (adjectif épithète/attribut, complément de nom, proposition relative) ; compléments
  circonstanciels ; pronoms personnels et substituts.
- **Orthographe** : homophones grammaticaux ; accords ; graphies de sons.
- **Vocabulaire** : synonymes, mots contraires, familles de mots, différents sens d'un mot, champs
  lexicaux thématiques.
- **Expression écrite / production** : insérée dans les exercices (« Fais parler… », « Raconte… »,
  « Écris un paragraphe… », « Rédige un texte pour présenter… ») — récit, dialogue inséré, passage
  descriptif. **Pas de « projet d'écriture » formalisé** dans le cahier.

> **Borne haute (vs 8ème)** : la 7ème reste sur les **quatre temps de base** (présent, passé composé,
> futur simple, impératif) et l'**expansion simple du GN** ; l'imparfait/plus-que-parfait systématiques,
> le conditionnel, le subjonctif, la subordination complexe (complétives, circonstancielles), la voix
> passive et le discours rapporté relèvent de la suite du collège (8ème → 9ème). **À confirmer sur le
> guide officiel** (le cahier n'énonce pas ces bornes — voir §6).

### Module de contrôle et de consolidation des prérequis — p.3 sq.

- **Support de lecture d'ouverture** : « **Le conte de Robinson Crusoé** » (texte + questionnaire de
  compréhension : personnages, narrateur, synonymes, substituts du nom « Robinson », titre, forme
  affirmative/négative, GV porteur de l'information essentielle, description du visage du héros).
- **Faits de langue révisés** (encadrés-leçon) :
  - **Conjugaison — Le présent de l'indicatif** : « indique une action en train de se réaliser ; peut
    remplacer les temps du passé (récit plus vivant) ; exprime une vérité générale » ; terminaisons du
    présent (les trois groupes).
  - **Conjugaison — Le passé composé** : « formé de l'auxiliaire _être_ ou _avoir_ au présent + le
    participe passé » ; terminaisons du participe passé (-é / -i / -u / -is / -it…) ; **avec _être_, le
    participe passé s'accorde avec le sujet** ; verbes de mouvement & **verbes pronominaux** au passé
    composé avec _être_ (accord avec le GN sujet).
  - **Syntaxe — Les types de phrases** : phrase **déclarative** (donner une information),
    **interrogative** (poser une question), **exclamative** (exprimer un sentiment : joie/colère/
    tristesse) ; chaque type peut être de **forme affirmative ou négative**.
  - **Orthographe — Les homophones `ce / se`, `ces / ses`, `c'est / s'est`** : _ces_ = déterminant
    démonstratif (nom pluriel) ; _ses_ = déterminant possessif (nom pluriel) ; _ce_ = déterminant
    démonstratif (masc. sing.) ; _c'est_ ≠ se conjugue qu'à la 3ᵉ pers. sing. ; _s'est_ toujours suivi
    d'un participe passé.
- **Bornes de scope** :
  - ✅ INCLUS : réactivation présent / passé composé ; accord du PP avec _être_ ; types & formes de
    phrases ; homophones `ce/se`, `ces/ses`, `c'est/s'est`.
  - ⛔ EXCLU : ce module **ne fait que consolider** des acquis du primaire ; pas de notion nouvelle ici.

### Module 1 — Thème : **En famille** — p.6 sq.

- **Lecture** : textes narratifs (vie de famille ; ex. « Toute la soirée, j'ai fait la leçon à Rex… »,
  scènes scolaires/familiales) ; questions sur personnages, narrateur, substituts.
- **Conjugaison** :
  - **Le passé composé** (suite) — écrire/conjuguer au passé composé ; accord du PP ; verbes
    pronominaux (« se brosser les dents » au passé composé) et de mouvement avec _être_.
  - **Le futur simple** : « exprime une action à venir ou un ordre ; au futur, **tous les verbes ont
    les mêmes terminaisons** » (-rai, -ras, -ra, -rons, -rez, -ront).
  - **L'impératif** : « employé pour donner des ordres, des conseils ; le sujet n'est pas exprimé ; le
    verbe ne compte que **trois personnes** » (terminaisons de l'impératif).
- **Syntaxe** :
  - **Phrase verbale / non verbale** : « une phrase qui contient un verbe conjugué est _verbale_ ; sans
    verbe conjugué, _non verbale_ ».
  - **La phrase minimale** : « constituée d'un **groupe sujet** (fait l'action) et d'un **groupe
    verbal** (exprime ce que fait/est le sujet) » ; compléments circonstanciels (compléter une phrase).
- **Orthographe** : **homophones `à / as / a`**.
- **Production insérée** : « Fais parler les personnages » (impératif) ; « Sami projette de faire la
  même chose, fais parler Sami » (futur).
- **Bornes de scope** :
  - ✅ INCLUS : passé composé, futur simple, impératif ; phrase verbale/non verbale/minimale ;
    compléments circonstanciels ; homophones `à/as/a`.
  - ⛔ EXCLU : imparfait/plus-que-parfait/conditionnel comme temps travaillés ; subordination.

### Module 2 — Thème : **Ville et campagne** — p. (après M1)

- **Lecture** : textes descriptifs/narratifs ville↔campagne ; phrase minimale à relever ; titres.
- **Conjugaison** : **le futur simple** (réinvestissement : « écris les verbes au futur », corrigé :
  _mettrons, se dirigeront, s'ouvrira, s'installera, interviendrons…_).
- **Syntaxe / déterminants** : **L'expansion du groupe nominal** — GN minimal (nom propre / pronom /
  déterminant + nom) vs **GN étendu** (déterminant + **adjectif épithète** + nom) ; _grand_ = expansion,
  adjectif épithète ; **adjectif attribut vs épithète** (souligner l'attribut, entourer l'épithète).
- **Orthographe** : **homophones `et / est / es`, `on / ont`** (« _on_ = pronom personnel sujet d'un
  verbe à la 3ᵉ pers. ») ; compléter un texte avec « et »/« est ».
- **Production insérée** : récit/description ; insertion d'un passage descriptif.
- **Bornes de scope** :
  - ✅ INCLUS : futur ; expansion du GN par adjectif épithète ; épithète vs attribut ; homophones
    `et/est/es`, `on/ont`.
  - ⛔ EXCLU : degrés de l'adjectif (comparatif/superlatif) si non explicités.

### Module 3 — Thème : **Nos amis les animaux** — p. (après M2)

- **Lecture** : textes sur les animaux ; relever un GN étendu selon le type d'expansion ; synonymes.
- **Conjugaison** : futur / passé composé / **impératif** (réinvestissement : « écris au futur »,
  « au passé composé », « à l'impératif à la 2ᵉ personne du singulier »).
- **Syntaxe — Les déterminants** : « les déterminants (**articles** – **adjectifs**) apportent des
  précisions sur le **genre et le nombre** du nom » ; **articles** (définis, indéfinis, **définis
  contractés**), **adjectif/déterminant démonstratif**, **déterminant indéfini**, **déterminant
  exclamatif**, **adjectif numéral** ; relever/classer les déterminants ; forme négative (« l'article
  _des_ devient _de_ à la forme négative »).
- **Syntaxe — Expansion du GN (suite)** : relever un GN étendu dont l'expansion est **un adjectif
  épithète**, **un complément de nom**, **une proposition relative** ; remplacer une expansion (prop.
  relative) ; **compléments circonstanciels** (dont **complément circ. de cause** : « relie la phrase à
  son complément de cause »).
- **Orthographe** : **les déterminants `sa / son / ses / leur / leurs`** (homophones / accord).
- **Bornes de scope** :
  - ✅ INCLUS : système des déterminants (articles + démonstratifs/possessifs/indéfinis/exclamatifs/
    numéraux, article défini contracté) ; expansion du GN (adj. épithète, complément de nom, prop.
    relative — **repérage**) ; compléments circonstanciels (dont cause) ; `sa/son/ses/leur/leurs`.
  - ⛔ EXCLU : analyse formelle complète de la proposition relative (pronoms relatifs composés, emboîtements)
    — ici **repérage/substitution** seulement.

### Module 4 — Thème : **Les secrets de la nature** — p. (après M3)

- **Lecture** : textes documentaires/narratifs sur la nature ; **article de journal** (« Ce texte est…
  un article de journal », « D'après l'article paru dans France-Antilles ») ; **fiche de lecture** ;
  cyclone (conséquences) ; sentiments du lecteur.
- **Conjugaison / Orthographe** : passé composé / futur / présent / impératif (transformations de temps
  sur un même texte : « écris au futur », « au passé composé », « à l'impératif »).
- **Syntaxe** : compléments circonstanciels (compléter chaque phrase par un complément circ.) ;
  pronoms (remplacer un pronom sujet par un GNS).
- **Orthographe** : graphies de sons (ex. « ran**g**e / navi**g**ation ») ; **homophones avec `la / l'a /
l'as`** (« _la_ devant un verbe… » ; accord du PP avec _avoir_ + COD antéposé : « Elle l'a récitée sa
  leçon »).
- **Vocabulaire** : champs lexicaux thématiques (mer / campagne / ville).
- **Bornes de scope** :
  - ✅ INCLUS : transformations de temps (présent/PC/futur/impératif) ; compléments circonstanciels ;
    pronoms/substituts ; homophones `la/l'a/l'as` ; **accord du PP avec _avoir_ (COD antéposé)** apparaît
    ici (≠ 6ème, où il était exclu).
  - ⛔ EXCLU : typologie exhaustive des subordonnées.

### Module 5 — Thème : **Histoires réelles – Histoires imaginaires** — p. (après M4)

- **Lecture** : récits réels vs imaginaires ; **Le conte merveilleux** (encadré) — structure du conte
  (roi, fée, sort…) ; numéroter les parties d'un texte ; imaginer une fin.
- **Conjugaison** : futur simple (« complète avec le pronom qui convient et conjugue au futur ») ;
  réinvestissement des temps.
- **Syntaxe** : substituts / reprises pronominales (« Le renard choisit un poulet. Il emporte le
  poulet. » → pronominaliser) ; pronoms personnels compléments.
- **Orthographe** : **homophones `la / l'a` + accord du PP** (« Le maître l'a interrogé », « Elle l'a
  bien récitée sa leçon ») ; pronom personnel COD + accord.
- **Vocabulaire** (en fin de cahier) : **Les synonymes** (« mots de sens proches »), **Les mots
  contraires** (« sens opposé »), **Les mots de la même famille**, **Les différents sens d'un mot**
  (polysémie : « terre » = sol / monde / planète / matière ; « pièce », « vague »…).
- **Production insérée** : rédiger un récit ; insérer dialogue/passage descriptif ; présenter un
  personnage à ses camarades.
- **Bornes de scope** :
  - ✅ INCLUS : structure du **conte merveilleux** ; substituts & pronominalisation ; homophones `la/l'a`
    - accord du PP ; synonymes / contraires / familles de mots / polysémie.
  - ⛔ EXCLU : schéma narratif théorisé (situation initiale/élément perturbateur/…) si non nommé tel quel
    dans le cahier — ici approche pratique (numéroter les parties, imaginer une fin).

## 6. Incertitudes / à revérifier

- **🔴 Nature du document (réserve majeure)** : `521702P00.pdf` est un **cahier d'exercices parascolaire
  avec corrigés** (« Le Français en 7ème année », L.M. Beya, « Assistante pédagogique »), **pas le guide
  enseignant / cadre de référence officiel du CNP**. Il se déclare seulement _« conforme aux programmes
  officiels 2006-2007 »_. **Conséquence** : §1 (compétences) et §3 (méthode) sont **reconstitués** à
  partir de la structure du cahier, faute de page officielle. **Action recommandée** : localiser et
  transcrire le **véritable guide CNP 7ème** pour valider/compléter compétences terminales, progression
  trimestrielle et critères d'évaluation, puis recouper les **bornes de scope** du §2.
- **Ampleur réelle du guide** : le fichier est **petit en octets** (~447 Ko) **mais compte 95 pages
  imprimées** (texte vectoriel léger, pas un scan) — ce n'est donc **pas** un « programme bref / sommaire »
  comme l'avertissement le supposait, mais un **cahier d'exercices complet** (≈ 90 p. d'exercices + leur
  section corrigés). La couche-texte est **propre** (pdftotext exploitable), ce qui a permis une
  transcription du contenu **sans dépendre uniquement de la vision**.
- **Faits de langue = relevé du cahier, pas liste officielle** : la liste des faits de langue (§2) est
  **exhaustive du cahier** (tous les encadrés-leçon ont été relevés), mais le **programme officiel**
  pourrait en **lister d'autres** (non traités par ce parascolaire) ou **ordonner différemment**. À
  valider sur le guide CNP.
- **Pagination de début des modules 1–5** : non alignée précisément (pas de sommaire paginé ; offset
  pieds-de-page non calé). Seules p.3–5 (prérequis) sont sûres. Colonne `pages` du §4 partielle.
- **Titres de quelques supports de lecture** : relevés via couche-texte (fiables pour les **titres de
  modules** et **encadrés-leçon**) ; certains titres de textes internes sont fragmentés par la mise en
  page multi-colonnes (sans impact sur le scope de langue).
- **Pas de « projet d'écriture » ni d'évaluation formalisés** : le cahier intègre la production **dans
  les exercices** (« Fais parler… », « Raconte… », « Rédige… ») mais ne formalise ni projet d'écriture ni
  grille de critères ni performances attendues — à chercher dans le guide officiel.
