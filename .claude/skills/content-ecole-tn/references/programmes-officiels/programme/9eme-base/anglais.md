## 1. Cadre & compétences (Introduction p.4 + Methodology p.6–8)

> Comme les guides 7ème/8ème, ce document n'est **pas** un recueil d'« Answer Keys » : c'est un **document méthodologique** (Introduction + Methodology, p.4–8) **+ des « Module maps / lesson descriptions / teaching hints »** (p.9–44). Le **scope linguistique officiel** se lit dans les **cartes de module** (une par module), qui donnent **par leçon** des colonnes explicites **Functions / Grammar / Lexis / Stress & pronunciation / Project work**, puis dans les **fiches-leçon** (Objective / Vocabulary / Language = Structures + Functions / Procedure). C'est la source de scope de référence ci-dessous (§2). **Contrairement à la 8ème** (« Proceed with English » **tabule** Grammar/Functions/Lexis/Pronunciation par leçon dans la carte de module).

### 1.1 Nature du guide (Introduction p.4)

- « This document is a guide for the teacher to make the best use of the materials which they are expected to use so as to cover the **official programme**. » (p.4)
- Le guide : présente le teaching–learning material conçu pour la classe ; éclaire la **méthodologie et les principes** de conception ; donne pour la **4ème partie** (= module maps) une **description séquencée** des leçons + des **teaching hints** ; propose une **suggestion** d'agenda annuel (« merely intended to help teachers and should in no way enslave them », p.4).

### 1.2 Le matériel & les 6 modules (« The package of material », p.5)

- **Composants** (p.5) : **a)** The Student's Book ; **b)** The Audio-Cassette or CD (texts designed for the listening activities) ; **c)** The Activity Book (recycles, in pairs/groups, the language acquired in the Student's Book) ; **d)** The Teacher's Book.
- **Le Student's Book = 6 modules** (p.5, **liste officielle imprimée**) :
  1. **FAMILY LIFE**
  2. **EDUCATION**
  3. **HEALTH AND ENVIRONMENT**
  4. **SERVICES**
  5. **ENTERTAINMENT**
  6. **CIVILITY**
- Le Student's Book est « composed of an introductory lesson + **six modules** (Reading, Listening, Speaking or Writing) », travaillant les **quatre macro-skills** ; il « will help the Tunisian **9th form learners carry on and progress** … in reference to the official programmes of English for Basic Education » (p.5).

### 1.3 Principes pédagogiques (« Methodology · Guiding principles », p.6)

- **1- The learner is an intelligent human being** (p.6) : on bâtit sur ses attitudes, sa connaissance antérieure, ses activités pour donner du sens à l'apprentissage.
- **2- The learner is at the core of the learning process** (p.6) : l'apprentissage est **constructif** (constructivisme) — l'enseignant régule, l'apprenant **construit** ses savoirs ; le savoir-faire est privilégié.
- **3- The learner's participation and interaction are important in learning the language** (p.6) : interaction et **group work** ; appui sur les différences (skills, backgrounds) pour la coopération.
- **4- Contextuality in learning** (p.6) : « we have made it possible for the learners to discover the language and the texts work … the learner **understands the system and retain its rules** » — la **grammaire est apprise EN CONTEXTE** (par découverte des textes), **pas** par règles isolées.

### 1.4 Architecture interne d'une leçon (« How are the textbooks organised? », p.7–8)

Le Student's Book est en **trois parties** par leçon (p.7) :

- **1- Get Ready** (p.7) : la leçon est introduite par des images/visuals ; phase de mise en route — visuals « deeply believe that visuals help trigger the learner's imagination and make it easier for the teacher … to start with a brainstorming activity », et **« anticipate »** ce que la leçon va couvrir.
- **2- Learn About** = la partie centrale, avec :
  - **Reading** (p.7) : un texte avec une **tâche de lecture** (gist / specific information / inference) ; **« More examples … to locate it in the passage »** ; expression à partir du texte.
  - **Language** (p.7) : « **The Language Box provides … to further enrich the topic** » — pratique de grammaire et de vocabulaire **en lien avec le sujet** de la leçon.
  - **Pronunciation** (p.7) : « short exercises that deal with sounds, spelling or stress » (sons / orthographe / accentuation).
- **4- Writing and/or Project Work** (p.8) : tâches **découpées en étapes** (steps) autour d'un même sujet, mobilisant plusieurs skills + coopération → **produit tangible** servant à l'évaluation.
- **5- The Lighter Side and Grandma's Corner** (p.8) : pause-détente après l'effort des tâches (humour / culture).
- **Envoi** (p.8, « The authors ») : Student's Book + Activity Book + Teacher's Book ensemble pour améliorer la performance dans la **langue cible**.

> **Ossature « compétences » à respecter pour générer** : la 9ème est l'**année de concours / fin de collège (≈ B1)**. Les **4 skills** (Reading / Listening / Speaking / Writing) + **Pronunciation** sont travaillées par **objectifs communicatifs** (colonne _Functions_ de chaque leçon) au sein de **6 modules thématiques** (Family Life → Education → Health & Environment → Services → Entertainment → Civility). La **grammaire est enseignée en contexte** (Get Ready → Reading → Language Box → Writing/Project) ; le **métalangage n'est pas l'objet d'étude**. Notation **standard** : chiffres latins, LTR, unités SI ; aucun chiffre arabo-indien.

## 2. Plan annuel (les 6 modules / 30 leçons — « Module maps », p.9–44)

> Le programme est organisé en **6 modules**, chacun de **5 leçons** = **30 leçons**. Pour **chaque module**, le guide donne une **carte de module** (grille à 9 colonnes, transcrite intégralement ci-dessous) **puis** une **fiche par leçon** (Objective / Main skills / Subskills / Vocabulary / Language [Structures + Functions] / Procedure). Les colonnes **Functions · Grammar · Lexis · Stress & pronunciation** de la carte = le **scope linguistique officiel** de la 9ème, **par leçon**. #### Module 1 — **FAMILY LIFE** (carte p.9 ; fiches p.10–14)

- **Lesson 1 — Family relationships** (p.10). **Objective** : to enable students to talk about family relationships and realise that family life is not always a happy one. **Skills** : Reading, Writing. **Sub-skills** : read for the gist ; scan a text for specific information ; transfer information onto a table ; produce written descriptions from prompts ; listing and writing notes. **Functions** : 1- Asking questions ; 2- Describing 1. **Grammar** : 1- Question words ; 2- Long / short adjectives. **Lexis** : magic, supportive, severe, easy-going, wisdom, fighting, miserable, orphanage, to afford. **Pronunciation** : /æ/, /ɑː/. **Project** : Project work Step 1 ; Writing one.
- **Lesson 2 — Sharing family responsibilities** (p.11). **Objective** : to listen to a passage in order to compare different types of family roles and share opinions about them. **Skills** : Listening, Reading, Writing. **Sub-skills** : distinguishing between fact and opinion ; drawing on background knowledge to understand text ; listen to infer emotional state. **Functions** : 1- Comparing 1 ; 2- Describing 2. **Grammar** : 1- Comparatives ; 2- The superlative. **Lexis** : to cook, to feed, to rock, role, close, to bring up, exciting, demanding, to look after. **Pronunciation** : Spelling 1 ; short competitions. **Project** : Project work Step 2 ; Writing two.
- **Lesson 3 — The Generation Gap** (p.12). **Objective** : to enable students to read about and react to family issues. **Skills** : Reading, Writing. **Sub-skills** : infer relationships between interlocutors ; predicting possible outcomes ; producing a text from various sources of information. **Functions** : Comparing 2. **Grammar** : Compound adjectives. **Lexis** : to agree, to argue, embarrassed, to break (rules), dog-headed, to sneak (out). **Pronunciation** : main "s" and "es". **Project** : Project work Step 3 ; Writing three. _(Procédure p.12 : « a good way to **revise the future tense** » + thème des conflits parents-enfants / fashion.)_
- **Lesson 4 — Pocket money** (p.13). **Objective** : to enable students to read a passage about pocket money in order to discuss its importance in family life and to write a short paragraph about it. **Skills** : Speaking, Writing. **Sub-skills** : write to recycle structures covered in class ; free writing leading to fluency. **Functions** : Comparing 3. **Grammar** : As + Adjective + As. **Lexis** : stationeries, savings, extras, to purchase, overspending, an addiction, hard-earned. **Pronunciation** : Word stress 1. **Project** : Writing four.
- **Lesson 5 — Safety at home** (p.14). **Objective** : to have students read about, discuss safety at home and use appropriate (language) to write about this topic. **Skills** : Reading, Writing. **Sub-skills** : information transfer ; scanning for specific information ; reinventing previously acquired (language). **Functions** : Contrasting. **Grammar** : While, Whereas. **Lexis** : to bite, owner, chained, isolation, to occur, hazard, infected, to supervise. **Pronunciation** : Spelling 1 ; new plurals (Y → ...es). **Project** : Project work Step 4 ; Presentation.

#### Module 2 — **EDUCATION** (carte p.15 ; fiches p.16–20)

- **Lesson 1 — School memories** (p.16). **Objective** : to have pupils recycle (language) related to school life in order to prepare them for the comprehension of a reading passage about the same topic and subsequently start a project dealing with school problems. **Skills** : Reading, Writing. **Sub-skills** : read for the gist ; scan for specific information ; make lists. **Functions** : Expressing possession. **Grammar** : 1- The genitive ; 2- Possessive pronouns. **Lexis** : memory, preparatory, mark, grade, to notice, to erase, test, instead, proud. **Pronunciation** : /u/, /ʌ/ (the letter "u"). **Project** : Project work Step 1 ; Writing one.

- **Lesson 3 — First day at school** (p.18). **Skills** : Reading, Writing. **Sub-skills** : recognising patterns and devices ; information acquired in one's writing. **Functions** : to express a completed action in the past. **Grammar** : Simple past ; irregular verbs ; past participles. **Lexis** : fun, to pretend, shake hands, to forget, to melt. **Pronunciation** : Word stress 3 ; nouns & adj. **Project** : Project work Step 2 ; Writing three.
- **Lesson 4 — Violence at school** (p.19). **Skills** : Speaking, Writing. **Sub-skills** : working in groups and sharing information to perform an oral task ; guided writing. **Functions** : Obligation. **Grammar** : have to / must (+ how to ; prepositions of location / town + N). **Lexis** : to believe, to use, to move, accent, to haunt, mad, to participate. **Pronunciation** : /e/ /æ/. **Project** : Writing four.
- **Lesson 5 — School life** (p.20). **Skills** : Reading, Writing. **Sub-skills** : skimming and scanning for information ; sharing information to perform an oral task and solve a problem. **Functions** : Expressing intention. **Grammar** : The future — 1- I intend to ; 2- I am going to + base form (… ). **Lexis** : to get on, expense, expensive, to realise, to dread, sensitive, to rest. **Pronunciation** : Spelling 2 ; double consonants. **Project** : Project work Step 3 ; Presentation.

#### Module 3 — **HEALTH AND ENVIRONMENT** (« A Healthy Environment for a Better Life ») (carte p.21 ; fiches p.22–26)

#### Module 6 — **CIVILITY** (carte p.39 ; fiches p.40–44)

## 3. Notes pédagogiques / méthode

### 3.1 Démarche générale (Methodology p.6–8)

Approche **communicative** centrée sur l'apprenant (« the learner is at the core »), **constructiviste**, **apprentissage en contexte** (« contextuality in learning » : l'apprenant **découvre** le système et **en retient les règles**, p.6), **interaction & group work**, **project work** par module, **process writing**. Le matériel travaille les **4 macro-skills** (Reading / Listening / Speaking / Writing) + **Pronunciation**. Les **visuals** (« Get Ready ») déclenchent le brainstorming et anticipent la leçon (p.7). **The Lighter Side / Grandma's Corner** = respiration culturelle/humoristique (p.8).

### 3.2 Architecture par leçon (« Module maps », p.9–44)

L'unité d'enseignement est la **leçon** (5 par module). Pour chaque module, une **carte** récapitule, **par leçon**, _Skills · Sub-skills · Functions · Grammar · Lexis · Stress & pronunciation · Project_. Chaque leçon, en détail, comprend **Objective / Main skills / Subskills / Vocabulary / Language (Structures + Functions) / Procedure**. La leçon se déroule en **Get Ready → Learn About (Reading + Language Box + Pronunciation) → Writing/Project** (p.7–8). Chaque module porte **un Project Work en 3–4 étapes** (Step 1 … → Presentation, dernière colonne de la carte).

### 3.3 Évaluation & projet

- **Project Work** par module (group work, produit tangible servant à l'évaluation, p.8), réparti en **Steps** au fil des 5 leçons et **présenté** (Presentation) à la dernière leçon.
- **Activity Book** : recycle/consolide en pairs/groups la langue acquise dans le Student's Book (p.5).
- **Année de concours** : la 9ème prépare l'examen national de fin de Basic Education (ختم التعليم الأساسي) — accent mis sur la **réutilisation fidèle** des structures couvertes (« write to recycle structures covered in class », récurrent dans les Sub-skills).

### 3.4 Prononciation & notation standard

- **Prononciation tabulée par leçon** (colonne _Stress & pronunciation_) : sons (IPA), **spelling**, **stress**, **mute consonants**, **lettres** — voir §2. La 9ème **tabule** explicitement ces points (≠ certaines années où la prononciation est seulement contextuelle).
- **Notation standard** (règle matière, `content-engine/references/math-and-notation.md`) : **chiffres latins 0–9**, équations/textes **LTR**, unités SI ; dates et nombres en **chiffres latins**. Aucun chiffre arabo-indien.
