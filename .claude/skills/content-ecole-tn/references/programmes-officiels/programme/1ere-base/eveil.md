## 1. Cadre & compétences (الكفايات / الاقتدارات / المؤشرات)

Le guide pose (p.11) le triptyque **الكفايات → الاقتدارات → المؤشرات** ciblé pour la 1ère année.

- **الكفاية النهائية** (compétence terminale, p.11/13) : **« حلّ وضعيات مشكل دالّة »** — résoudre des situations-problèmes signifiantes.
- **الكفاية الفرعية 1** (p.11/13) : **« حلّ وضعيات مشكل بإجراء تجارب ومشاريع متّصلة ببعض الظّواهر الفيزيائيّة »** — par des expériences/projets liés à **quelques phénomènes physiques**.
- **الكفاية الفرعية 2** (p.11/13) : **« حلّ وضعيات مشكل بإجراء تجارب ومشاريع متّصلة بالوظائف الحيويّة للكائنات الحيّة في علاقتها بالمحيط »** — par des expériences/projets liés aux **fonctions vitales des êtres vivants** dans leur rapport au milieu.

**الاقتدارات ↔ المؤشرات** (p.13) :

| الاقتدار (capacité)                       | المؤشرات (indicateurs, extraits fidèles)                                                                                                                                                                            |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| توظيف الممارسة العمليّة في تعرّف الظّواهر | إنجاز أعمال عمليّة متعدّدة : القيس، التركيب، التصنيف، استخدام الحواسّ… ؛ ابتكار وسائل لإنجاز عمل (الحوار، المحاولات، استنباط وسائل عمل جديدة…)                                                                      |
| توظيف المفاهيم المكتسبة في إيجاد حلول     | تحديد معطيات المشكل بالتمييز بينها (تنوّع أشكال التعبير : الإشارة – الرسم – التمثيل – المحاكاة – التخطيط…) ؛ انتقاء لغة علميّة سليمة ؛ تقديم حلول ممكنة ؛ البحث عن علاقات في إعداد حلول (ربط علاقة بين مفهوم وآخر…) |

**Démarches recommandées** (p.5–6) :

- **المقاربة بالوضعيّات الإشكاليّة** : الوضعيّة الإشكاليّة → الافتراضات → الممارسة العمليّة والتّجريب → عرض النتائج واكتشاف المفهوم → الاستنتاج → التطبيق (توظيف المفهوم) → التقييم.
- **المقاربة بالمشروع البيداغوجي** : إشكاليّة المشروع → الإعداد والتخطيط → تنفيذ المشروع → استثمار المشروع → تقييم المشروع.

## 2. Plan annuel (المخطّط السّنوي)

> Présentation (p.16) : le المخطّط السّنوي comprend **المحاور · الأهداف المميّزة · المفاهيم · المستلزمات الديداكتيكيّة**, et répartit l'année en **3 فترات زمنيّة** (le guide insiste sur la liberté de l'enseignant dans le séquençage selon la spécificité de sa classe). Le programme s'organise en **3 مجالات / 9 محاور**. Chaque مجال se clôt par une **وضعيّة إدماجيّة** + تقييم.

### مجال / Domaine : **جسم الإنسان** (le corps humain) — p.17

#### محور / Thème 1 : **الحواسّ الخمس** (les cinq sens)

- **Objectifs (الأهداف المميّزة)** : يتعرّف الحواسّ الخمس ؛ يربط كلّ حاسّة بوظيفتها ؛ يطبّق قواعد حفظ الصّحّة المرتبطة بالحواسّ.
- **Concepts (المفاهيم)** : العين والبصر · الأذن والسّمع · الأنف والشّمّ · اللّسان والذّوق · الجلد واللّمس · قواعد حفظ الصّحّة (المرتبطة بالحواسّ).
- **Exemples / المستلزمات (du guide)** : أدوات ومواد للإحساس (مواد ذات روائح، أشياء بملمس مختلف، مصادر أصوات…)، صور، آلة تسجيل.
- **Bornes de scope** :
  - ✅ INCLUS : les 5 sens, leur fonction, l'hygiène liée aux sens.
  - ⛔ EXCLU : anatomie interne détaillée des organes (relève des niveaux supérieurs).

### مجال / Domaine : **علم الأحياء** (sciences du vivant) — p.18–20

#### محور / Thème 2 : **التغذية** (l'alimentation) — p.18

- **Objectifs** : يتعرّف أصناف الأغذية ؛ يتعرّف الأغذية التي يحتاجها الإنسان ؛ يطبّق قواعد حفظ الصّحّة الغذائيّة.
- **Concepts** : أصناف الأغذية · التغذية عند الإنسان · قواعد الصّحّة الغذائيّة.

#### محور / Thème 3 : **النموّ** (la croissance) — p.19

- **Objectifs** : يتعرّف مظاهر النموّ عند الإنسان وعند الحيوان وعند النبات.
- **Concepts** : نموّ الإنسان · نموّ الحيوان · نموّ النبات.

#### محور / Thème 4 : **تنقّل الحيوانات** (la locomotion animale) — p.20

- **Objectifs** : يصنّف الحيوانات حسب أنماط تنقّلها.
- **Concepts** : أنماط تنقّل الحيوانات (المشي · الطيران · السّباحة · الزّحف…).

#### محور / Thème 5 : **التنفّس** (la respiration) — p.20

- **Objectifs** : يتعرّف تنفّس الحيوانات (والإنسان) ؛ يربط الحيوان بوسطه.
- **Concepts** : التنفّس · الحيوان والوسط (هواء / ماء).
- **وضعيّة إدماجيّة** (fin du مجال علم الأحياء).

### مجال / Domaine : **الفيزياء** (la physique) — p.21–23

#### محور / Thème 6 : **التموقّع في الفضاء** (le repérage dans l'espace) — p.21

- **Objectifs** : تعيين موقع جسم في الفضاء بالنسبة إلى جسم آخر وبالنسبة إلى ذاته ؛ مقارنة الأجسام حسب الطّول.
- **Concepts** : فوق / تحت · أمام / وراء · يمين / يسار · قريب / بعيد · أكبر / أصغر · أطول / أقصر.

#### محور / Thème 7 : **الزّمن** (le temps) — p.22

- **Objectifs** : ذكر أيّام الأسبوع وترتيبها ؛ إدراك تداول الليل والنّهار.
- **Concepts** : الأسبوع · أيّام الأسبوع · الليل والنّهار · التوقيت.

#### محور / Thème 8 : **المادّة** (la matière) — p.23

- **Objectifs** : تمييز موادّ الصّنع ؛ تصنيف المواد.
- **Concepts** : المادّة · الصّلب / السّائل · موادّ الصّنع.

#### محور / Thème 9 : **القوّة (الدفع والجذب)** (la force : pousser / tirer) — p.22

- **Objectifs** : إدراك أنّ الجسم يتحرّك بدفعه أو جذبه.
- **Concepts** : الدفع · الجذب · الحركة.
- **وضعيّة إدماجيّة** (fin du مجال الفيزياء).

## 3. Notes pédagogiques / méthode

- **Public ~6 ans, majoritairement non-lecteurs** → démarche **active et très visuelle** : observation, manipulation, tri, comparaison, expérimentation ; l'écrit est minimal (le guide enseignant fait foi, il n'existe **pas de manuel élève** d'éveil avant la 3ème année).
- Le programme est **piloté par les compétences** : chaque محور sert la résolution de situations-problèmes (الكفاية النهائية), pas la mémorisation de définitions.
- **3 فترات** sur l'année ; chaque مجال se clôt par une **وضعيّة إدماجيّة** + تقييم.
- **Notation** : chiffres latins (0–9), pas de symbolique mathématique ; vocabulaire scientifique simple et juste.
- Démarche-type d'une séance : situation-problème → hypothèses → expérience/observation → conceptualisation → application → évaluation (cf. §1).
