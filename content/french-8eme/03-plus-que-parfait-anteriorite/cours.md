# ⚔️ Le plus-que-parfait et l'antériorité — sur les routes du voyage

> 💡 « Pour raconter un voyage, il ne suffit pas de dire ce qui s'est passé : il faut aussi savoir dire ce qui s'était passé **avant**. »

Nour prépare son carnet de voyage. Elle veut raconter la traversée du désert que sa grand-mère avait faite bien avant sa naissance. Mais très vite, un problème surgit : dans son récit au passé, certaines actions se sont produites **avant** les autres. Comment marquer ce passé « plus ancien » que le passé ? La réponse est une arme de conjugaison que tu débloques dans cette quête : le **plus-que-parfait**.

## 🗺️ À quoi sert le plus-que-parfait ?

Le plus-que-parfait est le **passé du passé**. Il exprime l'**antériorité** : une action **déjà accomplie avant** une autre action passée.

Regarde ces deux actions dans le récit de Nour :

> _Quand la caravane **arriva** à l'oasis, le soleil **s'était** déjà **couché**._

Deux moments passés, mais pas au même niveau :

1. le soleil se couche → **d'abord** (action la plus ancienne) → **plus-que-parfait** : _s'était couché_
2. la caravane arrive → **ensuite** → passé simple : _arriva_

> 🗡️ Le plus-que-parfait pose le **décor déjà accompli**. Les mots _déjà, avant, la veille, auparavant, quand / lorsque_ annoncent souvent une action au plus-que-parfait.

## ⚙️ Comment former le plus-que-parfait ?

Le plus-que-parfait est un **temps composé** : il se forme en **deux morceaux**.

**être** ou **avoir** à l'**imparfait** + participe passé

Tout se joue sur l'auxiliaire : il est **à l'imparfait**. C'est ce qui distingue le plus-que-parfait du passé composé.

Rappel des auxiliaires à l'imparfait :

| avoir (imparfait)     | être (imparfait)      |
| --------------------- | --------------------- |
| j'**avais**           | j'**étais**           |
| tu **avais**          | tu **étais**          |
| il/elle **avait**     | il/elle **était**     |
| nous **avions**       | nous **étions**       |
| vous **aviez**        | vous **étiez**        |
| ils/elles **avaient** | ils/elles **étaient** |

_Trois exemples formés :_

- _préparer_ → j'**avais préparé** mon sac.
- _partir_ → le guide **était parti** avant l'aube.
- _voir_ → nous **avions vu** les dunes de loin.

> ⚠️ Le choix de l'auxiliaire (**être** ou **avoir**) obéit aux **mêmes règles** que le passé composé : les verbes de mouvement (_aller, venir, arriver, partir, entrer, sortir, monter, descendre, rester, tomber_) et les verbes pronominaux prennent **être** ; les autres prennent **avoir**.

## 🎯 Plus-que-parfait ou passé composé ? Ne les confonds pas !

C'est le piège numéro un. Les deux temps sont composés d'un auxiliaire + un participe passé. **Un seul détail les sépare : le temps de l'auxiliaire.**

| Temps            | Auxiliaire        | Exemple                                    |
| ---------------- | ----------------- | ------------------------------------------ |
| Passé composé    | **au présent**    | il **a** mangé · elle **est** partie       |
| Plus-que-parfait | **à l'imparfait** | il **avait** mangé · elle **était** partie |

> 🗡️ Le test infaillible : regarde l'auxiliaire. _a / est_ → passé composé. _avait / était_ → plus-que-parfait. Le participe passé, lui, ne change pas (_mangé_, _partie_) ; il ne t'aide donc pas à trancher.

## 🛡️ L'accord du participe passé

Le plus-que-parfait suit **exactement** les règles d'accord du passé composé.

- **Avec l'auxiliaire _être_**, le participe passé **s'accorde avec le sujet** (genre et nombre) :
  > _Les exploratrices **étaient parties** à l'aube._ (parti**es** : féminin pluriel, comme « les exploratrices »)
- **Avec l'auxiliaire _avoir_**, le participe passé **ne s'accorde pas avec le sujet**. Il s'accorde seulement avec le **COD placé avant** le verbe :
  > _La piste \**qu'\**il **avait suivie** menait à l'oasis._ (« qu' » reprend « la piste », féminin singulier, placé avant → suivi**e**)
  > Mais : _Il **avait suivi** la piste._ (COD « la piste » placé **après** → pas d'accord)

> ⚠️ Piège classique : accorder le participe avec le sujet quand l'auxiliaire est _avoir_. On n'écrit **pas** « les voyageurs avaient partis » ni « elle avait préparés ses bagages ».

## 📜 Le plus-que-parfait dans le récit

Dans un récit au passé, les temps travaillent en équipe :

- l'**imparfait** peint le décor et les actions qui durent ;
- le **passé simple** (ou le passé composé) fait avancer l'action de premier plan ;
- le **plus-que-parfait** revient en arrière pour dire ce qui **s'était déjà passé avant**.

> _Nour ouvrit le vieux carnet. Sa grand-mère y **avait noté** chaque étape du voyage qu'elle **avait fait** trente ans plus tôt. Les pages **étaient devenues** jaunes._

Ici, _ouvrit_ est le premier plan (passé simple) ; _avait noté_, _avait fait_, _étaient devenues_ renvoient toutes à un passé **antérieur**, plus ancien.

> 🏰 Retiens l'ordre du temps : **plus-que-parfait → imparfait / passé simple**. Ce qui est au plus-que-parfait s'est passé **en premier**, même si la phrase le cite en second.

## 🧭 Récapitulatif de la quête

1. Le plus-que-parfait = le **passé du passé** : il marque l'**antériorité**.
2. Formation : **être ou avoir à l'imparfait** (_avait_, _était_…) **+ participe passé**.
3. Il se distingue du passé composé **uniquement** par l'auxiliaire (imparfait, pas présent).
4. Accord : avec _être_ → avec le **sujet** ; avec _avoir_ → avec le **COD placé avant**.
5. Dans le récit : ce qui est au plus-que-parfait s'est passé **avant** le reste.
