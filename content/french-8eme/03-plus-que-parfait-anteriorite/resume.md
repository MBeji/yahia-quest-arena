# 📜 Résumé — Le plus-que-parfait et l'antériorité

## À quoi il sert

Le **plus-que-parfait** est le **passé du passé**. Il exprime l'**antériorité** : une action **accomplie avant** une autre action passée.

> _Quand la caravane arriva, le soleil **s'était** déjà **couché**._
> → le soleil se couche **d'abord**, la caravane arrive **ensuite**.

Mots qui l'annoncent souvent : _déjà, avant, la veille, auparavant, quand, lorsque_.

## Comment le former

**Auxiliaire _être_ ou _avoir_ à l'IMPARFAIT + participe passé.**

- avoir → j'avais, tu avais, il avait, nous avions, vous aviez, ils avaient
- être → j'étais, tu étais, il était, nous étions, vous étiez, ils étaient

_Exemples :_ j'**avais préparé** · elle **était partie** · nous **avions vu**.

Même choix d'auxiliaire qu'au passé composé (verbes de mouvement et pronominaux → _être_).

## Ne pas confondre avec le passé composé

| Temps            | Auxiliaire        | Exemple                                    |
| ---------------- | ----------------- | ------------------------------------------ |
| Passé composé    | **au présent**    | il **a** mangé · elle **est** partie       |
| Plus-que-parfait | **à l'imparfait** | il **avait** mangé · elle **était** partie |

👉 Seul l'**auxiliaire** change. _a / est_ → passé composé. _avait / était_ → plus-que-parfait.

## L'accord du participe passé

- Avec **être** → accord avec le **sujet** : _elles étaient **parties**_.
- Avec **avoir** → accord seulement avec le **COD placé avant** : _la piste qu'il avait **suivie**_ (mais _il avait **suivi** la piste_).
- ⚠️ Jamais d'accord avec le sujet quand l'auxiliaire est _avoir_.

## Dans le récit

**plus-que-parfait → imparfait / passé simple.**
Ce qui est au plus-que-parfait s'est passé **en premier**, même s'il est cité après.
