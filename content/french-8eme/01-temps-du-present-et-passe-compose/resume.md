# 📜 Résumé : le présent et le passé composé (révision)

- **Présent — les 3 groupes** : 1er groupe _-e, -es, -e, -ons, -ez, -ent_ (je marche, jamais « je marches ») ; 2e groupe avec **-iss-** au pluriel (_ils réfléchissent_) ; 3e groupe radical variable (_je prends → ils prennent_).
- **Emplois du présent** : action en cours (_le train entre_), habitude (_chaque matin…_), vérité générale (_l'eau bout à 100 °C_), **présent de narration** (récit passé rendu vivant : _hier, la lampe s'éteint_), futur proche (_j'arrive dans deux minutes_).
- **Passé composé** = auxiliaire _être_ ou _avoir_ **au présent** + participe passé (**-é**, **-i**, **-u**, **-is**, **-it/-ert**). Piège **-é / -er** : après un auxiliaire → participe en **-é** (_a préparé_, jamais « a préparer » ; test : _a mordu_ ✓ / _a mordre_ ✗).
- **Choix de l'auxiliaire** : **être** pour les verbes de déplacement/changement (_aller, partir, arriver, rester, tomber…_) et les pronominaux. Faux amis avec **avoir** : _courir, marcher, nager_ (_il a couru_). Un verbe change d'auxiliaire selon le sens : _elle est descendue_ / _elle a descendu les valises_.
- **Accord du PP avec _être_** : accord avec le **sujet** (_Rania est partie_, _elles sont rentrées_, _Aziz et Nour sont arrivés_).
- **Accord du PP avec _avoir_** (nouveauté 8ème) : **jamais avec le sujet** ; accord avec le **COD seulement s'il est placé AVANT** le verbe (_les photos que Sofien a **prises**_, _il les a **écrites**_) ; COD après → **pas d'accord** (_Nada a **écrit** trois cartes_).
- **Verbes pronominaux** : auxiliaire **être**, accord avec le sujet (_Dorra s'est levée_, _les jumelles se sont retrouvées_). Ne pas confondre **s'est** (+ participe passé) et **c'est** (= « cela est »).
- **Dans le récit** : le **présent** peint l'habitude/le vivant, le **passé composé** rapporte le fait précis et achevé.
