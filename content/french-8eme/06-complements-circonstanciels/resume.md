# 🧭 Résumé — Les compléments circonstanciels

## Ce qu'est un CC

Le **complément circonstanciel (CC)** précise les **circonstances** de l'action. Il répond à quatre questions — les quatre boussoles :

| Question       | Type de CC        | Exemple                                    |
| -------------- | ----------------- | ------------------------------------------ |
| **Quand ?**    | CC de **temps**   | _Le car est parti **à l'aube**._           |
| **Où ?**       | CC de **lieu**    | _Ils se reposent **sous les palmiers**._   |
| **Comment ?**  | CC de **manière** | _Sofien avance **prudemment**._            |
| **Pourquoi ?** | CC de **cause**   | _La visite s'écourte **à cause du vent**._ |

## Le test des deux pouvoirs

Un CC est **supprimable** ET **déplaçable** :

- _Nadia raconte le voyage **le soir**._ → _Nadia raconte le voyage._ (supprimé ✅)
- _**Le soir**, Nadia raconte le voyage._ (déplacé ✅)

Formes possibles : un **adverbe** (_hier, ici, lentement_), un **groupe nominal** (_ce matin_) ou un **groupe prépositionnel** (_dans l'oasis, à cause de la chaleur_).

## CC ou COD / COI ?

Le **COD** (_quoi ?_) et le **COI** (_à qui ? à quoi ?_) sont **attachés au verbe** : on ne peut **ni les supprimer ni les déplacer**.

- _Nadia visite **la médina**._ → **COD** (visite quoi ?).
- _Le guide parle **aux élèves**._ → **COI** (parle à qui ?).
- _Nadia se promène **dans la médina**._ → **CC de lieu** (supprimable et mobile).

👉 D'abord la **question**, ensuite le **test des deux pouvoirs**.

## Cause : grâce à ≠ à cause de

- **grâce à** = cause **favorable** : _**Grâce à** son plan, Nadia a retrouvé le chemin._
- **à cause de** = cause **fâcheuse** : _**À cause du** retard, ils ont raté le train._

## Les connecteurs du récit

- **temps** : d'abord, ensuite, puis, enfin, soudain, le lendemain ;
- **lieu** : ici, là-bas, plus loin, à l'horizon ;
- **cause** : car, parce que, en effet, comme.

> 🏆 Boussoles en main : ajoute le **temps**, le **lieu**, la **manière**, la **cause**, et ne confonds jamais un CC avec un COD ou un COI.
