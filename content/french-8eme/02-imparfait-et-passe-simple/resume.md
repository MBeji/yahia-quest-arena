# 📜 Résumé : L'imparfait et le passé simple (les temps du récit)

- **Imparfait** = radical du **« nous » au présent** (_nous chantons → chant-_) + terminaisons **identiques pour tous** : **-ais, -ais, -ait, -ions, -iez, -aient** (_je chantais, il finissait, nous faisions_). Seul irrégulier : **être → j'étais**.
- **Valeurs de l'imparfait** : le **décor / l'arrière-plan** (_la ville dormait_), l'**action qui dure** (_il lisait_), l'**habitude / la répétition** (_chaque été, il partait_), le **portrait / l'état** (_il avait les mains rugueuses_). Indices : _chaque jour, souvent, autrefois, pendant que_.
- **Passé simple** = temps de l'**action ponctuelle, brève, achevée**, de **premier plan**. Terminaisons **selon le groupe** :
  - **1er groupe (-er) et _aller_** : -ai, -as, **-a**, -âmes, -âtes, **-èrent** (_il chanta, ils chantèrent_).
  - **2e groupe (finir) + beaucoup de 3e** : -is, -is, **-it**, -îmes, -îtes, **-irent** (_il finit, ils partirent, il prit, il vit, il fit_).
  - **3e groupe en -u** : -us, -us, **-ut**, -ûmes, -ûtes, -urent (_il voulut, il put, il eut, il fut_). Cas nasaux : _il vint, il tint_.
- ⚠️ **Ne pas confondre** : _il chant**ait**_ (imparfait) ≠ _il chant**a**_ (passé simple) ; _il finit_ peut être présent **ou** passé simple → c'est le **contexte** qui tranche.
- **Alternance dans le récit** : l'**imparfait** pose l'**arrière-plan** (décor, durée, habitude) ; le **passé simple** lance le **premier plan** (l'événement soudain qui fait avancer l'histoire). _Le soleil **se couchait** quand un cri **déchira** le silence._
- Test réflexe : **est-ce que ça dure / se répète (imparfait) ou est-ce que ça surgit une fois (passé simple) ?** _Soudain, il **tomba**_ ✓ (et non « il tombait ») ; _Il **faisait** beau_ ✓ (et non « il fit beau »).
