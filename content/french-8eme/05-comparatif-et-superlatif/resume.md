# 📜 Résumé : La comparaison — comparatif et superlatif

- **Comparatif** = comparer **deux** éléments. Trois degrés : **supériorité** (plus … que), **égalité** (aussi … que), **infériorité** (moins … que). Complément introduit par **que**.
- **Selon le mot comparé** :
  - adjectif / adverbe → **plus / aussi / moins … que** (_plus grand que_) ;
  - nom (quantité) → **plus de / autant de / moins de … que** (_plus de danseurs que_) ;
  - verbe (action) → **plus / autant / moins que** (_répète plus que_).
- Les trois « égalités » à ne pas confondre : **aussi** (adj./adv.), **autant de** (nom), **autant** (verbe).
- **Superlatif relatif** = le champion **d'un groupe** : **article défini (le/la/les) + plus / moins** ; complément introduit par **de** (_la plus applaudie **de** la troupe_). L'**article est obligatoire**.
- **Que vs de** : comparatif → **que** (_meilleur que son voisin_) ; superlatif → **de** (_le meilleur de la classe_).
- **Superlatif absolu** = très haut degré **sans comparaison** : **très, fort, extrêmement, si, tellement** + adjectif/adverbe (_très émouvant_). Ni « que » ni « de ».
- **Irréguliers** (à connaître par cœur) : bon → **meilleur** / le meilleur ; bien → **mieux** / le mieux ; mauvais → **pire** (ou plus mauvais) ; petit → **moindre**. ⚠️ « plus bon » et « plus bien » n'existent pas.
- **meilleur ou mieux ?** : **meilleur** remplace _bon_ → adjectif, avec un **nom**, s'accorde (meilleure, meilleurs). **mieux** remplace _bien_ → adverbe, avec un **verbe**, **invariable**.
