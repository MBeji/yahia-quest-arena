# 📜 Résumé — Le discours direct et le dialogue

**Discours direct** = rapporter les **paroles exactes** d'un personnage (mot pour mot). _(Transformer les paroles = discours indirect → réservé à la 9ème.)_

## Les trois signes de ponctuation

- **Deux-points ( : )** → **annoncent** la parole : _Il dit **:**_
- **Guillemets ( « » )** → **encadrent** la parole ; elle commence par une **majuscule** : _« Bonjour. »_
- **Tiret ( — )** → marque le **changement d'interlocuteur** (retour à la ligne). Les guillemets ouvrent au début et ferment à la fin du dialogue.

```
La mère demanda :
« As-tu faim ?
— Un peu, répondit Sami. »
```

## Les verbes introducteurs

Ils **présentent** la parole. On **varie** (au lieu de répéter « dire ») et on **respecte le ton** :

- question → **demander, interroger** · réponse → **répondre, répliquer**
- cri → **s'écrier, s'exclamer** · voix basse → **murmurer, chuchoter**
- ordre → **ordonner, lancer** · suite → **ajouter, reprendre**

⚠️ Le verbe colle au ton : un ordre ne se **chuchote** pas.

## La proposition incise

Verbe introducteur placé **après** ou **au milieu** de la parole :

- **Sujet inversé** : _« J'arrive », **dit-il**._ (pas « il dit »)
- **Ponctuation** : virgule avant l'incise (_« Je pars, **dit-elle**. »_) ; le **?** ou **!** reste sur la réplique (_« Où vas-tu ? **demanda-t-il**. »_) ; deux virgules si l'incise est **au milieu** (_« Ce soir, **dit-il**, nous partirons. »_).
- **« -t- »** si le verbe finit par une voyelle + **il / elle / on** : _demanda\**-t-\**elle_. Avec un **nom** sujet : pas de -t- (_demanda **la femme**_).

## Insérer le dialogue dans le récit

Le narrateur **raconte** (passé) → **cède la parole** (discours direct) → **reprend** le récit. Les verbes introducteurs sont les **charnières** entre narration et paroles.
