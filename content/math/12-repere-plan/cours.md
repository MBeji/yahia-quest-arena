# 🗺️ المعلّم في المستوي — خريطة الهندسة التحليلية

> 💡 «المعلّم هو نظام الإحداثيّات الذي يحوّل الهندسة إلى حساب — كلّ نقطة في المستوي لها عنوانها الخاصّ!»

## 🏰 المعلّم المتعامد والمتجانس

**المعلّم المتعامد والمتجانس** (يُرمز له بـ (O ; $\vec{i}$ ; $\vec{j}$)) يتكوّن من:

- **النقطة الأصل O**: نقطة تقاطع المحورين.
- **المحور الأوّل (محور الفاصلات)**: المستقيم الأفقي، اتّجاهه $\vec{i}$.
- **المحور الثاني (محور الترتيبات)**: المستقيم العمودي، اتّجاهه $\vec{j}$.
- **المتجانس**: كلّ وحدة على المحورين لها نفس الطول (المقياس موحَّد).

الخاصيّتان الجوهريّتان:

- المحوران عموديّان: زاوية 90° بين محور الفاصلات ومحور الترتيبات.
- الوحدة متساوية على كلا المحورين.

> 🗡️ «متعامد» = المحوران عموديّان؛ «متجانس» = نفس المقياس على المحورين. الفرق مهمّ لرسم الأشكال بدقّة.

## ⚡ إحداثيّات نقطة

كلّ نقطة M في المستوي تُعرَّف بزوج ترتيبي **(x ; y)**:

- **الفاصلة**: x = الإسقاط الأفقي على محور الفاصلات.
- **الترتيبة**: y = الإسقاط العمودي على محور الترتيبات.

نكتب: $M(x_M ; y_M)$.

| النقطة | الفاصلة | الترتيبة | الموضع             |
| ------ | ------- | -------- | ------------------ |
| O      | 0       | 0        | الأصل              |
| A      | 3       | 2        | الربع الأوّل (+,+) |
| B      | −4      | 1        | الربع الثاني (−,+) |
| C      | −2      | −3       | الربع الثالث (−,−) |
| D      | 5       | −1       | الربع الرابع (+,−) |
| E      | 0       | 4        | على محور الترتيبات |
| F      | −3      | 0        | على محور الفاصلات  |

إذا كانت الفاصلة = 0 أو الترتيبة = 0، فالنقطة تقع **على أحد المحورين** ولا تنتمي إلى أيّ ربع.

> ⚠️ الترتيب مهمّ: الفاصلة أوّلًا ثمّ الترتيبة. A(3 ; 2) ≠ A(2 ; 3).

## 🛡️ إحداثيّات شعاع من نقطتين

إذا كانت $A(x_A ; y_A)$ و $B(x_B ; y_B)$، فإنّ إحداثيّات الشعاع $\overrightarrow{AB}$ هي:

$$\boxed{\overrightarrow{AB}\begin{pmatrix} x_B - x_A \\ y_B - y_A \end{pmatrix}}$$

- المركّبة الأولى: $x_B - x_A$ (فاصلة الطرف − فاصلة المنشأ).
- المركّبة الثانية: $y_B - y_A$ (ترتيبة الطرف − ترتيبة المنشأ).

**مثال محلول:** $A(1 ; 3)$ و $B(5 ; -1)$:

$$\overrightarrow{AB}\begin{pmatrix} 5-1 \\ -1-3 \end{pmatrix} = \begin{pmatrix} 4 \\ -4 \end{pmatrix}$$

> 🗡️ حيلة: الشعاع يشير دائمًا من المنشأ (الحرف الأوّل) نحو الطرف (الحرف الثاني). «طرف ناقص منشأ».

## 🧭 تساوي شعاعين بالمركّبات — إيجاد الرأس الرابع

شعاعان متساويان إذا وفقط إذا كانت لهما **نفس المركّبتين**:

$$\overrightarrow{AB} = \overrightarrow{DC} \iff \begin{cases} x_B - x_A = x_C - x_D \\ y_B - y_A = y_C - y_D \end{cases}$$

**طريقة الرأس الرابع:** ABCD متوازي أضلاع $\iff$ $\overrightarrow{AB} = \overrightarrow{DC}$ — وهذا يكفي لإيجاد رأس مجهول.

**مثال محلول:** A(1 ; 2)، B(4 ; 3)، C(6 ; 7). أوجد D حتى يكون ABCD متوازي أضلاع.

- $\overrightarrow{AB}\begin{pmatrix} 4-1 \\ 3-2 \end{pmatrix} = \begin{pmatrix} 3 \\ 1 \end{pmatrix}$ و $\overrightarrow{DC}\begin{pmatrix} 6-x_D \\ 7-y_D \end{pmatrix}$.
- من $6 - x_D = 3$ نجد $x_D = 3$، ومن $7 - y_D = 1$ نجد $y_D = 6$. إذن D(3 ; 6).

> 🗡️ يكافئ ذلك الصيغة المباشرة: $x_D = x_A + x_C - x_B$ و $y_D = y_A + y_C - y_B$ (هنا: 1 + 6 − 4 = 3 و 2 + 7 − 3 = 6).

## 🌀 الانسحاب في المعلّم

إذا كانت للشعاع $\overrightarrow{u}$ المركّبتان (a ; b)، فإنّ صورة النقطة M(x ; y) بالانسحاب ذي الشعاع $\overrightarrow{u}$ هي:

$$M'(x + a \;;\; y + b)$$

**مثال محلول:** $\overrightarrow{u}(3 ; -2)$ و M(1 ; 4): الصورة هي M'(1 + 3 ; 4 − 2) = M'(4 ; 2).

**السابقة:** سابقة النقطة M' هي النقطة M التي صورتها M'؛ نعود بالشعاع المعاكس: M(x' − a ; y' − b).

**مثال:** سابقة N'(5 ; 0) بالانسحاب نفسه هي N(5 − 3 ; 0 + 2) = N(2 ; 2).

## 🔮 إحداثيّات منتصف قطعة

إذا كانت $A(x_A ; y_A)$ و $B(x_B ; y_B)$، فإنّ المنتصف I للقطعة [AB] له إحداثيّات:

$$\boxed{I\left(\frac{x_A + x_B}{2} \;;\; \frac{y_A + y_B}{2}\right)}$$

الفاصلة والترتيبة هما **المعدّل الحسابي** لفاصلتَي A وB وترتيبتَيهما.

**مثال محلول:** $A(2 ; 6)$ و $B(8 ; 2)$:

$$I\left(\frac{2+8}{2} \;;\; \frac{6+2}{2}\right) = I(5 \;;\; 4)$$

**التحقّق:** $\overrightarrow{AI}\begin{pmatrix} 5-2 \\ 4-6 \end{pmatrix} = \begin{pmatrix} 3 \\ -2 \end{pmatrix}$ و $\overrightarrow{IB}\begin{pmatrix} 8-5 \\ 2-4 \end{pmatrix} = \begin{pmatrix} 3 \\ -2 \end{pmatrix}$. متساويان، إذن I منتصف [AB]. ✓

**الطريقة العكسية — إيجاد طرف من المنتصف:** إذا عُرف المنتصف I وأحد الطرفين A، نعزل الطرف الآخر:

$$\frac{x_A + x_C}{2} = x_I \iff x_C = 2x_I - x_A$$

(والترتيبة بنفس الطريقة: $y_C = 2y_I - y_A$.)

**مثال محلول:** I(4 ; 1) منتصف [AC] حيث A(2 ; 5): $x_C = 2 \times 4 - 2 = 6$ و $y_C = 2 \times 1 - 5 = -3$، إذن C(6 ; −3).

> 🗡️ **مماثل نقطة بالنسبة إلى نقطة**: مماثل A بالنسبة إلى B هو النقطة M حيث B منتصف [AM]، أي $x_M = 2x_B - x_A$ و $y_M = 2y_B - y_A$.

## 🧪 المسافة بين نقطتين

إذا كانت $A(x_A ; y_A)$ و $B(x_B ; y_B)$، فإنّ المسافة AB هي:

$$\boxed{AB = \sqrt{(x_B - x_A)^2 + (y_B - y_A)^2}}$$

هذه الصيغة مستنبَطة من **نظرية فيثاغورس** المطبَّقة على المثلّث القائم الذي ساقاه $(x_B - x_A)$ و $(y_B - y_A)$.

**مثال محلول:** $A(1 ; 2)$ و $B(5 ; 5)$:

$$(x_B - x_A) = 5 - 1 = 4$$
$$(y_B - y_A) = 5 - 2 = 3$$
$$AB = \sqrt{4^2 + 3^2} = \sqrt{16 + 9} = \sqrt{25} = 5$$

> 🏆 المثلّث ذو الأبعاد 3-4-5 كلاسيكي في الامتحانات! احفظه: $3^2 + 4^2 = 9 + 16 = 25 = 5^2$.

## 🧮 تطبيقات مجتمعة — الشعاع والمسافة والمنتصف

**تطبيق:** المثلّث ABC بـ $A(0;0)$، $B(6;0)$، $C(3;4)$. أثبت أنّه متساوي الساقين.

- $AB = \sqrt{(6-0)^2+(0-0)^2} = \sqrt{36} = 6$
- $AC = \sqrt{(3-0)^2+(4-0)^2} = \sqrt{9+16} = \sqrt{25} = 5$
- $BC = \sqrt{(3-6)^2+(4-0)^2} = \sqrt{9+16} = \sqrt{25} = 5$

بما أنّ AC = BC = 5، فالمثلّث ABC متساوي الساقين **رأسه C** (الضلعان المتساويان يلتقيان في C). ✓

**منتصف [AB]**: $I\left(\frac{0+6}{2};\frac{0+0}{2}\right) = I(3;0)$.

$CI = \sqrt{(3-3)^2+(0-4)^2} = \sqrt{0+16} = 4$.

**تطبيق — التحقّق من زاوية قائمة (عكس فيثاغورس):** A(1 ; 1)، B(4 ; 1)، D(1 ; 5). هل الزاوية BAD قائمة؟

- $AB = \sqrt{(4-1)^2+(1-1)^2} = \sqrt{9} = 3$
- $AD = \sqrt{(1-1)^2+(5-1)^2} = \sqrt{16} = 4$
- $BD = \sqrt{(4-1)^2+(1-5)^2} = \sqrt{9+16} = \sqrt{25} = 5$

بما أنّ $AB^2 + AD^2 = 9 + 16 = 25 = BD^2$، فبموجب **عكس نظرية فيثاغورس** المثلّث ABD قائم في A، أي الزاوية BAD قائمة. ✓

> 🗡️ عند حلّ مسائل المعلّم: الخطوة الأولى دائمًا — احسب إحداثيّات الأشعّة والمسافات بالصيغ، ثمّ استنتج الخاصيّات الهندسية.

## 🏅 خلاصة الصيغ — بطاقة البطل

| المطلوب                         | الصيغة                                                                             |
| ------------------------------- | ---------------------------------------------------------------------------------- |
| إحداثيّات $\overrightarrow{AB}$ | $\begin{pmatrix} x_B - x_A \\ y_B - y_A \end{pmatrix}$                             |
| منتصف [AB]                      | $\left(\frac{x_A+x_B}{2} ; \frac{y_A+y_B}{2}\right)$                               |
| المسافة AB                      | $\sqrt{(x_B-x_A)^2+(y_B-y_A)^2}$                                                   |
| شعاعان متساويان                 | $\overrightarrow{AB}=\overrightarrow{CD} \iff x_B-x_A=x_D-x_C$ و $y_B-y_A=y_D-y_C$ |

> 🏆 اكتملت خريطة المستوي: رأس رابع، انسحاب، منتصف ومسافة — كلّها صارت حسابًا بين يديك. استعدّ للفصل القادم: الهندسة تغادر الورقة نحو **الفضاء**!
