# 📜 ملخّص: المعلّم في المستوي

- **المعلّم (O ; i ; j)**: محوران عموديّان بنفس المقياس. الأصل O عند (0 ; 0).
- **إحداثيّات نقطة**: M(x ; y) — الفاصلة أوّلًا ثمّ الترتيبة.
- **إحداثيّات شعاع**: $\overrightarrow{AB}$ له مركّبتان $(x_B - x_A \;;\; y_B - y_A)$ — «طرف ناقص منشأ».
- **منتصف قطعة**: $I\left(\dfrac{x_A+x_B}{2} ; \dfrac{y_A+y_B}{2}\right)$ — معدّل الإحداثيّات.
- **المسافة**: $AB = \sqrt{(x_B-x_A)^2+(y_B-y_A)^2}$ — مستنبَطة من فيثاغورس.
- **شعاعان متساويان** $\overrightarrow{AB} = \overrightarrow{CD}$: إذا وفقط إذا $x_B-x_A = x_D-x_C$ **و** $y_B-y_A = y_D-y_C$.
- **مثلّث كلاسيكي**: الأبعاد 3، 4، 5 تحقّق $3^2+4^2=5^2$ — احفظه للامتحانات.
