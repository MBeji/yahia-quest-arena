- **المعلّم (O ; i ; j)**: محوران عموديّان بنفس المقياس. الأصل O عند (0 ; 0).
- **إحداثيّات نقطة**: M(x ; y) — الفاصلة أوّلًا ثمّ الترتيبة؛ وإذا كان x = 0 أو y = 0 فالنقطة على أحد المحورين لا في ربع.
- **إحداثيّات شعاع**: الشعاع AB له مركّبتان (xB − xA ; yB − yA) — «طرف ناقص منشأ».
- **تساوي شعاعين والرأس الرابع**: AB = DC إذا وفقط إذا تساوت المركّبتان؛ في متوازي الأضلاع ABCD: xD = xA + xC − xB و yD = yA + yC − yB.
- **الانسحاب في المعلّم**: صورة M(x ; y) بانسحاب شعاعه (a ; b) هي M'(x + a ; y + b)؛ والسابقة: M(x' − a ; y' − b).
- **منتصف قطعة**: I = ((xA + xB)/2 ; (yA + yB)/2)؛ عكسيًّا: xC = 2xI − xA، ومماثل A بالنسبة إلى B: xM = 2xB − xA و yM = 2yB − yA.
- **المسافة**: AB = √((xB − xA)² + (yB − yA)²) — مستنبَطة من فيثاغورس؛ احفظ المثلّث الكلاسيكي 3-4-5: 3² + 4² = 5².
- **منهجية البرهان**: مثلّث متساوي الساقين = احسب المسافات الثلاث وقارنها؛ زاوية قائمة = قارن AB² + AD² مع BD² (عكس فيثاغورس).
