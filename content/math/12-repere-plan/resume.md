# 📜 ملخّص: المعلّم في المستوي

- **المعلّم (O ; i ; j)**: محوران عموديّان بنفس المقياس. الأصل O عند (0 ; 0).
- **إحداثيّات نقطة**: M(x ; y) — الفاصلة أوّلًا ثمّ الترتيبة؛ وإذا كان x = 0 أو y = 0 فالنقطة على أحد المحورين لا في ربع.
- **إحداثيّات شعاع**: $\overrightarrow{AB}$ له مركّبتان $(x_B - x_A \;;\; y_B - y_A)$ — «طرف ناقص منشأ».
- **تساوي شعاعين والرأس الرابع**: $\overrightarrow{AB} = \overrightarrow{DC}$ إذا وفقط إذا تساوت المركّبتان؛ في متوازي الأضلاع ABCD: $x_D = x_A + x_C - x_B$ و $y_D = y_A + y_C - y_B$.
- **الانسحاب في المعلّم**: صورة M(x ; y) بانسحاب شعاعه (a ; b) هي M'(x + a ; y + b)؛ والسابقة: M(x' − a ; y' − b).
- **منتصف قطعة**: $I\left(\dfrac{x_A+x_B}{2} ; \dfrac{y_A+y_B}{2}\right)$؛ عكسيًّا: $x_C = 2x_I - x_A$، ومماثل A بالنسبة إلى B: $x_M = 2x_B - x_A$ و $y_M = 2y_B - y_A$.
- **المسافة**: $AB = \sqrt{(x_B-x_A)^2+(y_B-y_A)^2}$ — مستنبَطة من فيثاغورس؛ احفظ المثلّث الكلاسيكي 3-4-5: $3^2+4^2=5^2$.
- **منهجية البرهان**: مثلّث متساوي الساقين = احسب المسافات الثلاث وقارنها؛ زاوية قائمة = قارن $AB^2 + AD^2$ مع $BD^2$ (عكس فيثاغورس).
