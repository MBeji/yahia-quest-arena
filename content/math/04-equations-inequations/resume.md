# 📜 ملخّص: المعادلات والمتراجحات

- **المعادلة من الدرجة الأولى**: عبارة على الشكل ax + b = 0 حيث a ≠ 0؛ حلّها هو قيمة x التي تجعل الطرفين متساويين.
- **حلّ المعادلة**: نُجري العملية نفسها على الطرفين لعزل x، ثمّ **نتحقّق** بتعويض الحلّ في المعادلة الأصلية.
- **الجداء المعدوم**: a × b = 0 ⟺ a = 0 أو b = 0. القاعدة تنطبق على الجداء **فقط** وليس الجمع.
- **المتراجحة من الدرجة الأولى**: عبارة بعلامة < أو > أو ≤ أو ≥؛ حلّها **مجموعة أعداد** لا عدد واحد.
- **حلّ المتراجحة**: نفس خطوات المعادلة، مع **قلب إشارة التراجح** عند الضرب أو القسمة بعدد **سالب** فقط.
- **المتراجحة المزدوجة**: نُجري العملية نفسها على الأطراف الثلاثة (مثل −3 ≤ 2x − 5 < 7 → 1 ≤ x < 6)؛ وشرطان منفصلان حلُّهما تقاطعهما.
- **تمثيل الحلول على المستقيم**: دائرة مملوءة = مشمول (≤ أو ≥)؛ دائرة مفتوحة = غير مشمول (< أو >)، مع سهم في اتّجاه الحلول.
- **النشر أوّلًا**: إذا احتوت المعادلة أقواسًا، ننشرها ونبسّط الحدود المتشابهة قبل عزل x.
- **ترجمة المسائل إلى معادلات**: سمِّ المجهول → معادلة لكلّ شرط → حلّ → فسّر النتيجة؛ وانتبه: المطلوب قد يكون غير المجهول الذي سمّيته.
