# 📜 ملخّص: الأشعّة والانسحاب

- **الشعاع $\overrightarrow{AB}$**: له منشأ (A)، اتجاه، منحى (من A نحو B)، وطول $\|\overrightarrow{AB}\|$.
- **الشعاع المعاكس**: $\overrightarrow{BA} = -\overrightarrow{AB}$ (نفس الاتجاه والطول، عكس المنحى).
- **تساوي شعاعين**: $\overrightarrow{AB} = \overrightarrow{CD}$ إذا تشاركا الاتجاه والمنحى والطول (ABDC متوازي أضلاع).
- **الانسحاب ذو الشعاع $\overrightarrow{u}$**: ينقل M إلى M' بحيث $\overrightarrow{MM'} = \overrightarrow{u}$. يحفظ الأطوال والزوايا والتوازي.
- **علاقة شال**: $\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}$ — سلاح التبسيط الأوّل.
- **الشعاع الصفري**: $\overrightarrow{AB} + \overrightarrow{BA} = \overrightarrow{0}$.
- **متوازي الأضلاع**: ABCD متوازي أضلاع $\iff$ $\overrightarrow{AB} = \overrightarrow{DC}$. مركزه O: $\overrightarrow{OA} + \overrightarrow{OC} = \overrightarrow{0}$ و $\overrightarrow{OB} + \overrightarrow{OD} = \overrightarrow{0}$.
