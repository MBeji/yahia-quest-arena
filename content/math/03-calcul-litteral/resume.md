# 📜 ملخّص: الحساب الحرفي والمتطابقات الشهيرة

- **الحدود المتشابهة**: لها نفس الجزء الحرفي؛ نجمعها بجمع معاملاتها: 4x + 3x = 7x.
- **التوزيع**: k(a + b) = ka + kb. انتبه لإشارة السالب: −(a − b) = −a + b.
- **نشر جداء ثنائيّين**: (a + b)(c + d) = ac + ad + bc + bd.
- **المتطابقات الشهيرة الثلاث**: (a + b)² = a² + 2ab + b²؛ (a − b)² = a² − 2ab + b²؛ (a − b)(a + b) = a² − b². تحذير: (a − b)² ≠ a² − b².
- **الحساب الذهني بالمتطابقات**: 99 × 101 = (100 − 1)(100 + 1) = 9999؛ 102² = (100 + 2)² = 10404.
- **التعميل بعامل مشترك**: نُخرج العامل المشترك خارج القوس (4x² − 8x = 4x(x − 2))، وقد يكون العامل قوسًا كاملًا: (x + 1)(x − 3) + (x + 1)(2x + 5) = (x + 1)(3x + 2).
- **التعميل بالمتطابقات والتعميل التامّ**: a² − b² = (a − b)(a + b)؛ a² + 2ab + b² = (a + b)²؛ a² − 2ab + b² = (a − b)²؛ والتعميل التامّ: عامل مشترك ثمّ متطابقة، مثل 2x² − 8 = 2(x − 2)(x + 2).
