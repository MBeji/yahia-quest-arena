# 📜 ملخّص: الحساب الحرفي والمتطابقات الشهيرة

- **الحدود المتشابهة**: لها نفس الجزء الحرفي؛ نجمعها بجمع معاملاتها: 4x + 3x = 7x.
- **التوزيع**: k(a + b) = ka + kb. انتبه لإشارة السالب: −(a − b) = −a + b.
- **نشر جداء ثنائيّين**: (a + b)(c + d) = ac + ad + bc + bd.
- **المتطابقة الأولى**: (a + b)² = a² + 2ab + b² (مربّع المجموع).
- **المتطابقة الثانية**: (a − b)² = a² − 2ab + b² (مربّع الفرق). تحذير: (a − b)² ≠ a² − b².
- **المتطابقة الثالثة**: (a − b)(a + b) = a² − b² (فرق المربّعين).
- **التعميل بعامل مشترك**: نُخرج القاسم المشترك الأكبر خارج القوس، مثل 4x² − 8x = 4x(x − 2).
- **التعميل بالمتطابقات**: a² − b² = (a − b)(a + b)؛ a² + 2ab + b² = (a + b)²؛ a² − 2ab + b² = (a − b)².
- **الحساب الذهني**: 99 × 101 = (100 − 1)(100 + 1) = 9999؛ 102² = (100 + 2)² = 10404.
