نرتّب عوالم الأعداد من الأصغر إلى الأكبر، كلّ عالم يحتوي ما قبله:

- **ℕ** الأعداد الطبيعية: 0، 1، 2، 3، …
- **ℤ** الأعداد الصحيحة النسبية: …، −2، −1، 0، 1، 2، …
- **𝔻** الأعداد العشرية: لها كتابة عشرية **متناهية** (مثل 3,25).
- **ℚ** الأعداد الكسرية (الناطقة): تُكتب على شكل كسر a/b حيث b ≠ 0.
- **ℝ** الأعداد الحقيقية: كلّ الأعداد التي تمثّل فاصلة نقطة على المستقيم العددي.

نكتب سلسلة الاحتواء:

$$ ℕ ⊂ ℤ ⊂ 𝔻 ⊂ ℚ ⊂ ℝ $$

> 🗡️ كلّ عدد طبيعي هو عدد صحيح، وكلّ عدد صحيح هو كسري… لكن العكس غير صحيح!

## ⚡ الأعداد الصمّاء (غير الناطقة)

العدد **الناطق** كتابته العشرية **متناهية** (مثل 1/2 = 0,5) أو **دوريّة** (مثل 1/3 = 0,333…).
أمّا العدد **الأصمّ** فلا يمكن كتابته على شكل كسر؛ كتابته العشرية **غير متناهية وغير دوريّة**.
أشهر الأمثلة:

- العدد √2 ≈ 1,414213…
- العدد π ≈ 3,14159…

مجموعة الأعداد الحقيقية ℝ هي اتّحاد الأعداد الناطقة (ℚ) والأعداد الصمّاء.

## 🛡️ ترتيب الأعداد الحقيقية

لكلّ عددين حقيقيّين a وb، تتحقّق **حالة واحدة فقط**: إمّا a < b، أو a = b، أو a > b.

قواعد الترتيب التي ستستعملها كثيرًا:

- إذا كان a < b فإنّ a + c < b + c (الجمع لا يغيّر الترتيب). _مثال_: من 2 < 5، إضافة 3 تعطي 5 < 8.
- إذا كان a < b و c > 0 فإنّ a × c < b × c (الضرب في عدد موجب يحفظ الترتيب). _مثال_: من 2 < 5، الضرب في 3 يعطي 6 < 15.
- إذا كان a < b و c < 0 فإنّ a × c > b × c (**الضرب في عدد سالب يقلب الترتيب**). _مثال_: من 2 < 5، الضرب في −3 يعطي −6 > −15.

> ⚠️ الفخّ الأكثر شيوعًا: نسيان قلب الإشارة عند الضرب أو القسمة على عدد سالب.

## 📐 المجالات

المجال هو مجموعة من الأعداد الحقيقية محصورة بين حدّين:

| المجال   | الكتابة بعدم المساواة | الحدود               |
| -------- | --------------------- | -------------------- |
| [a ; b]  | a ≤ x ≤ b             | مغلق (يشمل a وb)     |
| ]a ; b[  | a < x < b             | مفتوح (لا يشمل a وb) |
| [a ; +∞[ | x ≥ a                 | نصف مستقيم           |
| ]−∞ ; b] | x ≤ b                 | نصف مستقيم           |

القوس المتّجه نحو الحدّ يعني أنّ الحدّ **غير مشمول**؛ والقوس المربّع يعني أنّه **مشمول**.

_مثال محسوب_: المجال ]2 ; 5] يعني 2 < x ≤ 5، فالعدد 5 **ينتمي** إليه (لأنّ 5 ≤ 5 صحيحة)
والعدد 2 **لا ينتمي** إليه (لأنّ 2 < 2 خاطئة)، بينما 3 و4 ينتميان إليه.

## 🔮 القيمة المطلقة

القيمة المطلقة للعدد x هي **بعده عن الصفر** على المستقيم العددي، ونرمز لها بـ |x|:

| الحالة | القيمة المطلقة |
| ------ | -------------- |
| x ≥ 0  | \|x\| = x      |
| x < 0  | \|x\| = −x     |

أمثلة: |5| = 5، و |−7| = 7، و |0| = 0. القيمة المطلقة **لا تكون أبدًا سالبة**.

**معادلة بقيمة مطلقة**: إذا كان k > 0 فإنّ |X| = k ⟺ X = k أو X = −k.
_مثال_: |2x − 3| = 5 تعني 2x − 3 = 5 أو 2x − 3 = −5، أي 2x = 8 أو 2x = −2، ومنه x = 4 أو x = −1.

**حصر بقيمة مطلقة**: |x| ≤ k ⟺ −k ≤ x ≤ k.
_مثال_: |x| ≤ 3 تعني −3 ≤ x ≤ 3، أي x ∈ [−3 ; 3].

**معادلة من الشكل x² = a**: إذا كان a > 0 فللمعادلة x² = a **حلّان** هما √a و −√a.
_مثال_: المعادلة x² = 16 حلّاها 4 و −4 (لأنّ 4² = 16 و (−4)² = 16)، فمجموعة الحلول هي {−4 ; 4}.

## 🧮 العمليّات الأربع في ℝ

سلاح الجمع وسلاح الضرب لهما خصائص قويّة:

- **التبديل**: a + b = b + a و a × b = b × a
- **التجميع**: (a + b) + c = a + (b + c)
- **العنصر المحايد**: a + 0 = a و a × 1 = a
- **التوزيع**: a × (b + c) = a × b + a × c

> 🗡️ **عدّة سريعة** — تذكير من السنوات السابقة:
> **قواعد الإشارات**: سالب × سالب = موجب. _مثال_: (−3) × (−4) + (−2) = 12 − 2 = 10.
> **قوى الأعداد السالبة**: الأسّ الفردي يُبقي الإشارة سالبة. _مثال_: (−2)³ = −8.
> **جذر مربّع كامل**: √9 = 3 و √16 = 4.
> **المتطابقات الشهيرة**: (a ± b)² = a² ± 2ab + b² و a² − b² = (a − b)(a + b) — ستتعمّق فيها في فصل الحساب الحرفي.

## 🧪 الكتابة العلمية

**الكتابة العلمية** لعدد هي كتابته على الشكل a × 10ⁿ حيث a ∈ [1 ; 10[ (أي 1 ≤ a < 10) و n عدد صحيح نسبي.

_مثال محسوب_: 2500 = 2,5 × 1000 = 2,5 × 10³.

> ⚠️ يجب أن يكون 1 ≤ a < 10: الكتابة 25 × 10² تساوي 2500 لكنّها **ليست** كتابة علمية لأنّ 25 ≥ 10.
