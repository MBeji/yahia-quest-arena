# ⚔️ الأعداد الحقيقية — ساحة المعركة الأولى

> 💡 «قبل أن تخوض أيّ معركة حسابية، عليك أن تعرف عالم الأعداد الذي تقاتل فيه.»

## 🏰 مجموعات الأعداد

نرتّب عوالم الأعداد من الأصغر إلى الأكبر، كلّ عالم يحتوي ما قبله:

- **ℕ** الأعداد الطبيعية: 0، 1، 2، 3، …
- **ℤ** الأعداد الصحيحة النسبية: …، −2، −1، 0، 1، 2، …
- **𝔻** الأعداد العشرية: لها كتابة عشرية **متناهية** (مثل 3.25).
- **ℚ** الأعداد الكسرية (الناطقة): تُكتب على شكل كسر a/b حيث b ≠ 0.
- **ℝ** الأعداد الحقيقية: كلّ الأعداد التي تمثّل فاصلة نقطة على المستقيم العددي.

نكتب سلسلة الاحتواء:

$$ ℕ ⊂ ℤ ⊂ 𝔻 ⊂ ℚ ⊂ ℝ $$

> 🗡️ كلّ عدد طبيعي هو عدد صحيح، وكلّ عدد صحيح هو كسري… لكن العكس غير صحيح!

## ⚡ الأعداد الصمّاء (غير الناطقة)

العدد **الأصمّ** لا يمكن كتابته على شكل كسر؛ كتابته العشرية **غير متناهية وغير دوريّة**.
أشهر الأمثلة:

- العدد √2 ≈ 1.414213…
- العدد π ≈ 3.14159…

مجموعة الأعداد الحقيقية ℝ هي اتّحاد الأعداد الناطقة (ℚ) والأعداد الصمّاء.

## 🛡️ ترتيب الأعداد الحقيقية

لكلّ عددين حقيقيّين a وb، تتحقّق **حالة واحدة فقط**: إمّا a < b، أو a = b، أو a > b.

قواعد الترتيب التي ستستعملها كثيرًا:

- إذا كان a < b فإنّ a + c < b + c (الجمع لا يغيّر الترتيب).
- إذا كان a < b و c > 0 فإنّ a × c < b × c (الضرب في عدد موجب يحفظ الترتيب).
- إذا كان a < b و c < 0 فإنّ a × c > b × c (**الضرب في عدد سالب يقلب الترتيب**).

> ⚠️ الفخّ الأكثر شيوعًا: نسيان قلب الإشارة عند الضرب أو القسمة على عدد سالب.

## 📐 المجالات

المجال هو مجموعة من الأعداد الحقيقية محصورة بين حدّين:

| المجال   | الكتابة بعدم المساواة | الحدود               |
| -------- | --------------------- | -------------------- |
| [a ، b]  | a ≤ x ≤ b             | مغلق (يشمل a وb)     |
| ]a ، b[  | a < x < b             | مفتوح (لا يشمل a وb) |
| [a ، +∞[ | x ≥ a                 | نصف مستقيم           |
| ]−∞ ، b] | x ≤ b                 | نصف مستقيم           |

القوس المتّجه نحو الحدّ يعني أنّ الحدّ **غير مشمول**؛ والقوس المربّع يعني أنّه **مشمول**.

## 🔮 القيمة المطلقة

القيمة المطلقة للعدد x هي **بعده عن الصفر** على المستقيم العددي، ونرمز لها بـ |x|:

$$ |x| = x \text{ إذا كان } x ≥ 0، \quad |x| = -x \text{ إذا كان } x < 0 $$

أمثلة: |5| = 5، و |−7| = 7، و |0| = 0. القيمة المطلقة **لا تكون أبدًا سالبة**.

## 🧮 العمليّات الأربع في ℝ

سلاح الجمع وسلاح الضرب لهما خصائص قويّة:

- **التبديل**: a + b = b + a و a × b = b × a
- **التجميع**: (a + b) + c = a + (b + c)
- **العنصر المحايد**: a + 0 = a و a × 1 = a
- **التوزيع**: a × (b + c) = a × b + a × c
