# 📜 ملخّص: الدوال الخطية والتآلفية

- **الدالة الخطية**: f(x) = ax (مع a ≠ 0). تُعبّر عن التناسب المباشر وتمرّ دائمًا بنقطة الأصل O(0 ; 0).
- **الدالة التآلفية**: f(x) = ax + b (مع a ≠ 0). الدالة الخطية حالة خاصّة منها عندما b = 0.
- **التمثيل البياني**: مستقيم. a > 0 صاعد؛ a < 0 نازل؛ b = f(0) هو القطع مع محور التراتيب؛ نرسم بنقطتَين.
- **التوازي**: مستقيمان متوازيان ⟺ لهما معامل التوجيه نفسه (a₁ = a₂ مع b₁ ≠ b₂).
- **الصورة والسابقة**: الصورة بتعويض x في f(x) = ax + b؛ السابقة بحلّ ax + b = k أي x = (k − b)/a.
- **سابقة الصفر**: حلّ ax + b = 0 أي x = −b/a؛ بيانيًّا هي فاصلة تقاطع المستقيم مع محور الفواصل.
- **تحديد a وb من نقطتَين**: a = (y₂ − y₁)/(x₂ − x₁) ثمّ b = y₁ − a × x₁.
- **تقاطع مستقيمَين والنمذجة**: نحلّ a₁x + b₁ = a₂x + b₂؛ والمواقف الحياتية (ثمن = جزء ثابت + معدّل × كمية) تُنمذَج بدالة تآلفية.
