# 📜 ملخّص: المناعة والدفاع عن الجسم

- **المتعضّيات المجهرية**: **بكتيريا** (خلايا، تُعالَج بالمضادّات الحيوية) و**فيروسات** (لا تتكاثر إلّا داخل خلية، لا تتأثّر بالمضادّات الحيوية). **العدوى** = دخول ميكروب ممرض وتكاثره.
- **المستضدّ (الأنتيجان)**: جزيء **غريب** يتعرّف عليه جهاز المناعة كـ«غير ذاتي» فيستثير الدفاع.
- **الحواجز الطبيعية** (خطّ أوّل): الجلد، المخاطيّات والمخاط، العرق واللعاب والدموع والعصارة المعدية.
- **المناعة غير النوعية** (خطّ ثانٍ، سريع وعامّ): **البلعمة** بالكريات البيضاء — التصاق ← ابتلاع ← هضم ← إطراح. علامتها **الالتهاب** و**القيح**.
- **المناعة النوعية** (موجّهة، بذاكرة): **اللمفاويات B** تنتج **أجسامًا مضادّة** نوعية (مناعة خلطية)؛ **اللمفاويات T** تدمّر الخلايا المصابة (مناعة خلوية). **الذاكرة المناعية** تجعل الردّ الثاني أسرع وأقوى.
- **التلقيح**: ميكروب مُضعَّف/ميّت ← الجسم يصنع أجسامه ← **وقاية طويلة الأمد** (نشِطة). **المصل**: أجسام مضادّة جاهزة ← **علاج سريع** قصير الأمد (منفعلة).
- **السيدا (SIDA)**: فيروس **VIH** يدمّر **اللمفاويات T** فينهار جهاز المناعة وتظهر أمراض انتهازية. ينتقل بالدم والعلاقة الجنسية ومن الأمّ للجنين، **لا** بالمصافحة أو العطس.
