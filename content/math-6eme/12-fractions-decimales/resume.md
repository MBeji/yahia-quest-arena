# 📜 ملخّص: الأعداد الكسرية العشرية

- **كسرٌ عشريّ:** مقامُه 10 أو 100 أو 1000…
- **إلى عدد عشري:** عددُ أصفار المقام = عددُ الأرقام بعد الفاصلة (3/10 = 0,3 ; 7/100 = 0,07).
- **إلى كسر عشري:** 0,4 = 4/10، 1,25 = 125/100.
- **الاختزال:** 5/10 = 1/2، 25/100 = 1/4.
- **تحويلُ كسرٍ:** اجعل المقام 10/100 بالضرب (3/4 = 75/100 = 0,75).
