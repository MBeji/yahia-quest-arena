# 📜 ملخّص: جمع وطرح الأعداد الكسرية

- **مقامٌ مشترك:** نجمع/نطرح البُسُط ونُبقي المقام (2/7 + 3/7 = 5/7).
- **مقاماتٌ مختلفة:** نوحّد على مضاعفٍ مشترك، ثمّ نجمع/نطرح البُسُط (1/2 + 1/3 = 3/6 + 2/6 = 5/6).
- **كسرٌ مع عددٍ صحيح:** نكتب الصحيحَ كسرًا بالمقام نفسِه (1 + 1/4 = 4/4 + 1/4 = 5/4).
- **اختزالُ النتيجة** إن أمكن (2/6 + 1/6 = 3/6 = 1/2).
- ⚠️ لا نجمع المقامات؛ المقامُ يبقى (بعد التوحيد) ونجمع البُسُط فقط.
