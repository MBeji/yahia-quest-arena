# 📜 Résumé : Fonctions affines

- **Définition** : f : x ↦ ax + b. f(x) = ax + b est l'image de x ; un antécédent de y est un x tel que f(x) = y. Ex. f : x ↦ 2x + 3 donne f(4) = 11.
- **Coefficient et ordonnée à l'origine** : b = f(0) (ordonnée à l'origine) ; a = (f(x) − f(x'))/(x − x') (coefficient / taux d'accroissement). Cas particuliers : b = 0 ⟹ fonction linéaire ; a = 0 ⟹ fonction constante.
- **Représentation graphique** : droite d'équation y = ax + b, de coefficient directeur a (l'inclinaison), passant par le point (0 ; b). Deux points suffisent à la tracer.
- **Déterminer une fonction affine** : à partir de deux images, calculer a = taux d'accroissement, puis b en remplaçant dans ax + b. Ex. f(2) = 3 et f(4) = 1 ⟹ f(x) = −x + 5.
- **Résolution graphique** : l'équation f(x) = g(x) est l'abscisse du point d'intersection des deux droites ; l'inéquation f(x) ≤ g(x) est la zone où une droite est au-dessous de l'autre.
