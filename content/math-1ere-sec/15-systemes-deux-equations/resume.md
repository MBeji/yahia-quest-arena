# 📜 Résumé : Systèmes de deux équations à deux inconnues

- **Équation à deux inconnues** : ax + by + c = 0 admet une infinité de couples solutions (x ; y), qui sont les points d'une droite. Ex. x + y = 5 : (0 ; 5), (1 ; 4), (2 ; 3)…
- **Système** : deux équations imposées en même temps (ax + by = c et a'x + b'y = c'). Résoudre = trouver les couples vérifiant les deux égalités à la fois (en général, une seule solution).
- **Méthode par substitution** : exprimer une inconnue à partir d'une équation, la remplacer dans l'autre, résoudre. Ex. x + y = 5 et x − y = 1 ⟹ (3 ; 2).
- **Méthode par élimination** : multiplier les équations pour qu'en les additionnant une inconnue disparaisse. Ex. 2x + y = 7 et x − y = 2 : addition ⟹ 3x = 9, x = 3, y = 1.
- **Interprétation graphique** : chaque équation est une droite ; droites sécantes ⟹ une solution (le point d'intersection) ; strictement parallèles ⟹ aucune solution ; confondues ⟹ une infinité.
