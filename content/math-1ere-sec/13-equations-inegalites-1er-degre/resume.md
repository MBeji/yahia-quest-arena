# 📜 Résumé : Équations et inéquations du premier degré

- **Équation ax + b = 0 (a ≠ 0)** : unique solution x = −b/a. On la résout par équations équivalentes (ajouter un même nombre, multiplier par un même nombre non nul). Ex. 5x − 2 = 3x + 4 ⟹ x = 3.
- **Équation produit** : a × b = 0 ⟺ a = 0 ou b = 0. Ex. (x − 2)(x + 5) = 0 ⟹ x = 2 ou x = −5.
- **Équation x² = a** : si a > 0, x = √a ou x = −√a ; si a = 0, x = 0 ; si a < 0, aucune solution.
- **Inéquation du premier degré** : mêmes règles qu'une équation, mais on change le sens de l'inégalité en multipliant ou divisant par un nombre négatif (ex. −2x ≥ 4 ⟹ x ≤ −2).
- **Signe de ax + b et mise en équation** : ax + b s'annule en x = −b/a et change de signe (tableau de signe). Pour un problème : choisir l'inconnue, traduire, résoudre, interpréter.
