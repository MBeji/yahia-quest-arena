# ⚔️ Équations et inéquations du premier degré à une inconnue

> 💡 «Résoudre une équation, c'est retrouver l'inconnue cachée derrière l'égalité. Résoudre une inéquation, c'est trouver toutes les valeurs qui rendent une condition vraie. Le calcul littéral devient enfin une arme pour répondre à des questions.»

Grâce aux identités remarquables du chapitre précédent, tu peux maintenant **résoudre** : trouver la ou les valeurs de x qui satisfont une égalité ou une inégalité. Tu apprends la méthode générale pour ax + b = 0, les cas qui s'y ramènent (équation produit, x² = a), les inéquations et leur piège du changement de sens, et comment traduire un problème en équation.

## ⚖️ Résoudre une équation du premier degré

Une **équation du premier degré à une inconnue** s'écrit, après transformations, sous la forme ax + b = 0 (avec a ≠ 0). Elle admet **une unique solution** :

$$ ax + b = 0 ⟹ x = −b/a $$

On y parvient par des **équations équivalentes** : on peut ajouter un même nombre aux deux membres, ou les multiplier par un même nombre non nul, sans changer les solutions.

_Exemple détaillé_ : 5x − 2 = 3x + 4. On retranche 3x : 2x − 2 = 4. On ajoute 2 : 2x = 6. On divise par 2 : **x = 3**. Vérification : 5 × 3 − 2 = 13 et 3 × 3 + 4 = 13 ✓.

## 🟰 Équation produit et équation x² = a

**Équation produit** — une règle en or :

$$ a × b = 0 ⟺ a = 0 ou b = 0 $$

Un produit est nul si (et seulement si) l'un de ses facteurs est nul.

_Exemple détaillé_ : (x − 2)(x + 5) = 0 donne x − 2 = 0 ou x + 5 = 0, soit **x = 2 ou x = −5**.

**Équation x² = a** — trois cas selon le signe de a :

- si a > 0 : deux solutions, x = √a ou x = −√a ;
- si a = 0 : une solution, x = 0 ;
- si a < 0 : **aucune** solution (un carré n'est jamais négatif).

_Exemple détaillé_ : x² = 9 donne x = 3 ou x = −3 ; x² = −4 n'a **aucune** solution.

## ➕ Résoudre une inéquation du premier degré

On résout une **inéquation** comme une équation… avec un piège :

> Quand on **multiplie ou divise** les deux membres par un nombre **négatif**, on doit **changer le sens** de l'inégalité.

_Exemple détaillé_ : 2x − 6 ≥ 0 donne 2x ≥ 6, puis x ≥ 3 (on a divisé par +2, le sens ne change pas). En revanche, −2x ≥ 4 donne x ≤ −2 : en divisant par −2, l'inégalité **se retourne**.

## 📊 Le signe d'un binôme du premier degré

L'expression ax + b s'annule en x = −b/a et change de signe à cet endroit. On résume cela dans un **tableau de signe** : d'un côté de la racine le signe de −a, de l'autre le signe de a.

_Exemple détaillé_ : pour A(x) = 2x − 1, la racine est x = 1/2. Comme a = 2 > 0, A(x) est **négatif** pour x < 1/2, nul en 1/2, puis **positif** pour x > 1/2. Ainsi A(x) > 0 équivaut à x ∈ ]1/2 ; +∞[.

> 🗡️ Pour un produit comme (x − 2)(x + 3), on dresse le signe de chaque facteur, puis on applique la règle des signes : le produit est positif quand les deux facteurs ont le même signe.

## 🎯 Mettre un problème en équation

La démarche : choisir l'**inconnue** (x = la grandeur cherchée), **traduire** l'énoncé par une équation ou une inéquation, la **résoudre**, puis **interpréter** la solution dans le contexte (et vérifier qu'elle a un sens).

_Exemple détaillé_ : « Le triple d'un nombre augmenté de 5 vaut 26. » On pose 3x + 5 = 26, d'où 3x = 21 et **x = 7**. Le nombre cherché est 7.

> 🏆 Treizième quête franchie, héros : équations et inéquations n'ont plus de secret, et tu sais lire le signe d'une expression affine. Au prochain chapitre, ces droites deviennent des fonctions affines.
