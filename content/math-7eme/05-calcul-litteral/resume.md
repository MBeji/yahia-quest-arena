# 📜 ملخّص: الحساب الحرفي (العبارات الحرفية)

- **العبارةُ الحرفية والمتغيّر:** كتابةٌ رياضيّةٌ تحتوي حرفًا (متغيّرًا) يمثّل عددًا مجهولًا أو متغيّرًا، مربوطًا بأعدادٍ عبر عمليّات؛ إشارةُ الضرب تُحذف بين عددٍ وحرف (4x بدل 4×x).
- **القيمةُ العدديّة:** نستبدل المتغيّر بعددٍ محدَّد ونحسب باحترام أولويّة العمليّات (الضربُ قبل الجمع/الطرح).
- **اختزالُ عبارةٍ حرفية:** نجمع معامِلات الحدود المتشابهة (نفسُ الحرف) فقط؛ 2x + 3x = 5x وليس 5x² (الجمعُ لا يُربّع الحرف).
- **التوزيعية:** k×(a + b) = k×a + k×b وk×(a − b) = k×a − k×b — نضرب k في **كلّ** حدٍّ داخل القوسين.
- **التعميلُ بعاملٍ مشترك:** العكسُ الدقيق للتوزيعية: k×a + k×b = k×(a + b)؛ نستعمل م.م.ق بين المعامِلَين لإيجاد أفضل تعميل.
- **ترجمةُ نصٍّ إلى عبارةٍ حرفية:** ضعفُ x → 2x؛ ثلاثةُ أمثال x → 3x؛ مجموعُ x و7 → x + 7؛ الفرقُ بين x و4 → x − 4؛ نصفُ x → x / 2.
