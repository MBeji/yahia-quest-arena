# 📜 ملخّص: متوازي الأضلاع

- **التعريف:** رباعيٌّ أضلاعُه المتقابلةُ متوازيةٌ مثنى مثنى: (AB) ∥ (DC) و(AD) ∥ (BC)؛ يتقاطع قطراه [AC] و[BD] في المركز O.
- **الأضلاع:** الضلعان المتقابلان **متقايسان**: AB = DC وAD = BC؛ المحيطُ P = 2×(AB + BC).
- **الأقطار:** القطران **متناصفان** (يتقاطعان في منتصفهما المشترك O): OA = OC وOB = OD. متناصفان ≠ متقايسان.
- **الزوايا:** المتقابلتان **متقايستان** (Â = Ĉ، B̂ = D̂)؛ المتتاليتان **متكاملتان** (Â + B̂ = 180°).
- **شروطُ التعرّف (كلٌّ منها كافٍ وحده):** أضلاعٌ متقابلةٌ متوازية، أو متقايسة، أو قطران متناصفان، أو ضلعان متقابلان متوازيان **ومتقايسان معًا**، أو زاويتان متقابلتان متقايستان. ⚠️ ضلعان متقايسان فقط أو قطران متقايسان فقط **لا يكفيان**.
- **الحالات الخاصة (متوازي أضلاعٍ + شرط):** المستطيلُ = زاويةٌ قائمة ⟺ قطران **متقايسان**؛ المعيّنُ = ضلعان متتاليان متقايسان ⟺ قطران **متعامدان**؛ المربّعُ = مستطيلٌ ومعيّن ⟺ قطران **متقايسان ومتعامدان**.
