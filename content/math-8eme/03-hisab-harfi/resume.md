# 📜 ملخّص: الحساب الحرفي — النشر والتحليل

- **التوزيعية البسيطة:** k(a + b) = ka + kb وk(a − b) = ka − kb؛ نضرب العاملَ في **كلّ** حدٍّ داخل القوسين.
- **الإشارة السالبة أمام القوس:** −(a − b) = −a + b؛ الطرحُ قبل القوس يقلب إشارةَ كلّ الحدود.
- **التوزيعية المضاعفة:** (a + b)(c + d) = ac + ad + bc + bd؛ أربعةُ جُداءات ثمّ نختزل، مثل (x + 4)(x + 5) = x² + 9x + 20.
- **الاختزال:** نجمع الحدود المتشابهة فقط (نفسُ الحرف بنفس القوّة)؛ x² لا يُجمع مع x.
- **المتطابقات الهامة (تمهيد):** (a + b)² = a² + 2ab + b²؛ (a − b)² = a² − 2ab + b²؛ (a + b)(a − b) = a² − b²؛ لا تنسَ الحدّ الأوسط 2ab.
- **التعميل بالعامل المشترك:** العكسُ الدقيق للنشر: ka + kb = k(a + b)؛ أخرج العاملَ المشترك الأكبر عددًا وحرفًا، مثل 6x² + 9x = 3x(2x + 3).
