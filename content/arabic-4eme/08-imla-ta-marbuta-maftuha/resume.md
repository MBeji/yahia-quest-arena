# 📜 ملخّص: الإملاء: التاء المربوطة والمفتوحة

- **التاء المربوطة (ة / ـة)** تُرسم مستديرةً وعليها نقطتان، و**تُنطق هاءً ساكنةً عند الوقف**: «شجرة ← شجرهْ».
- **مواضعها:** آخر الأسماء المؤنّثة وصفاتها: «فاطمة، تلميذة، وردة، مجتهدة، جميلة».
- **التاء المفتوحة (ت / ـت)** تُرسم مبسوطةً، و**تبقى تاءً عند الوقف**: «بنت ← بنتْ».
- **مواضعها:** آخر الأفعال (تاء التأنيث/الفاعل): «كتبتْ، لعبتُ، ذهبتِ»، وبعض الأسماء مثل «بنت».
- **الحيلة الذهبيّة:** قف على الكلمة: سمعتَها هاءً ← مربوطة (ة)؛ بقيت تاءً ← مفتوحة (ت).
- **قاعدة:** تاء التأنيث في آخر الفعل **مفتوحةٌ دائمًا**: «زرعتْ» لا «زرعةْ».
- **الفخّ:** «اللعبة» (اسمٌ بتاءٍ مربوطة) غير «لعبتْ» (فعلٌ بتاءٍ مفتوحة).
