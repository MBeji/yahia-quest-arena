# ⚔️ الإملاء والكتابة — أكتب الكلمة صحيحةً وأضع علامتها

> 💡 «الكلمة الجميلة تُكتب صحيحةً… والجملة الجميلة تنتهي بعلامتها!»

عندما نكتب، لا يكفي أن نعرف الحروف؛ علينا أن نكتب آخر الكلمة صحيحًا، ونضع في آخر الجملة **علامةً** مناسبة. هيّا نتعلّم الإملاء وعلامات الترقيم.

## 🌸 التاء المربوطة (ة) في آخر الاسم

**التاء المربوطة** (ة) تاءٌ مستديرةٌ عليها نقطتان، تأتي في **آخر الاسم**. عند **الوقف** (التوقّف) ننطقها **هاءً**، مثل: **مَدْرَسة، تُفّاحة، وَرْدة، قِطّة**.

_مثال: نقول «هذه مَدْرَسة» وعند الوقف نسمعها «مَدْرَسهْ»، لكنّها تُكتب بالتاء المربوطة (ة)._

| الكلمة   | تُكتب بـ           |
| -------- | ------------------ |
| مَدْرَسة | التاء المربوطة (ة) |
| تُفّاحة  | التاء المربوطة (ة) |
| وَرْدة   | التاء المربوطة (ة) |

## ✋ التاء المفتوحة (ت) والهاء (ه)

ليست كلّ تاءٍ مربوطة! **التاء المفتوحة** (ت) تاءٌ ممدودةٌ عليها نقطتان، ننطقها **تاءً** حتّى عند الوقف، مثل: **بِنت، بَيت، أُخت**.

وهناك **الهاء** (ه) حرفٌ بلا نقطتين، مثل: **وَجه، فَمه**. الفرق: التاء المربوطة عليها نقطتان، والهاء بلا نقطتين.

| الكلمة | الحرف الأخير       |
| ------ | ------------------ |
| بِنت   | التاء المفتوحة (ت) |
| بَيت   | التاء المفتوحة (ت) |
| وَجه   | الهاء (ه)          |

## 📖 «ال» التعريف

نضع **«ال»** في أوّل الاسم لنُعرّفه، فيصير معروفًا معيَّنًا: **قَلَم ← القَلَم**، **شَمس ← الشَّمس**، **بَيت ← البَيت**. هذه الألف واللام تُسمّى **«ال» التعريف**.

_مثال: «كِتاب» اسمٌ عامّ، أمّا «الكِتاب» فهو كتابٌ نعرفه._

## ❓ علامات الترقيم: النقطة وعلامة الاستفهام وعلامة التعجّب

في آخر الجملة نضع **علامةً** تُناسبها:

- **النقطة (.)** نضعها في آخر الجملة **الخبريّة** التي تُخبر بشيء: _ذهبَ الولدُ إلى المدرسةِ._
- **علامة الاستفهام (؟)** نضعها في آخر **السؤال**: _هل تُحبّ القراءة؟_
- **علامة التعجّب (!)** نضعها للتعبير عن **الفرح أو الدهشة**: _ما أجملَ الحديقةَ!_

| الجملة              | علامتها             |
| ------------------- | ------------------- |
| أكلَ القطُّ السمكةَ | النقطة (.)          |
| أين ذهبَ أخوك       | علامة الاستفهام (؟) |
| ما أروعَ البحرَ     | علامة التعجّب (!)   |

> 🗡️ حِيلة سريعة: عند الوقف، إن نطقتَ آخر الكلمة **«ـه»** فهي **تاء مربوطة (ة)** مثل «مَدْرَسة»؛ وإن نطقتَها **«ـت»** فهي **تاء مفتوحة (ت)** مثل «بِنت».

> ⚠️ الفخّ الشائع: لا تكتب التاء المربوطة مكان المفتوحة! نكتب **«بِنت»** بالتاء المفتوحة لا «بِنة»، و**«بَيت»** لا «بَية». واعلم أنّ السؤال ينتهي بـ **؟** لا بالنقطة.

> 🏆 أحسنت! صرتَ تعرف التاء المربوطة (ة) والمفتوحة (ت) والهاء (ه)، و«ال» التعريف، وعلامات الترقيم. الآن تكتب كلماتك وجملك كتابةً صحيحةً جميلة!
